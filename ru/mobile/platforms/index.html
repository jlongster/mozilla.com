Не видите здесь своего телефона?

Если ваш телефон отсутствует в этом списке, вы всё же можете загрузить и использовать Firefox на вашем мобильном устройстве. Для запуска Firefox ваше мобильное устройство должно иметь достаточно оперативной памяти (128 МБ или больше). Тем не менее мы рекомендуем вам загрузить Firefox на протестированное устройство, чтобы вы могли использовать его в полную силу.

Вы также можете экспериментировать с Firefox, отправлять отзывы, писать дополнения и принимать участие в работе сообщества, установив версию Firefox для настольных ПК:

  • Windows
  • Mac OS X
  • Linux
