Abrochaos los cinturones

Firefox 4 incluye ahora mejoras de rendimiento importantes, incluyendo nuestro nuevo motor de JavaScript JägerMonkey. Desde unos tiempos más rápidos de arranque y de dibujado de gráficos hasta una velocidad mejorada en la carga de páginas, notarás la diferencia al instante.

3D en la Web

WebGL trae gráficos 3D a Firefox, abriendo la puerta a que los desarrolladores creen juegos llenos de vida y nuevos tipos de visualizaciones y experiencias para la Web.

Fabulosos tipos de letra

Las características de las tipografías OpenType de Firefox 4 ofrecen a los diseñadores y desarrolladores más control sobre una variedad de detalles tipográficos - como el kerning o las ligaturas - para crear sitios web preciosos.

Actualizar los formularios

Firefox 4 hace los formularios bellos y superfáciles de implementar. Con características como los nuevos atributos de lista que permiten el autocompletados y la validación de formularios de HTML5, la nueva API de formularios proporciona las herramientas para hacer que la aburrida implementación de formularios sea cosa del pasado.
