Requisitos do Sistema do Firefox 3.6

Windows

Sistemas Operacionais
  • Windows 2000
  • Windows XP
  • Windows Server 2003
  • Windows Vista
  • Windows 7
Hardware Mínimo
  • Pentium 233 MHz (recomendado: Pentium 500MHz ou superior)
  • 64 MB RAM (recomendado: 128 MB RAM ou superior)
  • 52 MB de espaço em disco

Mac

Sistemas Operacionais
  • Mac OS X 10.4 e posteriores
Hardware Mínimo
  • Computador Macintosh com processador Intel x86 ou PowerPC G3, G4 ou G5
  • 128 MB RAM (recomendado: 256 MB RAM ou superior)
  • 200 MB de espaço em disco

Linux

Requisitos de Software
Por favor, perceba que distribuidores de Linux podem fornecer pacotes para sua distribuição com requisitos diferentes.
  • O Firefox não poderá ser executado sem as seguintes bibliotecas ou pacotes:
    • GTK+ 2.10 ou superior
    • GLib 2.12 ou superior
    • Pango 1.14 ou superior
    • X.Org 1.0 ou superior
  • Para uma melhor funcionalidade, recomendamos as seguintes bibliotecas ou pacotes:
    • NetworkManager 0.7 ou superior
    • DBus 1.0 ou superior
    • HAL 0.5.8 ou superior
    • GNOME 2.16 ou superior
