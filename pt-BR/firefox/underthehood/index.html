Recursos para desenvolvedores Web

Áudio e Vídeo Abertos

O Firefox 3.6 inclui a melhor implementação do suporte HTML5 a áudio e vídeo (Ogg Vorbis e Ogg Theora), permitindo que desenvolvedores incluam recursos multimídia controlados via Javascript, HTML e CSS em vez de tecnologias proprietárias. No Firefox 3.6, vídeos podem ser exibidos em tela cheia. Visualizar documentação.

Acesso a Redes e Arquivos

O Firefox 3.6 oferece novas capacidades de acesso a redes e arquivos para desenvolvedores, incluindo a habilidade de selecionar múltiplos arquivos para upload e uma nova API Arrastar-e-Soltar que torna mais fácil para páginas aceitar novos arquivos. Uma nova API Arquivo, baseada em padrões emergentes, agora permite acesso a arquivos baseado em eventos de forma assíncrona (ver em ação). Junto com o XMLHttpRequests cross-site introduzido no Firefox 3.5, esses padrões dão ao desenvolvedor a habilidade de construir mashups incríveis de vários sites ao mesmo tempo.

Suporte a Fontes

No Firefox 3.5 introduzimos o suporte a fontes baixáveis online para os padrões da indústria TrueType e OpenType. Agora no Firefox 3.6 nós também incluímos o suporte ao novo formato Web Open Font, que é largamente suportado por lojas de fontes. Esse novo formato também costuma ser mais rápido para baixar que fontes TrueType e OpenType e oferece novas opções a webmasters.

TraceMonkey

Sabendo que os usuários têm o benefício do motor TraceMonkey melhorado do Firefox 3.6, desenvolvedores podem construir aplicações baseadas em Javascript e sites mais rápidos e responsivos do que nunca Visualizar documentação.

Gradientes CSS

Gradientes CSS permitem que você exiba transições suaves entre duas ou mais cores. O uso de gradientes CSS pode substituir imagens e reduzir tempos de download, criar um leiaute mais flexível e ter melhor aparência ao ser ampliado. O Firefox 3.6 suporta dois tipos de gradientes CSS: linear e radial.

Navegação ciente de localização

No Firefox 3.6 os usuários podem compartilhar a sua localização com sites, permitindo que desenvolvedores personalizem suas aplicações e as tornem mais úteis e com resultados mais relevantes. Novidade no Firefox 3.6, desenvolvedores podem consultar o endereço correspondente a um local específico. Saber mais sobre esse recurso.

Orientação

Alguns notebook e dispositivos móveis podem detectar a sua própria orientação e posição. O Firefox 3.6 pode expor essa informação para páginas da web, permitindo que desenvolvedores construam interações interessantes e divertidas — incluindo jogos. Visualizar documentação.

JSON Nativo

O Firefox 3.6 melhorou seu suporte a JSON com manipulação nativa de objetos JSON, tornando a codificação e decodificação JSON mais rápidas do que nunca. Visualizar documentação.

Web Worker Threads

O processamento paralelo finalmente chegou à web por meio do novo recurso Web Workers, que permite que códigos Javascript espalhem-se em processos para manipular tarefas em segundo plano. Aplicações web agora podem tirar vantagem dos processadores modernos de vários núcleos. Visualizar documentação.

Media Queries

O suporte a consultas de mídia do CSS3 permite que folhas de estilo definam a aparência do conteúdo com maior precisão. Agora elas analisam de forma mais inteligente os detalhes do dispositivo que está exibindo o seu conteúdo, dando a você mais controle sobre a experiência final do usuário. Por exemplo, você pode renderizar o seu site de forma diferente para impressoras coloridas e para impressoras em preto e branco. Visualizar documentação.

Suporte Offline

O Firefox 3.6 suporta o recurso de cache offline do HTML5. Isso permite que aplicações da web armazenem conteúdo estático no sistema do usuário para reutilizá-lo em vez de carregar pela rede toda vez que ele for necessário. Resultado: tempos de carga de aplicações muito mais rápidos. Visualizar documentação.

Mais atrativos para o desenvolvedor

Além dos recursos citados, o Firefox 3.6 atualizou e adicionou outras propriedades CSS, eventos DOM e ferramentas para construir páginas ainda mais bonitas e responsivas. Dê uma olhada na lista completa na nossa página do Firefox 3.6 para desenvolvedores.
