Recursos em destaque

Firefox 4 está quase pronto. Aqui estão alguns dos recursos que você não pode perder:

Gráficos suaves

Novos objetos JavaScript Value significam que o Firefox pode processar códigos numéricos de forma muito mais eficiente, resultando em imagens mais suaves.

SQLite 3.7.0

Os recursos Campo de Endereços Inteligente, Gerenciador de Downloads e IndexedDB estão mais rápidos e suaves, graças à atualização do SQLite, um sistema de banco de dados SQL.
