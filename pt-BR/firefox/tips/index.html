Adicione Favoritos com um único clique

Catalogue os sites que você gosta (e economize tempo) com os Favoritos em Um Clique. Quando você visitar uma página que gostaria de lembrar, basta clicar no ícone da estrela no campo de endereços. O Firefox irá adicioná-lo à sua lista de favoritos para que você possa encontrá-lo facilmente.

Saiba com quem você está lidando

Uma tática comum entre espiões da Web é criar clones de sites, conhecidos como phishing sites, que fingem ser o seu banco, seu site de compras favorito, etc. Felizmente, o Firefox torna fácil determinar se os sites são quem dizem que são – basta clicar no ícone do site para obter uma visão geral da identidade do site instantaneamente.

Intermediário

Brinque com os marcadores (tags)

Uma vez que você criar uma grande lista de favoritos, localizá-los pode se tornar uma bagunça. Para mantê-los organizados, você pode atribuir marcadores aos seus favoritos: basta dar um clique duplo no ícone da estrela e você poderá inserir seus marcadores.

Você pode inserir quantos marcadores desejar em um favorito (lembre-se de separar os marcadores com vírgulas), e então encontrar facilmente os sites que procura digitando o marcador no campo de endereços. Por exemplo, digitar “viagem” no campo de endereço exibirá todos os sites os que você adicionou aos favoritos durante sua pesquisa de viagem de férias.

Saiba tudo sobre os Complementos

A versão padrão do Firefox está cheia de recursos, é claro, mas por que parar por aqui? Existem mais de 5.000 complementos — pequenos extras que acrescentam funções ao Firefox — que você pode instalar para melhorar praticamente qualquer tarefa que você pensar. Clique em Ferramentas → Complementos para abrir o Gerenciador de Complementos e descobrir todas as formas de tornar o Firefox só seu.

Pesquise de forma inteligente

O campo de pesquisa do Firefox já é familiar para você. Mas você sabia que você pode personalizá-la com praticamente qualquer serviço de pesquisa que desejar? Basta clicar no ícone à esquerda do campo de pesquisa para exibir a lista de opções padrão.

Melhor que isso, clique em Organizar pesquisas para reorganizar a lista, adicionar mais mecanismos de pesquisa e até mesmo atribuir um atalho para seus mecanismos favoritos. Por exemplo, você pode atribuir ao Google o atalho “G” — assim, se você quiser fazer uma pesquisa sobre receitas de bolo, digitar “G receitas de bolo” no campo de endereço exibiria uma lista completa de receitas.

Localize num piscar de olhos

O recurso Localizar ao digitar é outro útil poupador de tempo. Em vez de utilizar a barra Localizar para localizar uma palavra em uma página, simplesmente clique em qualquer lugar da página e comece a digitar a palavra que deseja localizar. O cursor imediatamente saltará para a primeira ocorrência do termo.

Você também pode usar esse recurso para links. Por exemplo, em vez de mover o mouse pela página até o link “Saiba mais”, apenas comece a digitar a palavra e quando o cursor encontrá-la, pressione Enter.

Elimine qualquer rastro

Por razão de segurança e privacidade, não é recomendável deixar rastros enquanto você navega (especialmente se você estiver usando um computador compartilhado). O Firefox permite que você apague seu histórico de navegação e downloads, limpe seu cache e elimine seus cookies em um único clique. Basta clicar em Ferramentas → Limpar dados pessoais para fazê-lo. Ou, para ser ainda mais cuidadoso, clique em Ferramentas → Opções → Privacidade e marque a opção “Limpar dados pessoais ao sair do Firefox”.

Avançado

Vá diretamente para seus sites favoritos

Você pode adicionar palavras-chave aos seus favoritos para um acesso mais rápido e fácil. Na Biblioteca, adicione uma palavra-chave curta no campo Palavra-chave, e você poderá acessar esse favorito simplesmente digitando essa palavra-chave no campo de endereço. Por exemplo, você pode adicionar a palavra-chave “links” ao seu favorito do del.icio.us, e ao digitar “links” no campo de endereço você obterá seus links favoritos.

Crie pastas inteligentes

Se você é um usuário avançado que frequentemente precisa manter uma lista de sites específicos, aqui está poupador de tempo: usando a Biblioteca do Firefox 3, você pode criar e salvar pesquisas em pastas que são automaticamente atualizadas à medida que você adiciona sites aos seus favoritos e histórico.

Primeiramente, abra a Biblioteca clicando em “Organizar favoritos” no menu Favoritos. Então, digite os termos da sua pesquisa no campo de pesquisa. Em seguida, clique no botão Salvar para criar uma Pasta Inteligente.

Gerencie seus downloads

Se você faz downloads com frequência, você pode utilizar o Gerenciador de Downloads para manter as informações de cada um deles. Você pode pausar e continuar downloads e poupar tempo abrindo seus arquivos diretamente do gerenciador.

Se você precisar das informações de um download antigo, clique em Ferramentas → Downloads e use o campo de pesquisa para localizar seu arquivo. Ao localizar seu arquivo, dê um clique duplo para abri-lo ou use o menu de contexto e selecione “Copiar link do download”.

Descubra as ferramentas de desenvolvimento

Se você é um desenvolvedor de websites, as ferramentas de desenvolvimento do Firefox tornarão sua vida mais fácil. O site Mozilla Add-ons oferece várias ferramentas para dinamizar o processo de desenvolvimento, incluindo o Firebug para editar, depurar e monitorar CSS, HTML e JavaScript em tempo real em qualquer página, o Tamper Data para visualizar e modificar cabeçalhos HTTP/HTTPS e parâmetros POST, e o DOM Inspector para examinar qualquer elemento HTML ou CSS com um simples clique.

Use a web como você preferir

Agora você pode utilizar serviços da web para manipular protocolos e ter acesso rápido às suas aplicações web favoritas. Por exemplo, você pode configurar o Firefox para que ao clicar em um link mailto: em qualquer site o Firefox abra uma nova mensagem no seu webmail favorito, e não no programa de email padrão do seu computador (perceba que esse recurso funciona somente com serviços de webmail registrados no Firefox 3).

Clique em Ferramentas → Opções → Programas para selecionar a aplicação padrão para cada protocolo ou selecionar “Perguntar” se você preferir escolher a aplicação manualmente todas as vezes.

Se você é um desenvolvedor de aplicações web, saiba mais sobre como adicionar suporte a manipuladores de protocolos baseados na web no Mozilla Developer Center.
