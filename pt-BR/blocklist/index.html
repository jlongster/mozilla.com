Complementos bloqueados

Esta página contém uma lista de complementos que não devem ser utilizados com produtos Mozilla.

  • Internet Download Manager, v2.1-3.3 para Firefox 3.0a1 e mais recentes. Motivo: causava travamentos ao iniciar (ver bug 382356).
  • Free Download Manager, v1.0-1.3.1 para Firefox 3.0a1 e mais recentes. Motivo: alto volume de travamentos (ver bug 408445).
  • Yahoo Application State Plugin, v1.0.0.5 ou anteriores para Firefox 3.0a1 e mais recentes. Motivo: alto volume de travamentos (ver bug 419127).
  • Pacote de idioma Vietnamita, v2.0 para todas as aplicações. Motivo: arquivos corrompidos (ver bug 432406).
  • Apple QuickTime Plugin, v7.1.*, para todas as versões do Firefox 3 no Windows. Motivo: execução de código remoto em várias versões (ver bug 430826).
  • Crawler Toolbar, para Firefox 3.0a1 e mais recentes. Motivo: alto volume de travamentos (ver bug 441649).
  • Daemon Tools Toolbar, versões anteriores a 1.0.0.5, para todas as aplicações. Motivo: alto volume de travamentos (ver bug 459850).
  • AVG SafeSearch, versões anteriores a 8.5 para todas as aplicações. Motivo: travamento de um método do núcleo de navegação (ver bug 479095).
  • Skype extension, versões anteriores a 3.3.0.3971, para todas as versões do Firefox. Motivo: alto volume de travamentos e outros problemas de estabilidade (ver bug 448768). Os usuários devem atualizar o Skype.
  • Enigmail, versões anteriores a 0.97a, para Thunderbird 3.0pre e mais recentes. Motivo: alto volume de travamentos e outros problemas de estabilidade (ver bug 531047). Os usuários devem atualizar o Enigmail.
  • YSlow, v2.0.5, para Firefox 3.5.7 e mais recentes. Motivo: alto volume de travamentos e outros problemas de estabilidade (ver bug 542686). Os usuários devem atualizar o YSlow para a versão 2.0.6.
  • Viewpoint Media Player, todas as versões, para Firefox 3.0 e mais recentes. Motivo: fabricante afirma haver incompatibilidade com a versão 3.0 e superiores, alto volume de travamentos e outros problemas de estabilidade (ver bug 539282). Para mais informações, visite a página do fabricante.
  • Java Deployment Toolkit, versão 6.0.200.0 ou anteriores. Motivo: vulnerabilidades de segurança (ver bug 558584). Para mais informações, visite a página do fabricante.
  • Internet Download Manager, versões 6.9.8 e anteriores, para Firefox 3.7a1pre e mais recentes. Motivo: alto volume de travamentos e outros problemas de estabilidade (ver bug 578443).
