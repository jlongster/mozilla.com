Acordos de Licença para o Usuário Final Mozilla (EULAs)

A partir do Firefox 3.0.5, os usuários não precisam mais aceitar os acordos de licença do usuário final (EULA) para baixar o Firefox. Entretanto, se você baixou alguma versão anterior do Firefox, você concordou com os termos de um EULA. Esses acordos estão disponíveis para sua referência. Se você tem o Firefox 3.0.6 ou uma versão superior, por favor, leia a nossa página de Avisos Legais do Firefox.
