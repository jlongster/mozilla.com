家是重要的，因此你需要一个告警系统保护你的私人数据。你的密码短语确保只有你能访问你的信息。

入门指南

  1. 在你的电脑上安装 免费的 Firefox 同步附加组件 。

    Firefox 之家使用 Firefox 同步附加组件的同步功能访问你的 Firefox 浏览器信息并将其安全地发送到你的 iPhone。

  2. 一旦 Firefox 重新启动，根据提示创建一个带有密码和密码短语的帐号。

    Firefox 之家使用你的密码短语在本地加密和保护你的信息。

  3. 从现在开始，Firefox 同步功能将会通过我们的服务安全同步你的数据，初次同步可能会花费最多20分钟时间。

    只有浏览器不处于隐私浏览模式时，Firefox 同步才会工作。一旦同步完成，你可以在工具 > 同步菜单看到最近更新时间。

  4. 要完成设置，请在你的 iPhone 上输入你的帐号，密码和密码短语，然后触按虚拟键盘上的 “完成” 。

    如果你在登录时遇到问题， 点击这里寻求帮助。

  5. 一旦完成，你将可以在 iPhone 上访问你的电脑上的书签，历史记录，以及打开的标签页。

    如果没有显示你的数据， 点击这里寻求帮助。
