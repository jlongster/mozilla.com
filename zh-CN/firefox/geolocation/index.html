位置定位体验

Firefox 能够告诉网站您在哪里，使您可以发现更加相关和更加有用的信息。 一切都是为了使 Web 更加智能，并且在您充分许可的情况下才会执行这些操作。 试试吧！

常见问题

什么是位置定位体验？

使用位置定位体验的网站询问您的方位，是为了给您提供更多相关的信息，从而节省您的搜索时间。比如说，您正在您所在的区域寻找一个比萨餐馆。一个网站可能会询问您的位置信息，然后您只需要简单的输入“比萨”就能得到想要的结果... 并不需要更多的信息和额外的输入。

您还可以获得行进路线，只要告诉网站您想从哪里到哪里就可以了。

这个服务完全是自由选择的； Firefox 不会在没有您的许可之下共享您的位置； 并且在极大的服从您的隐私下为你提供服务。和所有的 Firefox 元素一样，位置定位也用开放的标准创建， Web 开发人员很容易接受它。

位置定位是怎么工作的？

当您访问一个位置定位网站的时候， Firefox 将会询问您是否公开您的位置信息。

如果您同意了， Firefox 便会收集您周围的无线热点和您 PC 的 IP 地址。然后 Firefox 把这些信息发送给默认的位置定位服务提供者，也就是谷歌位置服务，由它来计算您的位置。最后您的位置信息就在您请求的网站上被共享出来。

如果您不同意， Firefox 什么也不做。

获得的位置定位有多精确呢？

精确度会根据不同的位置而有所差别。在某些地方，我们的位置服务提供商可能只能提供很小范围的定位服务。而在另一些地方提供商的服务范围是非常大的。被我们的服务商返回的位置是一个估算的值，我们不保证它的精确性。请不要在紧急时刻使用这些信息，只是作为日常参考。

什么信息会被发送？发送给谁？我的隐私怎样得到保护？

我们十分尊重您的隐私，所以 Firefox 绝对不会在没有您本人许可的情况下共享这些信息。当您访问一个需要您个人信息的网页时，我们将会提示您是否愿意让我们的第三方服务提供商共享这些信息。

默认的， Firefox 用谷歌服务根据您发送的信息判断您的位置：

  • 您 PC 的 IP 地址，
  • 在您附近的无线热点信息，
  • 加上一个被谷歌提供的两周期限的零时客户标识。

关于 Firefox 是怎样收集和使用这些信息的，请查阅 Firefox 隐私策略。

谷歌位置服务随后返回您请求的位置信息的评估结果（例如：纬度和经度）。关于谷歌是怎样收集和使用这些信息的，请查阅谷歌地理位置定位隐私策略。

为了保护您的隐私，被传输的信息使用加密通道。一旦 Firefox 拥有了您的位置信息，便把它交给您请求的那个网站。您请求的那个网站的名字和位置以及 cookie 绝对不会被谷歌位置服务共享。

Mozilla 和谷歌绝对不会用收集来的信息窥视您。

关于被请求的站点怎样处理您的位置信息，请参考站点隐私策略。

更多关于您的隐私，还可以阅读：

  • Mozilla Firefox 隐私策略
  • 谷歌隐私策略

当我浏览网站的时候会被跟踪吗？

当然不会。Firefox 只有在网站需要的时候才向您请求位置，也只有在您批准的情况下才会共享您的位置。Firefox 不会跟踪和记录您访问过的位置。

我怎样对一个网站撤销我的许可？

如果您没有默认允许 Firefox 每次都可以将您的位置给网站或者您后来改变想法了。可以这样做：

  • 首先去你要取消许可的那个网站
  • 找到 工具 菜单，然后选择页面信息
  • 接着选择 许可 标签
  • 最后改变 共享位置

我怎样清除“零时客户标识号”？

  • 到 工具 菜单， 然后选择 清楚最近历史
  • 随后点击 细节 箭头
  • 并确认 Cookies 被选中了
  • 最后点击 立刻清除

我怎样永久关闭位置体验？

在 Firefox 中位置体验总是可选添加的。在没有您的许可之下，位置信息绝对不会被发送。如果你需要完全停止这个功能，请做以下步骤：

  • 在地址栏，输入 about:config
  • 之后，再输入 geo.enabled
  • 双击 geo.enabled 属性
  • 这样就可以停止位置体验了
