Firefox 4 Beta 包含基于 Mozilla Test Pilot 的 Feedback 附加组件

一旦你安装 Firefox 4 Beta, 就会看到右上角的 Feedback 附加组件。你可以在任何时候通过它提供实时反馈。你还可以收到用户调研提醒（由 Mozilla Test Pilot 运作），该调研帮助我们了解你使用 Firefox 的一些信息，但是不会记录你访问的网页。你可以控制数据的分享，也可以选择不参加某个调研项目，或者查看系统收集了哪些数据。数据被发送给 Mozilla 之前会征求你的同意，参阅 Feedback 附加组件隐私策略 了解更多信息。
