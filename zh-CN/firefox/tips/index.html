中阶教程

玩转标签（Tag）

如果您收藏了大量的杂乱无章的书签，可以想象要管理起来得多么麻烦。要改变这一切，让您管理起来更方便,您可以在您的站点中加入一个标签：只需双击星型图标 Firefox 就会自动提示您输入标签了。

您可以对一个站点计入多个标签(记住在每个标签之间要加上一个逗号)，然后你可以在地址栏中输入标签来直接访问这些标识这个标签的站点。举个例子,在地址栏中输入"旅行"会自动将你标识了这个标签的站点。

了解更多关于扩展的信息

您现有的Firefox已经是一个功能非常完善的浏览器了，那么让我们来让它更加完美吧。距今为止已近有超过 5,000 个来让您的 Firefox 变得更加符合您的口味、更加强大的扩展了。只需点击工具 → 扩展来打开扩展管理器然后开始打造您自己的 Firefox 吧。

更个性化的搜索

现在您应该已经对Firefox中内置的搜索栏非常熟悉了吧？不过现在你可以自己设定您所需要的搜索引擎了，怎么样?只需点击搜索栏左边的图标来更改您所需要的搜索引擎。

还有,您可以点击搜索图标来重新排列搜索的列表，也可以增加附加的搜索引擎甚至您还可以重新组织用来搜索的键盘的快捷键。举个例子,您可以将 Google 赋予快捷键"G" - 然后如果您想搜索cookie recipes,只需在地址栏中输入"G cookie recipes"之后的工作就交给 Firefox 吧。

瞬间完成搜索

这项功能也是为了提高您的浏览效率而设计的.相比在一个页面上用“查找”栏来查找相关的内容,其实您只需点击页面上的任何一个位置然后输入您想查找的内容即可。您的光标会自动跳转到您搜索的内容上去了。

您也可以用这个来搜索链接。举个例子,如果你想将光标移到"learn more"这个链接上去时,只需输入这个名字就会直接跳转到这个链接上去了。

清除一切痕迹

基于安全和隐私的原因，在浏览之后最好要清楚掉您的浏览信息（尤其是你是在共享的计算机上进行浏览时）。Firefox可以让您方便的删除您的浏览历史和下载历史。只需单击工具 → 删除私密信息即可完成。或者,到工具 → 选项 → 隐私 并选择“在退出Firefox时始终删除我的私密信息”这个选项。

高级用户

直接浏览您最喜欢的网站

您可以在您的书签中加入关键字使您可以更方便使用这项功能。在资料库中，只需在关键字栏中输入一个简单的关于这个站点的关键字，然后您就可以直接在地址栏中输入那个关键字来直接访问这个书签了。举个例子,你可以对你的 del.icio.us 帐号取关键字“links”,然后在地址栏中输入“links”就可以直接访问你的 delicious 帐号了.

创建智能文件夹

如果您会经常的添加大量的书签来跟踪这些页面的更新的话，这里有一个小窍门:使用 Firefox 3 的资料库,你可以在一个自动将这些页面添加为书签和历史的文件夹中创建和保存搜索 。

首先，选择书签菜单中的“管理书签”来打开管理器。然后，在搜索栏中输入您要搜索的内容。然后点击保存按钮来创建一个“智能文件夹”。

管理您的下载

如果您经常在互联网上下载资料的话，您可以用下载窗口来监控您所有的下载。您可以暂停和恢复下载,然后直接在窗口中直接打开下载的文件。

如果您想了解一个曾经下载的文件的下载情况,点击工具→下载使用其中的搜索栏来查找文件。找到之后,双击打开这个文件,或者 Ctrl + 单击来选择"复制下载地址"。

探索开发者工具

如果您是一个网页开发者，Firefox 的开发者工具会让您的工作更加充满效率。Mozilla 扩展站点向您提供了大量的开发工具,比如用 Firebug 可以编辑、测试和实时观察任何页面中的 CSS，HTML，和JavaScript，控制查看的数据和 HTTP/HTTPS 的消息头和 POST 参数,还有 DOM Inspector 来检查任何的 HTM L或 CSS 组成。

将网页设置为您喜欢的样子

现在您可以使用网络协议接管服务来快速访问您最喜爱的网络应用程序了。举个例子，您可以将 Firefox 设置为：当单击 mailto: 协议会自动打开您喜欢的在线邮件而不必打开您的计算机默认的邮件客户端了(注意：这项功能只有当您的在线邮箱业务已经在 Firefox 中注册的情况下才能使用)。

转到 工具→选项→应用程序选择每个协议的默认程序或者如果你希望每次都询问你选择什么程序请选择“总是询问”。

如果你是网络程序开发者，关于如何增加基于网络的协议支持请查看Mozilla 开发者中心的相关文章。
