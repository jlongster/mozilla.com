内置于 Firefox 4 Beta 的功能: 同步(Sync)

无论是在工作地点、家中还是移动设备上，同步功能使你可以无缝地在多台电脑设备之间访问重要的 Firefox 数据 — 包括书签、密码、历史记录和打开的标签页。观看视频了解使用方法。

特色展台

Firefox 4 开了个好头。以下是其他一些你不希望错过的重要功能：

平滑图像

新的 JavaScript 引擎意味着 Firefox 可以更高效地执行更复杂的代码，从而使得图像更为平滑。

SQLite 3.7.0

得益于 SQL 数据库引擎 SQLite 的升级，智能地址栏、下载管理器和 IndexedDB 功能更为快速流畅。
