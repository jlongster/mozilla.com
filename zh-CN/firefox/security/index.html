最安全的网页浏览器

Firefox 保护您的个人信息私密性并且使您的网上冲浪远离危险。

那么我们将怎样去做呢？

是什么使 Firefox 与众不同？ 最重要的是，我们是开放的。这意味着世界上的每个人（我们有数以千计的专家在背后声援）都可以查看代码并找出防御盔甲上任何潜在的弱点。

而且当得知出现一个问题后，我们会立刻挽起袖子进行修复。虽然这意味着我们有些许的不完美，但关注这些问题都会为您（以及我们）争取到最大的好处。

简单的说, 您的安全是我们的当务之急。

有的方面我们很愚蠢。

没有人比我们更喜欢因特网的。但是，诡计, 垃圾广告和好战的病毒是确确实实存在的威胁，所以您需要在使用网页的时候保护自己。

那就是 Firefox 要解决的。

使用最安全的方式进行网上冲浪，因为:

  • 我们不会试图去独自寻找问题。 一个由安全专家组成的国际社区整日忙碌于将您的网页浏览器变得更加安全(这要感谢我们的开放源代码策略) 。就像是您的门卫由一群训练有素的武士率领一般。
  • 我们关注您每一步的安全。 在写下任何一行代码之前， 安全专家都会用心识别并寻找潜在的问题。
  • 我们保持最新的发行状态。 我们不间断的监控一切使用过程并先一步发行新的 Firefox 升级版本。在开放源代码世界中的运作意味着任何人都可以帮助我们发现并修复缺陷。

对于 Firefox 如何保护您上网安全的详细情况, 请访问我们的安全博客。
