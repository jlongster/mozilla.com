Firefox 4 Beta inneholder tilbakemelding-utvidelsen som er basert på Mozilla Test Pilot

Når du installerer Firefox 4 Beta vil du se Tilbakemelding-utvidelsen i hjørnet øverst til høyre. Du kan bruke denne for å gi umiddelbar tilbakemelding når som helst. Du vil også få informasjon om brukerstudier (som utføres av Mozilla Test Pilot) som samler inn informasjon om hvordan du bruker Firefox, men ikke hvilke nettsider du besøker. Du har full kontroll over deltakelsen din, og kan velge å melde deg ut av hvilket som helst studie, eller se hvilke data som er samlet inn. Du vil få spørsmål før data sendes til Mozilla. Se også Personvernbestemmelse for Tilbakemelding-utvidelsen for mer info.
