Warum Firefox Sync?

Tippen ist anstrengend

Seien wir ehrlich, Tippen auf einem Mobilgerät ist schwierig. Verabschieden Sie sich vom Tippen langer URLs und Suchbegriffen, wenn die intelligente Adressleiste Ihre Chronik, Lesezeichen und Tabs sowohl von Ihrem Computer als auch von Ihrem Mobilgerät speichert und es Ihren so ermöglicht, schnell dorthin zu kommen, wo Sie hinwollen, und dabei nur wenig tippen müssen.

Fangen Sie nicht immer von vorne an

Verabschiedenen Sie sich vom Zurücksetzen und neu schreiben. Firefox Sync synchronisiert Ihre Passwörter und Einstellungen, so dass Sie sofort loslegen können.

Unterwegs

Nehmen Sie Firefox mit, egal wo Sie hingehen. Sie surfen auf Ihrem Computer im Internet und wenn Sie rausgehen, wartet schon alles auf Ihrem Mobilgerät auf Sie: Ihre Chronik, geöffnete Tabs, Lesezeichen, die intelligente Adressleiste – so wie Sie es zurückgelassen haben.

Erste Schritte

  1. Installieren Sie Firefox Sync auf Ihrem Computer

    Überprüfen Sie, ob auf Ihrem Computer Firefox 3.5 oder höher läuft und Ihr Mobilgerät ein Firefox-getestetes Gerät ist (aktualisieren Sie Firefox).

    • Laden Sie Firefox Sync herunter
    • Wenn der Download abgeschlossen ist, klicken Sie auf "Firefox neustarten", um die Installation des Add-ons Firefox Sync abzuschließen.
    • Wenn Firefox neu gestartet wurde, nutzen Sie das Menü "Extras" oder das Weave-Symbol rechts unten auf der Statusleiste, um Firefox Sync zu öffnen und sich anzumelden.

  2. Installieren Sie Firefox Sync auf Ihrem Mobilgerät

    • Wählen Sie in den Einstellungen auf die Schaltfläche "Add-ons".
    • Suchen Sie nach dem Add-on Firefox Sync und wählen Sie es aus. Melden Sie sich an, synchronisieren Sie und los geht's.
