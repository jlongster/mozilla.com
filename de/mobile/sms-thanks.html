Danke, dass Sie sich für Firefox entschieden haben!

Folgen Sie diesen Schritten, um Firefox herunterzuladen und mehr über Firefox Sync zu erfahren

  1. Finden Sie in Ihrem SMS-Eingang die Nachricht, die wir Ihnen zugeschickt haben.
  2. Öffnen Sie die Nachricht und wählen Sie den enthaltenen Link, um Firefox direkt auf Ihr Mobilgerät herunterzuladen.

* Firefox selbst ist kostenlos, aber es entstehen die üblichen Kosten für die Internetnutzung. Bitte überpüfen Sie dazu den Vertrag mit Ihrem Anbieter.

Während Sie warten …

  1. Richten Sie Firefox Sync ein und nehmen Sie Ihren Firefox mit, wo auch immer Sie hingehen. Nutzen Sie die Chronik, Lesezeichen, Passwörter und die geöffneten Tabs, die Sie auf Ihrem Computer haben, auch auf Ihrem Mobilgerät.
  2. Installieren Sie die Erweiterung Firefox Sync auf Ihrem Computer und melden Sie sich bei Firefox Sync an. Folgen Sie unseren vorgeschlagenen nächsten Schritten, sobald Firefox auf Ihr Mobilgerät heruntergeladen wird, und wählen Sie "Firefox Sync herunterladen", um die Installation abzuschließen. Alternativ dazu können Sie die Einstellungen-Schaltfläche manuell auswählen, die Add-ons-Verwaltung aufrufen, dann nach Firefox Sync suchen und es auf Ihrem Mobilgerät installieren.
