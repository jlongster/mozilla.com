Es wird persönlich

Passen Sie Firefox Ihren Wünschen an und genießen Sie den ersten mobilen Browser mit Add-ons.

Machen Sie sich Ihren Browser zu eigen.

Add-ons für Firefox Mobile machen Ihr Leben leichter, wenn Sie unterwegs sind. Installieren Sie Add-ons, die Sie mit sozialen Netzwerken verbinden, URL-Schreibfehler verbessern oder Ihre Fragen in eine andere Sprache übersetzen – all das ganz einfach von Ihrem Mobilgerät aus. Add-ons sind kostenlos und direkt von Ihrem Mobilgerät aus leicht zu installieren. Also, legen Sie gleich los.

Passen Sie Firefox direkt von Ihrem Mobilgerät aus an Ihre Wünsche an.

Personalisieren Sie Firefox, indem Sie Add-ons direkt von Ihrem Mobilgerät aus suchen und installieren. Ziehen Sie Ihren Finger einfach über den Bildschirm und wählen Sie die "Extras"-Schaltfläche auf der rechten Seite des Bildschirms. Tippen Sie auf die "Add-ons"-Schaltfläche, um mit den Anpassungen zu beginnen.

Möchten Sie ein Add-on für Firefox Mobile entwickeln?

Ihre Innovationen sind eine Voraussetzung für den Erfolg von Firefox. Wir haben es einfach gemacht, Add-ons für Firefox Mobile zu entwickeln. Besuchen Sie die Seite Für Firefox Mobile entwickeln und erfahren Sie mehr darüber.
