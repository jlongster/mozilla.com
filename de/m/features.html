Funktionen von Firefox

So bringt Firefox das Internet auf Ihr Mobilgerät.

Das einzig Wahre
Vergessen Sie das klobige, eingeschränkte mobile Internet, das Sie bisher kannten. Firefox Mobile bringt die Leistung und die Sicherheit, die Sie erwarten und dazu die ganzen Kleinigkeiten, den Sie kennen und lieben.
Wunderschön anzusehen
Tabs und die Steuerung sind an den Seiten des Bildschirms verstaut. Jetzt können Sie sehen, was die ganze Zeit versteckt war – die komplette Webseite. Genießen Sie die Aussicht.
Ein genialer Handgriff
Setzen Sie Lesezeichen für Ihre Lieblingsseiten mit einer Berührung. Ziehen Sie Ihre Finger über den Bildschirm, um auf Tabs zuzugreifen oder nützliche Add-ons zu installieren. Einfach genial.
Intelligente Adressleiste
Vergessen Sie das Eintippen von Adressen, überlassen Sie der Intelligente Adressleiste das Suchen und rufen Sie Ihre Lieblingsseiten im Nu auf.
Es wird persönlich
Die Add-ons sind mobil geworden. Passen Sie Ihren Browser an und machen Sie sich ihn zu eigen. Suchen Sie Add-ons und installieren Sie diese unterwegs.
Passwort-Manager
Passwörter am Mobilgerät einzutippen ist einfach schwierig. Warum sollte man das mehr als einmal tun? Wählen Sie aus, dass Passwörter "gespeichert" werden sollen und ersparen Sie sich das langwierige und schwierige Tippen.
Wunderbare Tabs
Ihre Tabs sind in Griffweite. Ziehen sie einfach Ihren Finger nach rechts, um Vorschaubilder von allen geöffneten Tabs anzuzeigen, so dass Sie schnell das finden, was Sie suchen.
Seien Sie ein Einheimischer… egal wo Sie sind
Suchen Sie nach dem besten Livemusik-Club in der Stadt? Einer alternativen Strecke zur Autobahn? Holen Sie sich Insider-Informationen und greifen Sie auf Landkarten zu – mit standortbezogenem Surfen.
Unterwegs
Holen Sie sich Firefox Sync und surfen Sie an Ihrem Computer im Internet und wenn Sie rausgehen, wartet schon alles auf Ihrem Mobilgerät auf Sie: Ihre Chronik, geöffnete Tabs, Lesezeichen, die intelligente Adressleiste – so wie Sie es verlassen haben.
