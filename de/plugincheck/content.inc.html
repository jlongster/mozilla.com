Häufig gestellte Fragen

Was ist ein Plugin?

  • Plugins treiben Videos, Animationen und Spiele an.
  • Sie werden von Unternehmen außerhalb von Firefox entwickelt, wie >Adobe Systems und >Apple.
  • Plugins aktualisieren sich nicht immer automatisch.

Warum sollte ich meine Plugins aktualisieren?

  • Alte Plugins können das Surfen im Internet unterbrechen und Ihre Zeit verschwenden.
  • Alte Plugins erhöhen Ihr Risiko für Angriffe durch Schadprogramme, Viren und andere Sicherheitsbedrohungen.
  • Aktualisierte Plugins verfügen über Verbesserungen, die das Internet für Sie besser und sicherer machen.

Wie kann Firefox mir dabei helfen?

In Zukunft wird Firefox für Sie Plugins aktualisieren. Bis dahin sollten Sie regelmäßig diese Seite besuchen und Ihre Plugins aktualisieren, um geschützt zu bleiben.

Welche Plugins habe ich?

Wir haben oben automatisch Ihre Plugins erkannt. Um Ihre installierten Plugins in Firefox zu sehen, führen Sie die folgenden Schritte aus:

  1. Öffnen Sie das Menü Extras.
  2. Wählen Sie Add-ons.
  3. Klicken Sie auf den Reiter Plugins.

Wie kann ich ein Plugin deaktivieren?

In Firefox:

  1. Öffnen Sie das Menü Extras.
  2. Wählen Sie Add-ons.
  3. Klicken Sie auf den Reiter Plugins.
  4. Klicken Sie auf ein Plugin in der Liste.
  5. Klicken Sie auf die Schaltfläche Deaktivieren.

Achtung: Wenn Sie ein Plugin deaktivieren, bedeutet dies, dass Sie bestimmte Dinge nicht mehr tun können. Wenn Sie beispielsweise Flash deaktivieren, können Sie sich keine Videos auf beliebten Video-Stream-Steiten ansehen.
