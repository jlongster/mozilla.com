Willkommen zu Sync in der Beta-Version von Firefox 4
Mit Sync haben Sie direkten Zugang zu all Ihren wichtigen Daten in Firefox,
einschließlich Ihrer Lesezeichen, Chronik, gespeicherten Passwörter und sogar offenen Tabs,
egal wo Sie Firefox nutzen, einschließlich all Ihrer Rechner und einer wachsenden Zahl von Mobiltelefonen.
Sie müssen eilig aufbrechen, haben aber gerade eine interessante neue Website gefunden?
Mit Sync können Sie auf der Arbeit oder unterwegs genau da weiter machen, wo Sie zu Hause aufgehört haben – wie Magie.
Um Sync zu aktivieren, müssen Sie ein Benutzerkonto anlegen. Also los geht's.
Wenn Sie die neueste Version von Firefox 4 Beta heruntergeladen haben, werden Sie aufgefordert, ein Benutzerkonto für Firefox-Sync anzulegen.
Wenn Sie schon Firefox-Sync verwenden, klicken Sie einfach auf „Ich verwende Sync bereits auf einem anderen Computer” und melden Sie sich an. Dann sind Sie schon startklar.
Wenn Sie Sync zum ersten mal verwenden, klicken Sie auf „Ich habe Sync bisher noch nicht verwendet”.
Geben Sie einen Benutzernamen … Passwort … und E-Mail-Adresse … ein.
Legen Sie jetzt eine Passphrase an.
Dieses mindestens 12 Zeichen lange Wort ist wie der Schlüssel zu Ihrem Schließfach, zu dem nur Sie Zugang haben.
Verlieren Sie es nicht, denn es schützt Ihre Daten.
Klicken Sie auf „Weiter“, es sei denn, Sie möchten die Daten auf Ihrem eigenen Server speichern.
Tippen Sie die Worte ein, so wie sie erscheinen.
Klicken Sie dann auf „Fertig“. Jetzt haben Sie ein Benutzerkonto für Firefox-Sync!
Ihre Firefox-Daten werden jetzt im Hintergrund synchronisiert.
Wenn Sie Sync zum ersten Mal einrichten, kann dies einige Zeit dauern, aber Sie können währenddessen im Internet weitersurfen.
Wenn die erste Synchronisation abgeschlossen ist, verwandelt sich das rotierende Symbol in der rechten unteren Ecke Ihres Browsers in das Sync-Symbol.
Ab jetzt aktualisiert Sync Ihre Daten immer wieder, um sicherzugehen, dass Ihre Daten synchron bleiben.
Vergessen Sie nicht, Firefox-Sync auf allen Ihren Rechnern einzurichten.
