Mit Mozilla Firefox, eBay-Edition können Sie direkt im Firefox Browser Artikel auf eBay suchen, beobachten und darauf bieten, ohne sich auf der eBay-Website zu befinden.

Sie werden herausfinden, warum Millionen von Menschen weltweit auf Firefox umgestiegen sind, um eine bessere, schnellere und sicherere Online-Erfahrung genießen zu können. Firefox, entwickelt und betrieben von Mozilla, ist eine globale Gemeinschaft, die gemeinsam daran arbeitet, das Web für alle Nutzer weiter zu verbessern.

Um Mozilla Firefox, eBay-Edition nutzen zu können, führen Sie bitte diese einfachen Schritte durch:

  1. Klicken Sie auf das eBay-Symbol auf Ihrer Firefox-Symbolleiste. Es öffnet sich links in Ihrem Browser eine Sidebar.
  2. Loggen Sie sich bei eBay ein, indem Sie auf die Schaltfläche „Einloggen“ auf der Sidebar klicken und geben Sie Ihren Mitgliedsnamen und Ihr Passwort ein. Wenn Sie noch kein eBay-Mitglied sind, klicken Sie bitte auf die Schaltfläche „Anmelden“.
  3. Jetzt können Sie loslegen und das große Angebot auf eBay direkt in Ihrem Firefox Browser durchstöbern und beobachten.

Weitere Informationen zur eBay-Edition finden Sie unter Funktionen und FAQ.

Machen Sie Firefox zu Ihrem Standard-Webbrowser

Für ein optimales Online-Erlebnis gehen Sie im Firefox-Menü zu Einstellungen und aktivieren Sie das Kontrollkästchen „Beim Starten überprüfen, ob Firefox der Standard-Browser ist“ aktiviert ist. So können Sie die Vorteile von Firefox jederzeit nutzen, selbst dann, wenn Sie auf einen Link in einer E-Mail drücken.
