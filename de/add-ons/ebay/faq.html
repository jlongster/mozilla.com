Wie kann ich Firefox Companion für eBay zu meinem Firefox Webbrowser hinzufügen?

Es ist leicht hinzuzufügen. Wenn Firefox 2.0 oder ein höhere Version bereits auf Ihrem Computer installiert ist, können Sie Companion wie jede andere Erweiterung herunterladen und installieren. Nach der Installation müssen Sie Firefox neu starten.

Wenn Firefox 2.0 noch nicht auf Ihrem Computer installiert ist, können Sie den Firefox Browser und eBay Companion als eine einzige Datei herunterladen.

Firefox Companion für eBay öffnet sich in einer Sidebar in Ihrem Firefox Browser.

Wie stelle ich mit Companion die Verbindung zu eBay her?
Wenn die Companion-Sidebar nicht geöffnet ist, klicken Sie auf das eBay Symbol auf der Symbolleiste, um die eBay-Sidebar zu öffnen. Klicken Sie auf die Schaltfläche „Einloggen“ und Sie werden auf die eBay-Seite zum sicheren Einloggen weitergeleitet, wo Sie aufgefordert werden, Ihren eBay-Mitgliedsnamen und Ihr Passwort einzugeben. Nach Anmeldung wird Companion die Verbindung zu eBay automatisch bei jedem Start von Firefox herstellen.
Wie kann ich verhindern, dass eBay Companion automatisch bei jedem Start von Firefox die Verbindung zu eBay herstellt?
Klicken Sie auf das Auswahlmenü rechts neben dem eBay-Symbol auf der Symbolleiste. Gehen Sie zu „Optionen“. Klicken Sie auf „Mehr“ und aktivieren Sie „Bitte die Verbindung zum Firefox Companion für eBay trennen, wenn ich Firefox schließe“.
Welche Systemvoraussetzungen und welches Betriebsystem wird für Companion benötigt?
Firefox Companion für eBay funktioniert mit jedem Computer, auf dem Firefox installiert ist. Das schließt auch Apple Mac, Linux und Windows Plattformen mit ein. Sie müssen über Administrationsrechte verfügen, um Companion zu Ihrem Firefox Webbrowser hinzufügen zu können.
Wie öffne ich Firefox Companion für eBay?
Klicken Sie auf das Firefox Companion für eBay Symbol auf Ihrer Firefox Symbolleiste und eine Sidebar in Ihrem Firefox Webbrowser öffnet sich.
Wie kann ich nach einem Artikel von Companion aus suchen?
Benutzen Sie einfach die Companion-Suchleiste, um Artikel, die Sie interessieren, zu finden. Damit werden Sie direkt zur eBay Webseite weitergeleitet, wo Sie für den Artikel bieten können.
Kann ich mit eBay Companion ein Gebot erhöhen?
Ja. Klicken Sie einfach auf die Schaltfläche „Gebot erhöhen“ neben dem Artikel, für den Sie Ihr Gebot erhöhen wollen. Dadurch werden Sie sofort zur Gebotsseite für diesen Artikel weitergeleitet und können ein Gebot für diesen Artikel abgeben.
Warum werde ich aufgefordert, mich bei eBay anzumelden, wenn der eBay Companion mit eBay verbunden ist?
Zu Ihrer eigenen Sicherheit werden Sie aufgefordert, sich bei eBay anzumelden, unabhängig von dem Verbindungsstatus von eBay Companion.
Ist Firefox Companion für eBay sicher?
Ja, mit Firefox Companion können Sie sich immer sicher sein, dass Sie sich auf einer echten Seite von eBay aufhalten und nicht auf einer „Phishing“-Seite. Wenn Sie Firefox nutzen, schützen Sie sich ebenfalls vor Popup-Fenstern, Spyware und anderen Gefahren im Internet.
Wie deinstalliere ich die Firefox Companion für eBay?

Schade, dass Sie Ihre Meinung geändert haben. Mit diesen einfachen Schritten können Sie die eBay-Companion deinstallieren.

  1. Öffnen Sie die "Add-ons" unter "Extras"
  2. Klicken Sie auf "Erweiterungen" oben in der Leiste
  3. Suchen Sie die Firefox Companion für eBay in der Liste und klicken Sie darauf
  4. Wählen Sie "Deaktivieren"
  5. Starten Sie Firefox neu
Wie kann ich andere Artikel, die ich beobachte, überwachen?
Zur Auswahl von Artikeln, auf die Sie geboten haben, die Sie zurzeit beobachten oder verkaufen, benutzen Sie das Auswahlmenü oben bei dem Companion.
Kann ich herausfinden, wie viele Beobachter für einen Artikel, den ich verkaufe, zur Zeit den Artikel anschauen?
Ja, halten Sie Ihre Maus einfach über den Artikel, der Sie interessiert, und es erscheint eine Quickinfo mit der Zahl der Beobachter und Besucher.
Kann ich von dem Companion aus Artikel verkaufen?
Nein. Sie können aber das Auswahlmenü der eBay Schaltfläche benutzen, um schnell über eine Verknüpfung auf die eBay-Verkaufsseite zuzugreifen.
Kann ich die Größe der Firefox Companion für eBay Sidebar verändern?
Ja. Positionieren Sie Ihren Cursor einfach auf die Sidebar-Trennlinie, halten Sie die linke Maustaste gedrückt und bewegen Sie die Trennlinie bis zur gewünschten Breite.
Für welche Länder bietet Firefox Companion für eBay Support an?
Zunächst für Deutschland, Frankreich und Großbritannien mit möglicher Ausweitung auf andere Länder zu einem späteren Zeitpunkt.
Wenn ich nicht in einem dieser Länder bin, kann ich trotzdem den Companion benutzen?
Ja, das können Sie. Jedoch ist Support nur für Deutschland, Frankreich und Großbritannien verfügbar.
