Mehr Spaß im Internet mit dem Firefox Browser

Mit Firefox haben Sie mehr vom Internet und von eBay. Finden Sie heraus, wie Sie optimal und sicher surfen können und Ihre Aktivitäten auf eBay immer im Blick haben.

Schneller Zugriff auf eBay

Sobald Sie eingeloggt sind, sind Sie durch das eBay-Logo in Ihrem Browser nur einen Mausklick von eBay entfernt. Über das eBay-Logo öffnet und schließt sich die Sidebar und gewährt Ihnen schnellen Zugang zu Verlinkungen auf die eBay-Seite.

  • Sidebar

    Die Sidebar der eBay-Edition enthält alle wichtigen Informationen aus Mein eBay. Sie können Ihren Verkaufs-, Kauf- und Bewertungsstatus sofort überprüfen. Nutzen Sie die Vorteile der Schnellsuchleiste und der Verknüpfung mit Meine eBay Welt.

  • Erinnerungen

    Erinnerungsmeldungen machen Sie rechtzeitig auf Angebote aufmerksam, die Sie beobachten bzw. auf die Sie geboten haben.

    1. In einem kleinen Fenster oder in der Sidebar Diese Erinnerung dient dazu, dass Sie wichtige Meldungen zum richtigen Zeitpunkt erhalten. Wenn ein Angebot, das Sie beobachten, dem Ende entgegengeht oder wenn Sie überboten wurden, werden Sie sofort von dem Meldungsfeld, das in der unteren rechten Ecke Ihres Browserfensters erscheint, benachrichtigt.
    2. Bei weniger wichtigen Erinnerungen – wenn zum Bespiel Ihr Gebot automatisch erhöht wurde – leuchtet das eBay-Logo in der Firefox Toolbar (eBay-Logo) auf, um Sie auf eine Statusänderung aufmerksam zu machen. Sie können dann die Sidebar öffnen, um die Veränderungen anzeigen zu lassen.

    Sie können selbst bestimmen, welche Meldungen Sie erhalten möchten. Nehmen Sie die Einstellungen dazu unter „Optionen“ vor.

Kein Auktionsende mehr verpassen 

Ganz gleich, auf welchen Seiten Sie sich im Internet bewegen Sie haben Ihre eBay-Aktivitäten jederzeit im Blick: Beobachten, Kaufen oder Verkaufen.

  • Ausführliche Informationen in der Sidebar

    Sie sehen die wichtigsten Informationen zu allen Angeboten, wie zum Beispiel die Anzahl der Gebote und der aktuelle Preis. Weitere Informationen zu einem Angebot erhalten Sie, indem Sie mit Ihrer Maus darüber fahren.

  • Intelligente Aktualisierung

    Die Informationen werden in regelmäßigen Abständen aktualisiert. Die Aktualisierungsabstände werden immer kürzer, je näher das Angebotsende des Artikels, den Sie beobachten, auf den Sie geboten haben oder den Sie verkaufen, rückt.

  • Schneller Zugriff auf wichtige Funktionen

    Mit einem Klick auf die rechte Maustaste können Sie beispielsweise die Bewertungsseite eines Verkäufers aufrufen oder einen beobachteten Artikel aus „Mein eBay“ entfernen.

Sicherer Surfen 

Mit dem Sicherheits-Check der eBay-Edition des Mozilla Firefox-Browsers schützen Sie Ihre eBay- und PayPal-Mitgliedsdaten. Der Sicherheits-Check warnt Sie, wenn Sie im Begriff sind, Ihr eBay- oder PayPal-Passwort auf einer Webseite einzugeben, die zwar aussieht wie eine eBay- oder PayPal-Seite, jedoch nicht von eBay oder PayPal ist.

Mit dem Firefox Browser schützen Sie sich ebenfalls vor Popup-Fenstern, Spyware und anderen Gefahren im Internet.
