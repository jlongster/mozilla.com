Die Beta-Version von Firefox 4 enthält das Add-on „Feedback“, das auf Mozilla Test Pilot basiert

Sobald Sie die Beta-Version von Firefox 4 installieren, wird Ihnen das Add-on „Feedback“ in der rechten oberen Ecke auffallen. Sie können es jederzeit verwenden, um uns direkt Rückmeldung zu geben. Sie werden außerdem über Anwender-Studien informiert (die mit Mozilla Test Pilot durchgeführt werden), die Informationen darüber sammeln, wie Sie Firefox verwenden, aber nicht darüber, welche Websites Sie besuchen. Sie haben die Kontrolle über Ihre Teilnahme und können sich entscheiden, sich von einer Studie abzumelden, oder nachsehen, welche Daten gesammelt wurden. Sie werden gefragt, bevor irgendwelche Daten an Mozilla gesendet werden. Weitere Informationen finden Sie in der Datenschutzerklärung des Add-ons „Feedback“.
