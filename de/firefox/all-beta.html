Helfen Sie uns, indem Sie den zukünftigen Firefox testen!

Die aktuelle Beta-Version von Firefox ist jetzt als Vorab-Version verfügbar. Testen Sie sie ausgiebig und teilen Sie uns bitte mit, was Sie darüber denken!

Wichtige Informationen

Bitte lesen Sie Folgendes, bevor Sie die Version herunterladen:

  • Dies ist eine Vorab-Version: Kompatibilitätsprobleme mit einigen Websites und Add-ons können auftreten.
  • Sollten Sie auf Probleme stoßen, informieren Sie uns bitte darüber oder erstellen Sie einen Fehlerbericht.
  • Weitere Informationen finden Sie in den Versionshinweisen und in der Liste der bekannten Probleme .

Sie möchten keine Beta-Version verwenden? Holen Sie sich den aktuellen Firefox.
