Jetzt fest in Firefox 4 Beta eingebaut: Sync

Gehen Sie von der Arbeit über Zuhause zu Ihrem Mobilgerät und zurück! Eine Auswahl der neuen Funktionen

Firefox 4 steht ein großartiger Start bevor. Hier sind einige weitere Schlüsselfunktionen, die Sie nicht verpassen sollten:

Flüssige Animation

Neue JavaScript-Werte bedeuten, dass Firefox zahlenlastigen Code effizienter ausführen kann, was flüssigere Animationen bedeutet.

SQLite 3.7.0

Die intelligente Adressleiste, der Download-Manager und die IndexedDB-Funktionen laufen jetzt schneller und flüssiger, dank einer Aktualisierung von SQLite, einer SQL-Datenbank-Engine.
