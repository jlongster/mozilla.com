Sagen Sie uns, was Sie denken

Als Beta-Tester sind Sie Teil des Teams, das Firefox 4 voranbringt. Sagen Sie uns, wenn Sie etwas mögen oder nicht mögen, indem Sie auf die Schaltfläche Feedback klicken. Sie werden auch gebeten, an anonymen Anwender-Studien teilzunehmen oder nebenher die Umfrage auszufüllen.

Wie funktionieren die Anwender-Studien?

Anwender-Studien sammeln anonyme Statistiken darüber, wie Sie Firefox nutzen – aber nie darüber, was Sie im Internet sehen oder tun. Sie haben die Kontrolle und ohne Ihre Erlaubnis werden keine Daten an die Server von Mozillas Add-on Test Pilot gesendet. Diese Daten werden verwendet, um uns dabei zu helfen, Firefox 4 zu gestalten.

Was ist, wenn ich nicht teilnehmen möchte?

Sie werden informiert, bevor irgendwelche Anwender-Studien beginnen, Daten zu sammeln, und bevor irgendwelche Informationen an Mozilla gesendet werden. Sie können sich von einer Studie abmelden oder alle Studien deaktivieren, indem Sie auf die Schaltfläche Feedback klicken und Ihre Einstellungen bearbeiten.
