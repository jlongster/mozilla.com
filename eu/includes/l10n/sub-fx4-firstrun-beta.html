Kode irekiko proiektu eta komunitate gisa, zure iritzia behar dugu hurrengo belaunaldiko nabigatzailea sortzeko.
Ezagutu Firefox 4 proiektuan gogor lan egin dutenetariko batzuk.
HTML 5, File APIs eta IndexedDB bezalako web-estandarretan dihardut lanean.
Inguruko nabigatzaile onena eta seguruena egitera goaz!
Firefox 4 beta probatzen lan egiten dugu. Zure iritzia ezagutu nahi dugu.
Nabigatzailearen goialdean eskuman feedback botoi bat ikusiko duzu…
…botoi hau erabiliz esan ahal diguzu Firefox 4tik zer gustatu zaizun eta zer ez.
Esaidazu zer deritzozun.
Firefox 4 beta programan ari naiz lanean.
Firefox 4rako edukien segurtasun politikan lan egiten dut.
Kraskadurak eragiten dituzten erroreak konpontzen gabiltza.
David naiz eta Firefox 4ko garatzaileentzako tresnetan lan egiten dut, zure iritzia jakin nahiko nuke.
Ahalik eta iritzi gehien bildu nahi ditugu.
Firefox 4ren sinkronizazio zerbitzariekin lan egiten dut.
Firefox Sync-ek Firefoxen esperientzia edonora eramateko aukera ematen dizu.
Eta benetan ezagutu nahi dugu zure ikuspuntua.
JetPack da nire proiektua, eta gehigarriak errazago garatzeko eta erabiltzeko balio du.
Firefox 4ren gehigarri-kudeatzaile berrian egin dut lan.
Firefox 4ri buruzko zure iritzia behar dugu, zer maite duzun eta zer gorrotatzen.
Lagunduiguzu Firefox hobetzen!
Zure iritzia behar dugu.
