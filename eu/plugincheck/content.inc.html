Aztertu zure Plugin-ak

Hirugarrengoen plugin-ak eguneratuta mantenduz Firefoxek era seguruan eta abiada bizian funtzionatuko du.

Non dago FLASH?

Zure segurtasunerako, Firefoxek Flash-en bertsio zaharkitua desgaitu du. >Eguneratu azken bertsiora.


  • Adi, nabigatzailearen bertsio berriago bat eskuragarri dago. Eguneratu nabigatzailea.
  • Orri honek hobeto funtzionatzen du nabigatzaileen azken bertsioekin: Firefox 3.6+, Opera 10.5+, Safari 4+, Chrome 4+, edo IE 8+
  • Plugin-en azterketaren Internet Explorerrentzako sostengua mugatua da. Erabili Tresnak > Gehigarriak kudeatu
  • 1. urratsa: Egin klik Eguneratu loturan plugin bat eguneratzeko.
  • 2. urratsa: Burutu gomendatutako eguneraketa guztiak nabigatzailea berrabiarazi baino lehen.

Ohiko Galderak

Zer da plugin bat?

  • Plugin-ek bideoak, animazioak eta jokoak funtzionarazi ditzakete.
  • Firefoxez kanpoko enpresek garatzen dituzte, hala nola >Adobe Systems eta >Apple.
  • Plugin-ak ez dira beti automatikoki eguneratzen.

Zergatik eguneratu beharko nituzke plugin-ak?

  • Plugin zaharrek nabigazioa oztopatu eta denbora galarazi dezakete.
  • Plugin zaharrek malware, birus eta bestelako segurtasun-mehatxuen erasoak jasateko arriskua areagotzen dute.
  • Eguneratutako plugin-ek hobekuntzak dakartzate, web-a hobea eta seguruagoa izan dadin ahalbidetuz.

Nola lagun nazake Firefoxek?

Etorkizunean Firefoxek plugin-ak automatikoki eguneratuko ditu. Bitartean, bisitatu noizean behin orri hau eta eguneratu plugin-ak seguru mantentzeko.

Zein plugin dauzkat?

Goian zerrendatutako plugin-ak antzeman ditugu automatikoki, Firefoxen instalatutako plugin-ak ikusteko jarraitu urrats hauek:

  1. Ireki Tresnak menua.
  2. Hautatu Gehigarriak.
  3. Klikatu Plugin-ak fitxan.

Nola desgaitu dezaket plugin bat?

Firefoxen:

  1. Ireki Tresnak menua.
  2. Hautatu Gehigarriak.
  3. Klikatu Plugin-ak fitxan.
  4. Egin klik zerrendako plugin batean.
  5. Sakatu Desgaitu botoia.

Kontuz: plugin bat desgaitzeak esan nahi du aurrerantzean ezingo dituzula egin zenbait gauza. Adibidez, Flash desgaitzen baduzu, ezingo duzu bideorik ikusi streaming gune ezagunetan.
