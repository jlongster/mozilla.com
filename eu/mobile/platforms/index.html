Ez dago zure mugikorra hemen?

Zure gailu mugikorra zerrenda honetan agertu ez arren, hala ere Firefox deskargatu eta erabil dezakezu bertan. Zure gailuak nahiko RAM eduki behar du (128 MB edo gehiago) Firefox erabiltzeko. Firefoxekin probatu den gailu bat badaukazu, hura deskargatzera animatu nahi zaitugu, bere funtzionaltasun osoa erabili ahal izango baituzu.

Zure mahaigaineko PCan ere Firefoxen bertsio bat instala dezakezu, eta horrela esperimentatu egin dezakezu, zure iritzia eman, gehigarriak idatzi eta parte hartu:

  • Windows
  • Mac OS X
  • Linux
