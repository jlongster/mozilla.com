Joan kalera eta jarri martxan

Eskuratu Firefox Sync eta partekatu zure historia, laster-markak, fitxak eta gehiago zure PCaren eta zure telefono mugikorraren artean.

Eskuratu Firefox Sync gehigarria orain

Zertarako behar dut Firefox Sync?

Tekleatzea gogorra da

Onar dezagun, mugikorrean tekleatzea zaila da. Esan agur URL luzeak eta bilaketa-hitzak tekleatzeari, Barra Txundigarriak zure historia, laster-markak eta fitxak gogoratzen baititu, bai zure PCkoak zein mugikorrekoak, eta berehala eramaten zaitu nahi duzun lekura eta oso gutxi tekleatuta.

Ez hasi zerotik

Esan agur berrezarpenei eta berriro tekleatu behar izateari. Firefox Sync-ek zure Firefoxeko pasahitzak eta hobespenak sinkronizatzen dituenez, lanari azkar eta prest ekin diezaiokezu.

Joan kalera eta jarri martxan

Eraman zure Firefox zurekin nahi duzun tokira. Nabigatu webean zure PCtik, irten kalera eta edonora joan, eta orduan ere dena prest eduki zure telefonoan: zure historia, irekitako fitxak, laster-markak, Barra Txundigarria... den-dena etxean utzi duzun bezala.

Nola hasi

  1. Instalatu Firefox Sync zure PCan

    Egiaztatu zure PCak Firefox 3.5 edo goragokoa duela eta zure telefonoa Firefoxekin probatutako gailua dela (eguneratu Firefox).

    • Deskargatu Firefox Sync
    • Deskarga burutu ondoren, sakatu "Berrabiarazi Firefox" Firefox Sync gehigarriaren instalazioa amaitzeko.
    • Firefox berrabiarazi ondoren, erabili "Tresnak" menua edo egoera-barraren behean eskuinaldean ageriko den Firefox Sync ikonoa Firefox Sync ireki eta izena emateko.

  2. Instalatu Firefox Sync gehigarria zure mugikorrean

    • Joan zure hobespenetara, hautatu gehigarrien botoia.
    • Bilatu eta hautatu Firefox Sync gehigarria. Eman izena, sinkronizatu eta kalera irteteko prest.
