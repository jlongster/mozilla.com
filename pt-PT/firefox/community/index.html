Involva-se com a Mozilla

Somos uma comunidade global que trabalha para tornar a Web um melhor local para todos.

Em que é que deseja ajudar? Fantástico! Não precisa de ser um guru(ou o que isso significa) em C++ e não precisa perder muito tempo.

Maneiras Fáceis para se Envolver

  • Diga à sua família, amigos e colegas sobre os productos Mozilla
  • Ajude outras pessoas nos nossos grupos de discussão e foruns de suporte
  • Jnte-se à comunidade de marketing e ajude-os nas nossas campanhas
  • Assegure-se que submete dados de crash à equipa de desenvolvimento
  • Contacte sítios web sites que não funcionam correctamente com os productos Mozilla
  • Faça uma doação à Fundação Mozilla

Para Mais Orientações Técnicas

  • Junte-se à equipa de qualidade e crie bugs e outros problemas que devem ser reparados
  • Ajude a escrever ou edite documentação para developers
  • Ajude a escrever ou edite documentação para utilizadores finais
  • Contribua para a riqueza do ecossistema Firefox desenvolvendo um Extra
  • Repare um bug ou contribua com código para o projecto
