Apertem os cintos

O Firefox 4 inclui agora grandes melhoramentos de desempenho, incluindo o novo motor JavaScript JägerMonkey. De arranques rápidos a renderização de gráficos e melhoramento da velocidade de carregamento de páginas, vai notar a diferença instântaneamente.

3D na Web

O WebGL traz os gráficos 3D ao Firefox, abrindo a porta aos desenvolvedores para criarem jogos e novos tipos de visualizações e experiências para a Web.

Fabulosos tipos de letras

As funcionalidades dos tipos de letra OpenType do Firefox 4 dão aos desenvolvedores mais controlo sobre uma variedade de funcionalidades de tipos de letra—como kerning ou ligatures—para criar bonitos sítios web.

Actualização de formulários

O Firefox 4 torna os formulários bonitos e de fácil implementação. Com funcionalidades como uma nova lista de atributos que activam a conclusão automática e validação de formulários HTML5, a nova API de formulários fornece as ferramentas para que o desenvolvimento de formulários chatos seja coisa do passado.
