Funcionalidade Showcase

O Firefox 4 está pronto para um grande começo. Aqui estão algumas funcionalidades chave que não quer perder:

Gráficos suaves

Novos valores de JavaScript significam que o Firefox pode executar códigos numéricos mais exigentes com maior eficiência, resultando em gráficos suaves.

SQLite 3.7.0

As funcionalidades da barra fantástica, do gestor de transferências e da base de dados indexada são agoras mais rápidas e suaves graças a uma actualização do of SQLite, um motor SQL de bases de dados.
