Sobre o Firefox

Porquê um Firefox para telemóveis?
Trazer o Firefox para os telemóveis é o passo seguinte na missão da Mozilla de trazer o acesso à web para que todos possam aceder, independentemente do dispositivo ou da localização. Adicionalmente a Mozilla deseja poder fornecer plataformas de interesse para que os desenvolvedores criem conteúdos e novas aplicações. O mais avançado navegador web disponível, o Firefox é rápido, seguro, personalizável e sincroniza perfeitamente com o Firefox no seu PC.
Tem custos?
Não. O Firefox foi criado como parte da fundação sem fins lucrativos Mozilla que se comprometeu com a aberura, inovação e oportunidade da Internet. Achamos que software gratuito e de código aberto como o Firefox é fundamental para manter a internet como um recurso global. Saber mais.
Em que é que o navegador Firefox é diferente do meu navegador móvel actual?
A Mozilla está a trazer a experiência do Firefox para os dispositivos móveis. Construído sobre o motor de navegação mais moderno que existe, o Firefox traz funcionalidades familiares como a “Barra fantástica”, Gestor de senhas, Marcadores com um clique, navegação com o conhecimento da localização e muito mais. O Firefox é também o primeiro navegador web móvel a oferecer extras, permitindo aos utilizadores a personalizar o seu navegador.
O Firefox está disponível no meu idioma?
O Firefox para dispositivos móveis está actualmente disponível em mais de 25 idiomas, e traduções adicionais estão a decorrer. Veja a nossa lista completa dos idiomas disponíveis.
Quem faz o Firefox?
O Firefox foi produzido pela fundação sem fins lucrativos Mozilla com a ajuda de uma comunidade global. Devido às contribuições da comunidade, adicionamos novos extras e actualizações que tronam o Firefox mais rápido, seguro e potente. Quer ajudar? Visite a página Desenvolver para o Mobile.

Segurança

Como é que o Firefox me mantém seguro enquanto navego com o meu telemóvel?
O processo de código aberto da Mozilla faz com que reúna centenas de entendidos em segurança por todo o mundo, e o Firefox inclui funcionalidades como o ID do sítio web que verifica sítios suspeitos para assegurar que eles são o que dizem ser. Definições de segurança personalizáveis para gerir senhas, cookies, carregamento de imagens e instalar extras dão-lhe controlo total sobre a sua experiência de navegação. Saber mais.

Transferir & Dispositivos

Como transfiro o Firefox para o meu telemóvel?
Visite a página inicial do Mobile em Mozilla.com para enviar uma mensagem de texto que contém a ligação para transferir o Firefox para o seu telemóvel. Ou então, vá a mozilla.com/m com o seu telemóvel e transfira o Firefox directamente para o seu telefone.
O Firefox funciona no meu telemóvel?
O Firefox ié compatível com o Nokia N810 e N900. Veja a lista completa de dispositivos suportados.
Posso transferir o Firefox para o meu telemóvel mesmo que ele não seja suportado?
Pode executar o Firefox num telefone Maemo mesmo que o seu telefone não esteja na nossa lista de dispositivos testados, mas sem a garantia de funcionamento total. Encorajamos-o a experimentar o Firefox num dispositivo suportado para que possa experimentar as suas totais capacidades. Veja a lista completa de dispositivos suportados.
Posso transferir o Firefox para o iPhone?
De momento o Firefox não está disponível para o iPhone.

Usar o Firefox

Como é que utilizo a interface do Firefox para navegar na web?
Para ter espaço no ecrã do seu telemóvel,os controlos do navegador Firefox são deslocados para os lados do ecrã. São acedidos deslocando movendo o seu dedo da esquerda ou da direita no ecrã. Os separadores estão do lado esquerdo do ecrã, enquanto que os marcadores e as opções estão do lado direito. Para poder ver melhor um sítio a barra de endereço também desaparece ao deslocá-la para baixo, mas aparece de forma rápida movendo para cima ou para o lado.
Como é que aumento e diminuo?
Toque duas vezes no ecrão com o dedo para aumentar. Para diminuir volte a tocar duas vezes com o dedo.
Como é que adiciono uma página web aos marcadores?
Vá para o sítio que deseja adicionar aos marcadoes. Clique no botão “estrela” do marcador do lado direito do seu ecrã.
Como é que abro um novo separador?
Toque no botão de novo separador "mais" no lado esquerdo do seu ecrã para abrir um novo separador. os separadores aparecem como miniaturas para que possa ver que página seleccionar.

Funcionalidades

O que há de tão especial na Barra fantástica?
A barra fantástica leva-o aonde quer ir de uma forma rápida. Toque na barra fantástica e comece a escrever. À medida que escreve, irão aparecer os sítios web que visitou recentemente ou que estejam nos marcadores. A barra fantástica aprende à medida que a utiliza, adaptando-se às suas preferências e tornando a sua navegação mais pessoal. Também pode tocar na barra fantástica e escrever um URL ou um termo de pesquisa e começar a explorar a Web. Ou, toque em um dos motores de pesquisa no fundo do seu ecrã para ir para o destino desejado. 
O que é que são os extras do Firefox e como é que os posso ter no meu telemóvel?
Os extras deixam-o personalizar o Firefox adicionando funcionalidades e personalizações que ajudam a tornar o seu navegador só seu. Seleccione o botão opções “gear” no lado direito em baixo do seu ecrã, depois escolha o botão extras “puzzle piece” e pesquise e instale os extras que são os mais indicados para si directamente no seu telemóvel.
Como é que posso partilhar os meus marcadores, histórico, separadores do Firefox entre o meu PC e o meu telemóvel?
O extra Firefox Sync traz a sua experiência no Firefox para o seu telemóvel. Primeiro, crie uma conta Firefox Sync no seu PC. Depois vá ao Firefox do seu telemóvel e procure e instale o extra Firefox Sync . Ao escrever a a informação na sua conta do Firefox Sync e sincronizar, as suas preferências do Firefox, histórico e muito mais serão partilhadas entre o seu PC e o seu telemóvel. para saber mais, visite a página Synchronize.
Preciso de obter o extra Firefox Sync para utilizar o Firefox no meu telemóvel?
Não. Embora sincronizar o Firefox entre o seu PC e o telemóvel com o Firefox Sync, na nossa opinião, é super útil, é um serviço opcional que pode ou não escolher. Pode ainda desfrutar do Firefox em que confia sem o ter. O Firefox Sync permite partilhar o histórico, marcadores, preferências e muito mais, melhorando a sua experiência global de navegação onde quer que esteja.
O que é a navegação com conhecimento da localização? O Firefox segue os meus passos?
A navegação com conhecimento da localização poupa-lhe tempo ao permitir que os sítios web lhe perguntem onde está. Se escolher partilhar a sua localização com o sítio web, ele pode usar essa informação para encontrar pontos de interesse e indicar dados adicionas e úteis como mapas da áres onde se encontra. É tudo opcional o – Firefox não partilha a sua localização sem a sua autorização. Saiba mais sobre a navegação por conhecimento da localização e veja a política de privacidade no serviço de localização do Google.
Quão precisa é a localização?
A exactidão varia de localização para localização. Em alguns locais, os nossos fornecedores do serviço poderão indicar-lhe uma localização até alguns metros. No entanto, noutras áreas poderá ser muito mais que isso. Todas as localizações devolvidas pelos nossos fornecedores do serviço são apenas estimativas e não garantimos a precisão da localização.

Questões do desenvolvedor

Um extra que fiz para o Firefox pode ser transferido para o telemóvel? Se não, como é que o posso alterar?
Embora o Firefox para computador de secretária e móvel tenham a mesma plataforma, existem pequenas diferenças que fazem com que os desenvolvedores alterem os seus extras. Estas alterações são geralmente simples, para saber mais visite a página Desenvolver para o Mobile .
Que motor de navegação usa o Firefox?
As versões do Firefox para computador de secretária e móvel utilizam o motor de renderização Gecko. A versão móvel do Firefox utiliza a versão Gecko 1.9.2, que é o melhor motor de navegação que a Mozilla já criou e inclui funcionalidadescomo o desemprenho melhorado de JavaScript performance, suporte a geolocalização e muito mais.
Que motor de JavaScript usa o Firefox?
O Firefox inclui o novomotor de JavaScript “TraceMonkey” , um compilador JIT (“just in time”) avançado que usa tecnologia de rastreio. O motor do TraceMonkey torna as aplicações web mais rápidas compilando o código conforme o executa, permitindo que vários tipos de aplicações de uso intensivo da CPU possam ser executadas em velocidade quase nativa.
Como é que desenvolvo um sítio web de conhecimento da localização para o Firefox?
O Firefox implementou o primeiro rascunho da especificação da geolocalização da W3C. Uma antevisão de como desenvolver conhecimento da localização de sítios web encontra-se aqui.
O Firefox no telemóvel suporta HTML 5?
HTML 5 é a próxima revisão do HTML. A especificação HTML 5 é um trabalho em curso, mas existem diversas e relacionadas funcionalidades do HTML 5 que vão permitir aos desenvolvedores web a criação de novos sítios web e aplicações. O Firefox inclui importantes funcionalidades como Navegação por conhecimento da localização, o <canvas> e etiquetas <svg>, trabalhadores web e armazenamento desligado.
O Firefox no meu telemóvel suporta plugins?
Inicialmente não. O suporte para plugins no Firefox para Maemo está desactivado. A maioria das páginas web utiliza o plugin Adobe Flash, o desempenho do plugin não corrspondia aos nossos standards, e o impacto sobre a interactividade e desempenho numa página web era negativo. Era sentido especialmente em páginas com diversas instâncias do plugin. Os utilizadors avançados podem activar os plugins apenas para testes e experiências. Estamos a trabalhar em maneiras para que o utilizador controle em quais os sítios activar os plugins, assim como sítios, como o YouTube, que funcionam bastante bem. Esta capacidade será feita como um extra para o navegador.
