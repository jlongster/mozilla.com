Personalize o Firefox e disfrute do primeiro navegador a oferecer extras para o mobile.

Faça o seu navegador seu.

Os extras do Mobile ajudam a fazer a vida correr como uma brisa quando está em movimento. Instale extras que partilham info com redes sociais, corrigem erros de escrita, ou traduzem as suas perguntas para outros idiomas, tudo a partir do seu telefone móvel. Os extras são grátis e de fácil instalação a partir do seu telemóvel.

Personalize através do seu telemóvel.

Personalize o seu Firefox pesquisando e instalando extras a partir do seu telemóvel. Navegie através do ecrã e seleccione o botão Ferramentas do lado direito do seu ecrã. Clique no botão dos extras para começar.

Deseja desenvolver um extra para o mobile?

As suas inovações são essenciais para o sucesso do Firefox. Tornámos fácil o desenvolvimento de extras para o mobile. Para saber mais, visite a página Desenvolver para o Mobile.
