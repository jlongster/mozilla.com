Perguntas frequentes

O que é um plugin?

  • Os Plugins servem para vídeos, animações e jogos.
  • São feitos fora do Firefox por companhias como >Adobe Systems e >Apple.
  • Os Plugins nem sempre actualizam automaticamente.

Porque é que tenho que actualizar os meus plugins?

  • Os plugins antigos podem interromper a navegação e fazê-lo perder tempo.
  • Plugins antigos aumentam o risco de aqtaques de malware, vírus e outras ameaças de segurança.
  • Os plugins actualizados têm melhoramentos que tornam a web melhor e mais segura para si.

Como é que o Firefox me pode ajudar?

No futuro, o Firefox irá actualizar-lhe os seus plugins. Até lá, deve ver regularmente esta página e actualizar os seus plugins para se manter seguro.

Que plugins é que eu tenho?

Detectámos automaticamente os plugins acima, para ver os plugins instalados no Firefox siga estes passos:

  1. Abra o menu Ferramentas.
  2. Escolha Extras.
  3. Clique no separador plugins.

Como é que desactivo um plugin?

No Firefox:

  1. Abra o menu Ferramentas.
  2. Escolha Extras.
  3. Clique no separador plugins.
  4. Clique num plugin da lista.
  5. Clique no botão Desactivar.

Cuidado: desactivar um plugin significa que não poderá mais fazer certas coisas. Por exemplo, se desactivar o Flash, não poderá ver vídeos em sítios populares de streaming de vídeo.
