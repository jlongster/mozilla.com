Ελέγξτε τις προσθετες λειτουργιες σας.

Έχοντας ενημερωμένες τις πρόσθετες λειτουργίες τρίτων κατασκευαστών, βοηθάτε τον Firefox να εκτελείται απρόσκοπτα και με μεγαλύτερη ασφάλεια.

Σας λείπει το FLASH;

Για να προστατέψει την ασφάλεια σας, ο Firefox απενεργοποίησε την παρωχημένη έκδοση του Flash που έχετε. Σας παρακαλούμε να >αναβαθμιστείτε με την πιο πρόσφατη έκδοση του.


  • Προσοχή, υπάρχει διαθέσιμη νεότερη έκδοση του περιηγητή σας. Σας παρακαλούμε να αναβαθμίσετε τον περιηγητή σας.
  • Αυτή η σελίδα λειτουργεί καλύτερα με τους πιο σύγχρονους περιηγητές: Firefox 3.6+, Opera 10.5+, Safari 4+, Chrome 4+, ή IE 8+
  • Η υποστήριξη του ελέγχου πρόσθετων λειτουργιών είναι περιορισμένη στον Internet Explorer. Χρησιμοποιήστε και την «Διαχείριση πρόσθετων» μέσα από τα «Εργαλεία»
  • Πρώτο βήμα: Κάντε κλικ στο κουμπί Ενημέρωση για να ενημερώσετε μια πρόσθετη λειτουργία.
  • Δεύτερο βήμα: Ολοκληρώστε όλες τις συνιστώμενες ενημερώσεις πριν επανεκκινήσετε τον περιηγητή σας.

Συχνές ερωτησεις

Τι είναι οι πρόσθετες λειτουργίες;

  • Οι πρόσθετες λειτουργίες χρησιμοποιούνται για βίντεο, κινούμενα σχέδια και παιχνίδια.
  • Κατασκευάζονται ξεχωριστά από τον Firefoxαπό εταιρείες όπως η >Adobe Systems και η >Apple.
  • Οι πρόσθετες λειτουργίες, δεν ενημερώνονται πάντα αυτόματα.

Γιατί θα πρέπει να ενημερώσω τις πρόσθετες λειτουργίες μου;

  • Οι παρωχημένες πρόσθετες λειτουργίες, μπορεί να επηρεάσουν αρνητικά την περιήγηση σας και να σας κάνουν να χάσετε το χρόνο σας.
  • Οι παρωχημένες πρόσθετες λειτουργίες, αυξάνουν τον κίνδυνο έκθεσης σε κακόβουλο λογισμικό, ιούς κι άλλες απειλές για την διαδικτυακή σας ασφάλεια.
  • Οι ενημερωμένες πρόσθετες λειτουργίες περιλαμβάνουν βελτιώσεις που καθιστούν την περιήγηση σας στον ιστό καλύτερη και ασφαλέστερη.

Πως μπορεί να με βοηθήσει ο Firefox;

Στο μέλλον, ο Firefox θα ενημερώνει αυτόματα τις πρόσθετες λειτουργίες σας. Μέχρι τότε, θα πρέπει να ελέγχετε τακτικά αυτή τη σελίδα και να ενημερώνετε τις πρόσθετες λειτουργίες σας για να είστε ασφαλείς.

Ποιες πρόσθετες λειτουργίες έχω;

Παραπάνω, ανιχνεύσαμε αυτόματα τις πρόσθετες λειτουργίες σας, για να δείτε τις πρόσθετες λειτουργίες που έχετε εγκαταστήσει στον Firefox ακολουθήστε αυτά τα βήματα:

  1. Κάντε κλικ στο κουμπί Firefox.
  2. Επιλέξτε τα Πρόσθετα.
  3. Επιλέξτε από τη λίστα τις Πρόσθετες λειτουργίες.

Πως μπορώ να απενεργοποιήσω μια πρόσθετη λειτουργία;

Στον Firefox:

  1. Κάντε κλικ στο κουμπί Firefox.
  2. Επιλέξτε τα Πρόσθετα.
  3. Επιλέξτε από τη λίστα τις Πρόσθετες λειτουργίες.
  4. Κάντε κλικ σε μια πρόσθετη λειτουργία από της λίστα που θα εμφανιστεί.
  5. Κάντε κλικ στο κουμπί Απενεργοποίηση.

Προσοχή: η απενεργοποίηση μιας πρόσθετης λειτουργίας, σημαίνει ότι δεν θα μπορείτε να κάνετε ορισμένα πράγματα. Για παράδειγμα, αν απενεργοποιήσετε το Flash, δεν θα μπορείτε να βλέπετε βίντεο σε δημοφιλείς ιστοσελίδες που τα παρέχουν.
