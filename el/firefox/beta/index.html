Εγκατάσταση της δοκιμαστικής έκδοσης

Η δοκιμαστική έκδοση του Firefox 4 περιλαμβάνει το πρόσθετο σχολιασμού που χρησιμοποιείται για μελέτες χρήσης. Κανένα δεδομένο δεν αποστέλλεται στους διακομιστές του Mozilla χωρίς την άδεια σας. Περισσότερες πληροφορίες

Διαδώστε τα νέα: όσο περισσότεροι χρήστες της δοκιμαστικής έκδοσης μας βοηθήσουν να φτιάξουμε τον Firefox 4, τόσο καλύτερο θα είναι το τελικό αποτέλεσμα. Μάθετε περισσότερα για τη δοκιμαστική έκδοση.

Η δοκιμαστική έκδοση του Firefox 4 περιλαμβάνει το πρόσθετο σχολιασμού που βασίζεται στο Mozilla Test Pilot

Όταν εγκαταστήσετε τη δοκιμαστική έκδοση του Firefox 4, θα παρατηρήσετε το πρόσθετο σχολιασμού στην πάνω δεξιά γωνία. Μπορείτε να το χρησιμοποιήσετε οποιαδήποτε στιγμή για να παρέχετε σχολιασμό άμεσα. Θα ειδοποιηθείτε ακόμα για μελέτες χρήσης (που διενεργεί το Mozilla Test Pilot) οι οποίες συλλέγουν πληροφορίες για το πως χρησιμοποιείτε τον Firefox, όχι όμως και για το ποιες σελίδες επισκέπτεστε. Μπορείτε να δείτε τα δεδομένα που συλλέγονται, να αποχωρίσετε από οποιαδήποτε μελέτη και γενικότερα να έχετε τον απόλυτο έλεγχο για το πως θα συμμετέχετε. Θα ειδοποιήστε πριν γίνει οποιαδήποτε αποστολή δεδομένων προς τους διακομιστές του Mozilla. Παρακαλούμε δείτε την πολιτική απορρήτου του πρόσθετου σχολιασμού για περισσότερες πληροφορίες.
