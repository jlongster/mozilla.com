Συμμετέχετε στο εγχείρημα Mozilla

Είμαστε μια παγκόσμια κοινότητα και συνεργαζόμαστε για να κάνουμε τον παγκόσμιο ιστό ένα καλύτερο μέρος για όλους.

Θέλετε να βοηθήσετε; Θαυμάσια! Δε χρειάζεται να είστε γκουρού της C++ (ή ακόμη και να γνωρίζετε τι σημαίνει αυτό!) και δε χρειάζεται να διαθέσετε πολύ χρόνο.

Εύκολοι Τρόποι για να Συμμετέχετε

  • Ενημερώστε την οικογένειά σας, τους φίλους και τους συνεργάτες σας για τα προϊόντα Mozilla
  • Βοηθήστε άλλους χρήστες στους δικτυακούς τόπους συζητήσεων και υποστήριξης
  • Συμμετέχετε στην ομάδα μάρκετινγκ της κοινότητας και βοηθήστε στην εκστρατεία διάδοσης
  • Διασφαλίστε ότι υποβάλετε πληροφορίες για τυχόν προβληματική λειτουργία στην ομάδα ανάπτυξης
  • Επικοινωνήστε με τις ιστοσελίδες που δε λειτουργούν σωστά με τα προϊόντα Mozilla
  • Κάνετε μια δωρεά στο Ίδρυμα Mozilla

Για όσους έχουν τεχνική κατάρτιση

  • Συμμετέχετε στην ομάδα ποιοτικού ελέγχου και αναφέρετε ελαττώματα και άλλα προβλήματα που χρίζουν διόρθωσης.
  • Βοηθήστε στη συγγραφή ή τη διόρθωση συγγραμμάτων τεκμηρίωσης για τους προγραμματιστές.
  • Βοηθήστε στη συγγραφή ή τη διόρθωση συγγραμμάτων τεκμηρίωσης για τους τελικούς χρήστες.
  • Συμβάλλετε στον πλούτο του οικοσυστήματος του Firefox δημιουργώντας ένα πρόσθετο.
  • Διορθώστε κάποιο ελάττωμα ή συμβάλλετε στον κώδικά του Firefox.
