Χαρακτηριστικά του Firefox

Πως συγκεντρώνουμε ό,τι καλύτερο για να κάνουμε την περιήγηση σας καλύτερη.

  • Απλοποιημένη περιήγηση
  • Υψηλές επιδόσεις
  • Αναβαθμισμένη ασφάλεια
  • Πανίσχυρη προσαρμογή
  • Κορυφαία τεχνολογία
  • Οικουμενική πρόσβαση

Κορυφαία χαρακτηριστικά

(περάστε από πάνω για λεπτομέρειες)

  • Απίστευτη ταχύτητα

    Απίστευτη ταχύτητα

    Δείτε ιστοσελίδες πολύ πιο γρήγορα, χρησιμοποιώντας λιγότερη μνήμη του υπολογιστή σας.

  • Συγχρονισμός

    Συγχρονισμός

    Πάρτε τον Firefox μαζί σας όπου κι αν πάτε. Συγχρονίστε την περιήγηση σας μεταξύ διάφορων συσκευών.

  • Εύκολη προσαρμογή

    Εύκολη προσαρμογή

    Χιλιάδες πρόσθετα σας προσφέρουν την ελευθερία να κάνετε τον Firefox πραγματικά όπως τον θέλετε.

  • Έξυπνη εργαλειοθήκη

    Έξυπνη εργαλειοθήκη

    Ξεχάστε τα δύσκολα URL – βρείτε τις σελίδες που σας αρέσουν σε δευτερόλεπτα

  • Λειτουργική διασύνδεση

    Λειτουργικός σχεδιασμός διασύνδεσης χρήστη

    Ο Firefox φτιάχνεται με επίκεντρο τον τελικό χρήστη, γι' αυτό είναι εύκολη και απλή η χρήση του από την πρώτη στιγμή που θα τον εγκαταστήσετε.

Απλοποιημένη περιήγηση

Έξυπνη εργαλειοθήκη διευθύνσεων

Μεταβείτε στις αγαπημένες σας σελίδες γρήγορα – ακόμα κι αν δεν θυμάστε το URL. Πληκτρολογήστε έναν όρο στην εργαλειοθήκη διευθύνσεων (γνωστή και σαν «έξυπνη εργαλειοθήκη») και η λειτουργία αυτόματης συμπλήρωσης θα σας υποδείξει διευθύνσεις που θα βρει στο ιστορικό, τους σελιδοδείκτες και σε ανοικτές καρτέλες.

Η έξυπνη εργαλειοθήκη «εκπαιδεύεται» όσο τη χρησιμοποιείτε, προσαρμόζεται στις προτιμήσεις σας και σας προσφέρει αποτελέσματα που ταιριάζουν ακριβώς. Την έχουμε βελτιώσει για να σας προσφέρουμε περισσότερο έλεγχο πάνω στα αποτελέσματα (συμπεριλαμβανομένων των ρυθμίσεων για το απόρρητο) και καλύτερες επιδόσεις για να βρίσκετε αυτό που ψάχνετε ακόμα πιο γρήγορα.

Βελτιωμένη διασύνδεση χρήστη

Αναζητούμε πάντα νέους τρόπους για να κάνουμε την περιήγηση σας πιο εύκολη και πιο αποδοτική.
Διαβάστε για τις πιο πρόσφατες βελτιώσεις:

Καρτέλες στο πάνω μέρος

Οι καρτέλες είναι πάνω από την έξυπνη εργαλειοθήκη για να μπορείτε ευκολότερα να εστιάσετε στο περιεχόμενο των σελίδων που επισκέπτεστε.

Κουμπί Firefox (σε Windows και Linux)

Όλα τα στοιχεία των μενού βρίσκονται τώρα κάτω από ένα και μοναδικό κουμπί για να έχετε εύκολη πρόσβαση σ' αυτά.

Κουμπί σελιδοδεικτών

Η διαχείριση των σελιδοδεικτών σας τώρα γίνεται με ένα μόνο κουμπί. Βρείτε τους αγαπημένους σας δεσμούς χωρίς να ψάχνετε πολύ!

Απλοποιημένο κουμπί ανανέωσης/διακοπής

Για την ανανέωση ή τη διακοπή φόρτωσης μια σελίδας, η έξυπνη εργαλειοθήκη περιλαμβάνει τώρα ένα εύκολο και απλό στη χρήση του κουμπί.

Κουμπί αρχικής σελίδας

Το κουμπί για την αρχική σελίδα έχει μεταφερθεί στα δεξιά της εργαλειοθήκης αναζήτησης.

Καρτέλες

Περιηγηθείτε σε πολλαπλές σελίδες εύκολα και απλά. Κάθε νέα σελίδα εμφανίζεται σε νέα καρτέλα
(όχι σε νέο παράθυρο) και έχετε πρόσβαση σ' αυτή με ένα μόνο κλικ.

Καρτέλες εφαρμογών

Βγάλτε τις καρτέλες που έχετε συνεχώς ανοιχτές (όπως τη διαδικτυακή υπηρεσία ηλεκτρονικού ταχυδρομείου που χρησιμοποιείτε) από την εργαλειοθήκη καρτελών και δώστε τους μια μόνιμη θέση στον περιηγητή σας.

Επαναφορά κλεισμένων καρτελών και παραθύρων

Αν κλείσετε κατά λάθος μια καρτέλα ή ένα παράθυρο, μπορείτε να επαναφέρετε με ένα μόνο κλικ. Απλά δείτε τις πρόσφατα κλεισμένες καρτέλες ή παράθυρα στο μενού του ιστορικού και επιλέξτε την καρτέλα ή το παράθυρο που θέλετε να επαναφέρετε.

Μετάβαση σε καρτέλα

Όταν ανοίγετε μια νέα καρτέλα ή πληκτρολογείτε στην έξυπνη εργαλειοθήκη της διεύθυνση μιας σελίδας, ο Firefox ελέγξει μήπως έχετε ήδη φορτώσει αυτή τη σελίδα. Αν υπάρχει ήδη ανοιχτή η σελίδα, θα σας προτείνει να μεταβείτε σ' αυτή, αποφεύγοντας έτσι το άνοιγμα κι άλλης.

Εύκολη κύλιση

Θέλετε να ανοίγετε και τις 20 αγαπημένες σας σελίδες ταυτόχρονα; Ένα κομψό νέο χαρακτηριστικό σας επιτρέπει την εύκολη κύλιση ανάμεσα στις καρτέλες για να τις βλέπετε όλες και να έχετε εύκολη και γρήγορη πρόσβαση σε κάθε μια απ' αυτές.

Ομαδοποίηση καρτελών με το Panorama

Έχετε πολλές καρτέλες ανοιχτές; Απελευθερώστε τον περιηγητή σας από το χαμό με τις καρτέλες! Η ομαδοποίηση καρτελών (Panorama) σας επιτρέπει να οργανώσετε απλά και εύκολα τις καρτέλες σας με σύρσιμο και απόθεση, σε ομάδες που διαχειρίζονται εύκολα, αφού μπορείτε να τους δώσετε ονόματα, να τις ταξινομήσετε και να τις στοιχίσετε με ένα απλό και διασκεδαστικό τρόπο. Για να ξεκινήσετε με την ομαδοποίηση καρτελών (Panorama), απλά επιλέξτε το κουμπί ομαδοποίησης καρτελών της εργαλειοθήκης ή κάντε δεξί κλικ σε οποιαδήποτε καρτέλα και επιλέξτε «Μετακίνηση στην ομάδα».

Συγχρονισμός

Το Sync συνδέει τον Firefox που έχετε σε κάθε υπολογιστή ή στο κινητό σας, ώστε να μπορείτε να έχετε πρόσβαση στο ιστορικό περιήγησης, τους κωδικούς, τους σελιδοδείκτες, ακόμα και τις ανοιχτές καρτέλες που έχετε σε μια δεδομένη χρονική στιγμή, ανεξάρτητα από το ποια συσκευή χρησιμοποιείτε. Αποκτήστε πρόσβαση στο ιστορικό περιήγησης που έχετε εδώ και χρόνια στον υπολογιστή σας, από την πρώτη στιγμή που θα χρησιμοποιήσετε τον Firefox στο κινητό σας και χρησιμοποιήστε αποθηκευμένους κωδικούς από τον υπολογιστή σας για να συμπληρώσετε φόρμες στο κινητό σας.

Τώρα πια, μπορείτε να σερφάρετε στον υπολογιστή σας, να φύγετε και οι καρτέλες που είχατε να σας περιμένουν ανοιχτές και έτοιμες όπως ακριβώς τις είχατε αφήσει, στο κινητό σας. Η περιήγηση στο διαδίκτυο δεν θα είναι ποτέ ξανά όπως παλιά! Μάθετε περισσότερα για το Sync ή βάλτε τον Firefox στο κινητό σας.

Διαχειριστής κωδικών

Βαρεθήκατε να προσπαθείτε να θυμηθείτε κωδικούς ή να τους πληκτρολογείτε ξανά και ξανά κάθε φορά που επισκέπτεστε κάποια αγαπημένη σας σελίδα; Ο Firefox είναι εδώ για να σας βοηθήσει! Μετά την πρώτη είσοδο σας σε μια σελίδα, θα σας ρωτήσει διακριτικά αν θέλετε να απομνημονευτεί ο κωδικός, από 'κει και πέρα τα υπόλοιπα θα τα αναλάβει ο Firefox.

Συμπλήρωση φορμών

Όταν συμπληρώνετε στοιχεία σε φόρμες στο διαδίκτυο, όπως τη διεύθυνση σας κλπ, ο Firefox θα σας προτείνει δεδομένα για κάθε πεδίο με βάση τα δεδομένα που έχετε συμπληρώσει σε παρόμοια πεδία άλλων φορμών. Κουραστήκατε να περνάτε τα ίδια στοιχεία κάθε φορά; τώρα θα ξεμπερδεύετε με τις φόρμες σε χρόνο μηδέν!

Φραγή αναδυόμενων

Εξαφανίστε τα αναδυόμενα παράθυρα από τις περιηγήσεις σας στο διαδίκτυο μια για πάντα. Εναλλακτικά μπορείτε να ορίσετε τη χρυσή τομή επιλέγοντας ποια αναδυόμενα θα επιτρέπονται ή να δημιουργήστε μια λίστα ιστοσελίδων για τις οποίες να επιτρέπεται το άνοιγμα αναδυόμενων.

Εύκολη αναζήτηση

Το διαδίκτυο είναι τεράστιο σε όγκο πληροφοριών, αλλά ο Firefox είναι έτοιμος να σας βοηθήσει να βρείτε εύκολα οτιδήποτε ψάχνετε.

Ενσωματωμένη αναζήτηση στο διαδίκτυο

Η αναζήτηση στο διαδίκτυο είναι παιχνιδάκι με την ενσωματωμένη εργαλειοθήκη αναζήτησης που βρίσκεται δεξιά από την εργαλειοθήκη διευθύνσεων. Επιλέξτε τη μηχανή αναζήτησης που προτιμάτε και πληκτρολογήστε τους όρους που αναζητάτε στο πλαίσιο. Αν χρειάζεστε περισσότερο χώρο, το πλαίσιο αυτό έχει ακόμα και τη δυνατότητα να επεκτείνεται για να σας διευκολύνει.

Προτάσεις αναζήτησης

Ξεκινήστε να πληκτρολογείτε στην εργαλειοθήκη αναζήτησης και θα εμφανιστεί μια αναδυόμενη λίστα με προτάσεις για αναζήτηση. Επίσης, μπορείτε να χρησιμοποιήσετε την εργαλειοθήκη αναζήτησης σαν αριθμομηχανή, μετατροπέα κι ακόμη περισσότερα.

Εκατοντάδες μηχανές αναζήτησης

Επιλέξτε μια μηχανή αναζήτησης από την προκαθορισμένη αναδυόμενη λίστα ή κάντε κλικ στην επιλογή «Διαχείριση μηχανών αναζήτησης» για να βρείτε πολλές περισσότερες μηχανές. Δείτε τις διαθέσιμες επιλογές.

Έξυπνες λέξεις-κλειδιά

κάνε αναζητήσεις στο διαδίκτυο σε χρόνο ρεκόρ με τις έξυπνες λέξεις-κλειδιά. Με μερικά μόνο κλικ μπορείτε να συνδέσετε λέξεις-κλειδιά με μηχανές αναζήτησης και μετά απλώς να πληκτρολογείτε τη λέξη-κλειδί και τους όρους που αναζητάτε στην έξυπνη εργαλειοθήκη διευθύνσεων. Αν για παράδειγμα συνδέσετε πχ τη λέξη-κλειδί «βιβλία» με το Amazon.com, αυτό σημαίνει πως θα μπορείτε να πληκτρολογήσετε «βιβλία για κατασκευές» και να μεταφερθείτε στη σελίδα των αποτελεσμάτων αναζήτησης του Amazon.com χωρίς να χρειάζεται να περάσετε από την αρχική του σελίδα.

Περιήγηση με βάση την τοποθεσία

Χρειάζεστε οδηγίες για τη διαδρομή ή βοήθεια για να βρείτε ένα εστιατόριο; Ο Firefox μπορεί να γνωστοποιήσει σε ιστοσελίδες την τοποθεσία σας ώστε να σας παρέχουν τις πιο ακριβής και χρήσιμες πληροφορίες. Φυσικά ο Firefox δεν πρόκειται να γνωστοποιήσει την τοποθεσία σας χωρίς την άδεια σας – η προστασία του ιδιωτικού σας απόρρητου αποτελεί προτεραιότητα μας κι είναι απόλυτα σεβαστή.

Επαναφορά συνεδρίας

Η επαναφορά συνεδρίας, επαναφέρει άμεσα τα παράθυρα και τις καρτέλες σας, μαζί με οποιοδήποτε κείμενο γράφατε και τυχόν λήψεις που ήταν σε εξέλιξη. Μπορείτε να επανεκκινήσετε τον περιηγητή σας χωρίς να χάσετε τη συνέχεια όταν θέλετε να εγκαταστήσετε ένα πρόσθετο ή μια αναβάθμιση. Και φυσικά, αν ο Firefox ή ο υπολογιστής σας τερματιστεί απρόσμενα, δεν χρειάζεται να διαθέσετε χρόνο για να ανακτήσετε δεδομένα ή να βρείτε τις σελίδες που είχατε ανοίξει πριν. Αν για παράδειγμα γράφατε ένα μήνυμα ηλεκτρονικού ταχυδρομείου, θα συνεχίσετε ακριβώς από το σημείο που είχατε σταματήσει, από την τελευταία λέξη που είχατε πληκτρολογήσει.

Έλεγχος ορθογραφίας

Ο ενσωματωμένος έλεγχος ορθογραφίας δουλεύει άμεσα σε ιστοσελίδες που φιλοξενούν ιστολόγια και υπηρεσίες ηλεκτρονικού ταχυδρομείου. Πάψτε πια να ανησυχείτε για τυπογραφικά λάθη γλιτώστε από ένα παραπάνω κόπο να ελέγχετε τι γράφετε συνεχώς.

Οργάνωση

Στο διαδίκτυο θα βρείτε πάρα πολλές θαυμάσιες τοποθεσίες και πληροφορίες κι ο Firefox σας προσφέρει μια πληθώρα τρόπων για να τις οργανώσετε.

Σελιδοδείκτες με ένα κλικ

Διαχειριστείτε κατά βούληση τους σελιδοδείκτες σας. Η προσθήκη σελιδοδείκτη για μια σελίδα γίνεται με ένα απλό κλικ στο αστέρι που βρίσκεται στην άκρη της εργαλειοθήκης διευθύνσεων. Με δύο κλικ μπορείτε να επιλέξετε που θα αποθηκευθεί ο σελιδοδείκτης και αν θα του προσθέσετε ετικέτα. Βάλτε τις σελίδες που έχετε στους σελιδοδείκτες σας σε φακέλους για εύκολη πρόσβαση και οργανώστε τις ανά θέμα (όπως πχ «Δουλειά» ή «Ψώνια»)Εντοπίστε τις σελίδες που έχετε στους σελιδοδείκτες στη στιγμή πληκτρολογώντας στην εργαλειοθήκη διευθύνσεων την ετικέτα, το όνομα της σελίδας ή του σελιδοδείκτη. Όσο περισσότερο χρησιμοποιείτε τις ετικέτες και δίνετε ονόματα στους σελιδοδείκτες σας, τόσο πιο γρήγορα θα προσαρμοστεί το σύστημα στις προτιμήσεις σας.

Ετικέτες

Δώστε σε μια ιστοσελίδα τίτλο και εντάξτε την σε μια κατηγορία που έχει νόημα για σας. Για παράδειγμα, μπορείτε να βάλετε τις ετικέτες «ειδήσεις» και επενδύσεις στο www.tvxs.gr/ και μετά να βάλετε την ετικέτα «ειδήσεις» και στο www.nytimes.com. Όταν θα πληκτρολογείτε «ειδήσεις» στην εργαλειοθήκη διευθύνσεων, θα εμφανίζονται και οι δύο ιστότοποι σαν αποτελέσματα. Κάθε ιστότοπος μπορεί να έχει πολλαπλές ετικέτες χωρίς περιορισμούς στον αριθμό τους. Μπορεί να μην θυμάστε το όνομα του ιστότοπου αλλά με τις ετικέτες θα μπορείτε να τον βρείτε με ένα τρόπο που θα σας βολεύει.

Βιβλιοθήκη

Κάντε τον περιηγητή σας όσο αποδοτικό θέλετε. Το ιστορικό περιήγησης σας (όλες οι ιστοσελίδες που έχετε δει) και οι σελιδοδείκτες σας (όλες οι σελίδες που έχετε προσθέσει στους σελιδοδείκτες) αρχειοθετούνται στη βιβλιοθήκη, όπου μπορείτε εύκολα να τις εντοπίσετε και να τις οργανώσετε. Μπορείτε επίσης να αποθηκεύσετε τις πιο συχνές αναζητήσεις σας σε δυναμικούς «έξυπνους» φακέλους που ενημερώνονται αυτόματα καθώς η λίστα των σελιδοδεικτών και του ιστορικού σας θα μεγαλώνει.

Εύρεση σε σελίδες

Η λειτουργία εύρεσης σε σελίδες ενεργοποιείται με το απλό πάτημα ενός συνδυασμού πλήκτρων και σας επιτρέπει την αναζήτηση μιας λέξης ή φράσης μέσα σε μια ιστοσελίδα που έχετε ανοιχτή. Αν επιλέξετε μια λέξη ή φράση σε μια σελίδα και ενεργοποιήσετε μετά την εύρεση, θα ανοίξει με το κείμενο που έχετε επιλέξει έτοιμο για αναζήτηση. Μπορείτε να δείτε άμεσα όλα τα σημεία που εμφανίζεται στη σελίδα ή να πάτε μπρος-πίσω στη σελίδα για να τα δείτε όλα.

Ανάγνωση αρχείων ροής RSS

Περνάτε ώρες στο διαδίκτυο για να βρείτε τις τελευταίες ειδήσεις; Αντί να το κάνετε έτσι, προσθέστε ένα εικονίδιο RSS στην εργαλειοθήκη και παρακολουθήστε αρχεία ροής από διάφορες υπηρεσίες, προσθέστε ένα πρόσθετο για ανάγνωση αρχείων ροής ή απλά προσθέστε ένα δυναμικό σελιδοδείκτη στον Firefox.

Διαχειριστής λήψεων

Κάντε αδιάλειπτη και ασφαλή λήψη αρχείων! Υπάρχει δυνατότητα παύσης και συνέχισης μια λήψης, πράγμα που σημαίνει ότι μπορείτε να σταματήσετε προσωρινά και να συνεχίσετε μια λήψη όποτε εσείς θέλετε. Οπότε, αν έχετε κάνει ήδη λήψη μέρους ενός μεγάλου αρχείου αλλά πρέπει να φύγετε, απλά κάντε παύση της λήψης και συνεχίστε από το σημείο που είχατε μείνει όταν επιστρέψετε. Η δυνατότητα συνέχισης ισχύει και για την περίπτωση που το σύστημα σας καταρρεύσει ή επανεκκινηθεί αναγκαστικά. Στο διάλογο διαχείρισης λήψεων έχετε επίσης τη δυνατότητα να δείτε την πρόοδο των λήψεων ή ακόμα και να κάνετε αναζήτηση για λήψεις με βάση το όνομα του αρχείου ή τη διεύθυνση της σελίδας από τον οποία κάνατε λήψη.

Περιήγηση εκτός σύνδεσης

Αποσυνδέστε τον υπολογιστή σας από το δίκτυο αλλά κρατήστε τις πληροφορίες που θέλετε. Συγκεκριμένες σελίδες επιτρέπουν σε δικτυακές εφαρμογές - όπως συνήθως οι δικτυακές υπηρεσίες ηλεκτρονικού ταχυδρομείου - να μένουν ενεργές συνεχώς.

Πλήρης μεγέθυνση

Θέλετε να δείτε λεζάντες και εικόνες σε μεγαλύτερο μέγεθος; Το κομψό χαρακτηριστικό μεγέθυνσης που προσφέρει ο Firefox σας επιτρέπει να μεγεθύνετε όσο θέλετε για να δείτε και την πιο μικρή λεπτομέρεια μιας σελίδας, εστιάζοντας έτσι σ' αυτό που πραγματικά σας ενδιαφέρει. Η κλίμακα των σελίδων αλλάζει ομοιόμορφα, επιτρέποντας σε όλα της τα στοιχεία να διατηρήσουν τις σωστές αναλογίες.

Ενσωματωμένη εμφάνιση & αίσθηση

Η πλατφόρμα του Firefox platform έχει την εμφάνιση και την αίσθηση που έχει και το σύστημα σας. Δείτε τον Firefox σαν μια πολύ κοινωνική προσωπικότητα που κάνει πολύ εύκολα φίλους. Είτε χρησιμοποιείτε Windows&7, είτε Linux, είτε Mac, ο περιηγητής σας ενσωματώνεται πλήρως αισθητικά με το περιβάλλον του υπολογιστή σας. Κι είναι σημαντικό να μην σας ξενίζει, ώστε να μην σας αποσπά όταν τον χρησιμοποιείτε.

Υποστήριξη για τον Firefox

Χρειάζεστε βοήθεια; Οι σελίδες υποστήριξης διαθέτουν μια τεράστια βάση δεδομένων με άρθρα, φόρουμ και οδηγίες για να βρείτε λύσεις σε προβλήματα και συμβουλές για να κάνετε καλύτερη την περιήγηση σας.

Υψηλές επιδόσεις

Απίστευτη ταχύτητα

Θα διαπιστώσετε άμεσα τις σημαντικές βελτιώσεις στις επιδόσεις του Firefox, αφού περιλαμβάνει από συντομότερο χρόνο εκκίνησης, σχεδόν άμεση απεικόνιση γραφικών, βελτιωμένο χρόνο φόρτωσης σελίδας και πολλά άλλα. Από την ενσωμάτωση τμημάτων και την πιο αποδοτική διαχείριση αντικειμένων JavaScript, ως την υποστήριξη ECMAScript5 και τη δημιουργία της ολοκαίνουριας μηχανής JavaScript JägerMonkey, δουλέψαμε πολύ στη μηχανή του Firefox για να κάνουμε αυτή την έκδοση την ισχυρότερη που έχει βγει.

Προστασία από καταρρεύσεις

Ο Firefox παρέχει απρόσκοπτη περιήγηση ακόμα και όταν καταρρεύσει κάποια από τις πρόσθετες λειτουργίες Adobe Flash, Apple QuickTime ή Microsoft Silverlight. Αν κάποια απ' αυτές τις συχνά χρησιμοποιούμενες πρόσθετες λειτουργίες καταρρεύσει ή σταματήσει να αποκρίνεται, δεν θα επηρεάσει τις υπόλοιπες καρτέλες του Firefox. Θα μπορείτε απλά να φορτώσετε ξανά τη σελίδα για να επανεκκινηθεί η πρόσθετη λειτουργία και να προσπαθήσετε ξανά. Μάθετε περισσότερα για τις πρόσθετες λειτουργίες.

Μάθετε περισσότερα για τις επιδόσεις του Firefox.

Υψηλές επιδόσεις στην πραγματικότητα

Το να έχεις μια απίστευτα γρήγορη μηχανή JavaScript είναι σίγουρα σημαντικό, αλλά επειδή οι διαφορές στις επιδόσεις συνήθως μετρούνται σε millisecond, μερικές φορές αυτό που έχει σημασία είναι το πως συμπεριφέρεται ο περιηγητής σε καθημερινή χρήση κι όχι σε εξειδικευμένα μετροπρογράμματα. Βελτιώνουμε συνεχώς τον Firefox για να συμπεριφέρεται άψογα όταν τον χρησιμοποιείτε στο διαδίκτυο – βελτιώσεις όπως πχ το γρηγορότερο άνοιγμα μιας καρτέλας ή η ταχύτερη παροχή αποτελεσμάτων από την έξυπνη εργαλειοθήκη. Γιατί τελικά, αυτό που έχει σημασία, είναι να δείτε αυτό που θέλετε όσο πιο γρήγορα κι εύκολα είναι δυνατόν.

3D στο διαδίκτυο

Το WebGL φέρνει τα γραφικά 3D στον Firefox, ανοίγοντας έτσι το δρόμο στους προγραμματιστές για να δημιουργήσουν ακόμα πιο «ζωντανά» παιχνίδια και νέους τρόπους απεικόνισης και σχεδίασης στο διαδίκτυο.

Επιτάχυνση μέσω υλικού

Δείτε απίστευτα γρήγορη επιτάχυνση γραφικών σε βίντεο και περιεχόμενο του ιστού με το νέο σύστημα γραφικών που βασίζεται σε στρώματα και εκμεταλλεύεται τις δυνατότητες Direct2D και Direct3D που έχουν οι κάρτες γραφικών σε Windows και του OpenGL σε Mac.

Αναβαθμισμένη ασφάλεια

Άμεσες πληροφορίες ταυτότητας ιστότοπου

Θέλετε να είστε απόλυτα σίγουροι για τη νομιμότητα ενός ιστότοπου πριν προβείτε σε μια συναλλαγή; Κάντε κλικ στο favicon του ιστότοπου για μια άμεση επισκόπηση της ταυτότητάς του. Με ένα δεύτερο κλικ πάτε ακόμα πιο μακριά: πόσες φορές έχετε επισκεφθεί τη σελίδα; έχετε αποθηκεύσει κωδικούς για τη σελίδα; Κάντε έλεγχο σε ύποπτους ιστότοπους, αποφύγετε τις δικτυακές απάτες και βεβαιωθείτε πως ο κάθε ιστότοπος είναι αυτός που ισχυρίζεται πως είναι.

Πολιτική ασφαλείας περιεχομένου

Η πολιτική ασφαλείας περιεχομένου του Firefox έχει σχεδιαστεί για να μην επιτρέπει επιθέσεις cross-site, προσφέροντας στους ιστότοπους ένα μηχανισμό για να μπορούν να καθορίσουν αυστηρά ποιο περιεχόμενο είναι το γνήσιο. Έτσι, ο περιηγητής θα μπορεί να απορρίψει το περιεχόμενο που δεν αναγνωρίζει ο ιστότοπος, διατηρώντας το χρήστη ασφαλή.

Προσαρμοσμένες ρυθμίσεις ασφαλείας

Μπορείτε να ελέγξετε το βαθμό προφύλαξης του Firefox από ένα ιστότοπο. Μπορείτε ακόμα και να προσθέσετε εξαιρέσεις ασφαλείας για ιστότοπους που πιστεύετε πως δεν χρειάζονται ανάκριση για να πιστοποιηθούν. Μπορείτε επίσης να έχετε προσαρμοσμένες ρυθμίσεις για κωδικούς, cookies, φόρτωση εικόνων και εγκατάσταση πρόσθετων, έτσι ώστε να έχετε τον απόλυτο έλεγχο της περιήγησης.

Γονικός έλεγχος

Ενισχύστε τις ρυθμίσεις γονικού ελέγχου που έχετε καθορίσει στα Windows Vista — απαγορεύστε ανεπιθύμητες λήψεις και ακόμα περισσότερα.

Ασφαλείς ενημερώσεις

Ο Firefox αναζητεί ασφαλείς συνδέσεις πριν εγκαταστήσει ή ενημερώσει πρόσθετα, λογισμικό τρίτων και Personas.

Ανίχνευση παρωχημένων πρόσθετων λειτουργιών

Οι πρόσθετες λειτουργίες είναι μικρά προγράμματα τρίτων κατασκευαστών που φτιάχνονται από εταιρείες όπως η Adobe Systems ή η Apple για να υποστηρίξουν βίντεο, κινούμενα σχέδια και παιχνίδια. Όταν οι εκδόσεις που έχετε είναι παρωχημένες, αυτό μπορεί να σημαίνει ότι υπάρχουν κρίσιμα κενά ασφαλείας στο σύστημα σας. Γι' αυτό φτιάξαμε ένα εύκολο στη χρήση του εργαλείο για να είστε πάντα ενημερωμένοι. Ελέγξτε τις πρόσθετες λειτουργίες σας.

Ιδιωτική περιήγηση

Μερικές φορές είναι ωραία να περνάς απαρατήρητος. Όταν ενεργοποιηθεί αυτό το χαρακτηριστικό, δεν μένει κανένα αποτύπωμα της περιήγησης σας για να το ανακαλύψει κάποιος. Μπορείτε να βγείτε και να μπείτε σε κατάσταση ιδιωτικής περιήγησης γρήγορα, οπότε είναι εύκολο να επιστρέψετε σ' αυτό που κάνατε πριν σαν να μη συνέβηκε τίποτα (ή να χρησιμοποιείτε πάντα την ιδιωτική περιήγηση). Είναι ακριβώς αυτό που χρειάζεστε αν για παράδειγμα διαχειρίζεστε τους τραπεζικούς σας λογαριασμούς σε ένα υπολογιστή που χρησιμοποιούν κι άλλοι ή θέλετε να δείτε το ηλεκτρονικό σας ταχυδρομείο σε ένα Internet café.

Αλληλεπίδραση με λογισμικό προστασίας από ιούς

Ο Firefox συνεργάζεται κομψά με το λογισμικό προστασίας από ιούς σε περιβάλλον Windows. Όταν κάνετε λήψη ενός αρχείου, το λογισμικό προστασίας από ιούς του συστήματος σας το ελέγχει αυτόματα για να σας προστατέψει από ιούς ή άλλο κακόβουλο λογισμικό που θα μπορούσε να προκαλέσει προβλήματα στο σύστημα σας.

Προστασία από πλαστογραφίες (Anti-Phishing)

Κάντε τις συναλλαγές σας στο διαδίκτυο με ασφάλεια. Ο Firefox λαμβάνει ενημερώσεις για ιστότοπους ύποπτους για πλαστογραφία 48 φορές τη μέρα, οπότε αν προσπαθήσετε να επισκεφθείτε ένα κίβδηλο ιστότοπο που ισχυρίζεται πως είναι ένας ιστότοπος που εμπιστεύεστε (όπως η τράπεζά σας για παράδειγμα), θα σας σταματήσει ένα - υπερβολικά μεγάλο - μήνυμα του περιηγητή σας.

Εκκαθάριση πρόσφατου ιστορικού

Εκκαθαρίστε όλα τα προσωπικά σας δεδομένα ή μόνο ό,τι αφορά τις δραστηριότητες σας των τελευταίων ωρών με μερικά απλά κλικ. Έχετε τον απόλυτο έλεγχο για να διαγράψετε οτιδήποτε και έτσι όλες οι πληροφορίες θα εξαφανιστούν — είτε από το δικό σας υπολογιστή, είτε από κάποιον κοινόχρηστο. Είναι τόσο απλό!

Προστασία από κακόβουλες επιθέσεις

Ο Firefox 3 σας προστατεύει από ιούς, worms, δούρειους ίππους και λογισμικό υποκλοπής. Αν κατά λάθος επισκεφθείτε έναν ιστότοπο που έχει φτιαχτεί για τέτοιου είδους επιθέσεις, θα δείτε ένα προειδοποιητικό αποτρεπτικό μήνυμα που θα σας ενημερώνει για τους λόγους που δεν θεωρείται ασφαλής για χρήση.

Κεφαλίδα «Do Not Track»

Πολλές σελίδες παρακολουθούν τις συνήθειες σας στο διαδίκτυο και πωλούν τα δεδομένα που αποθηκεύουν σε διαφημιστές. Αν αυτή η συμπεριφορά δεν σας αρέσει, ο Firefox σας επιτρέπει να δηλώσετε στις σελίδες που επισκέπτεστε ότι δεν συμφωνείτε με την παρακολούθηση της δραστηριότητας σας και να διατηρήσει έτσι τις προτιμήσεις σας στο διαδίκτυο ιδιωτικές.

Απαγόρευση απομνημόνευσης σελίδας

Το μετανιώσατε που επισκεφθήκατε κάποια σελίδα; Μ' αυτό το χαρακτηριστικό μπορείτε να αφαιρέσετε κάθε ίχνος της από τον περιηγητή σας — χωρίς ερωτήσεις!

Διασφάλιση συνδέσεων με ιστότοπους

Ο Firefox δεν επιτρέπει σε τρίτους την υποκλοπή ευαίσθητων δεδομένων με το να πραγματοποιεί αυτόματα ασφαλείς συνδέσεις σε ιστότοπους που υποστηρίζουν ασφαλή σύνδεση https στους διακομιστές τους.

Αυτόματες ενημερώσεις

Ο Firefox θα σας ειδοποιήσει αυτόματα όταν υπάρχει διαθέσιμη μια νέα ή ενημερωμένη έκδοση, ώστε να είστε σίγουροι ότι ο περιηγητής σας θα είναι πάντα ενημερωμένος με τις πιο σύγχρονες αναβαθμίσεις για την ασφάλεια σας.

Πανίσχυρη προσαρμογή

Διαχείριση πρόσθετων

Ο διαχειριστής πρόσθετων σχεδιάστηκε απ' την αρχή για να σας επιτρέπει να ανακαλύψετε και να εγκαταστήσετε πρόσθετα χωρίς να χρειάζεται να βγείτε από τον Firefox. Δείτε βαθμολογίες, προτάσεις, περιγραφές και εικόνες των πρόσθετων σε λειτουργία που θα σας βοηθήσουν να επιλέξετε. Ο διαχειριστής πρόσθετων σας επιτρέπει ακόμα και να δείτε, να διαχειριστείτε ή και να απενεργοποιήσετε πρόσθετες λειτουργίες τρίτων κατασκευαστών, με μερικά μόνο κλικ, ενώ ελέγχει και ενημερώνει αυτόματα τα ήδη εγκατεστημένα πρόσθετα κάθε φορά που τον ανοίγετε.

«Ντύστε» τον Firefox με τα Personas

Αλλάξτε την εμφάνιση του Firefox σε πρακτικά οτιδήποτε μπορείτε να φανταστείτε με τα Personas: εκατοντάδες χιλιάδες ελαφριά κι έτοιμα για εγκατάσταση θέματα εμφάνισης, που τα φτιάχνουν χρήστες απ' όλο τον κόσμο (ή που τα φτιάχνετε κι εσείς οι ίδιοι!). Με ένα απλό κλικ, μπορείτε να «ντύσετε» τον περιηγητή σας με απλά ή πολύχρωμα σχέδια και παραστάσεις από δημοφιλή πρόσωπα και παραγωγές όπως πχ τον Bob Marley ή τον Harry Potter.

Προσαρμόσιμη διασύνδεση

Η περιήγηση του καθενός είναι προσωπική υπόθεση, γι' αυτό κάναμε εύκολη την προσαρμογή της διασύνδεσης του Firefox για να μπορείτε να κάνετε την εμφάνιση του όπως θέλετε: αναπροσαρμόστε, οργανώστε, προσθέστε ή αφαιρέστε κουμπιά ή εργαλειοθήκες, για να τον προσαρμόσετε όπως ακριβώς τον προτιμάτε. Για να ξεκινήσετε, απλά κάντε δεξί κλικ στην εργαλειοθήκη πλοήγησης και επιλέξτε «Προσαρμογή...» από το μενού.

Πέρα από τα πρόσθετα

Τα πρόσθετα είναι ο ακρογωνιαίος λίθος για την προσαρμογή του Firefox αλλά η προσαρμογή του στο δικό σας στυλ δεν σταματάει σ' αυτά. Μπορείτε να προσθέσετε νέες μηχανές αναζήτησης, να αλλάξετε τις προτιμήσεις των εργαλειοθηκών, να εμφανίζετε με διαφορετικά μεγέθη, σχήματα και στυλ τα κουμπιά περιήγησης κι άλλα πολλά. Με περαιτέρω προτιμήσεις, μπορείτε να ορίσετε πχ ποια υπηρεσία ηλεκτρονικού ταχυδρομείου (όπως το Gmail) θα χρησιμοποιηθεί όταν κάνετε κλικ σε μια διεύθυνση που βρίσκετε σε μια ιστοσελίδα ή να ορίσετε μια εφαρμογή ανάγνωσης ειδήσεων για τα ιστολόγια που συναντάτε.

100.000 τρόποι για προσαρμογή

Στην πραγματικότητα, δεν υπάρχει αριθμητικό όριο με την ολοένα αυξανόμενη βιβλιοθήκη μας που περιέχει χιλιάδες πρόσθετα για τον Firefox, μικρά προγράμματα που μπορείτε να εγκαταστήσετε στον μοναδικό περιηγητή που δουλεύει όπως εσείς θέλετε. Διαχειριστείτε διαδικτυακές δημοπρασίες, ανεβάστε στο διαδίκτυο τις ψηφιακές σας φωτογραφίες, δείτε το μετεωρολογικό δελτίο πρόγνωσης με μια ματιά και ακούστε μουσική εύκολα μέσα από τον ίδιο σας τον περιηγητή. Αν δεν είστε σίγουροι με ποιο πρόσθετο να ξεκινήσετε, δείτε τη σελίδα Rock Your Firefox για να δείτε ενδιαφέρουσες προτάσεις.

Κορυφαία τεχνολογία

Η πιο πρόσφατες τεχνολογικές εξελίξεις για το διαδίκτυο

Οι προγραμματιστές και όλοι όσοι φτιάχνουν ιστοσελίδες, θα λατρέψουν τις νέες και βελτιωμένες δυνατότητες του Firefox. Αν σας αρέσει να φτιάχνετε προχωρημένες εφαρμογές και περιεχόμενο στο διαδίκτυο, ο Firefox αποτελεί την ιδανική επιλογή για εσάς. Δείτε τη λίστα με τις δυνατότητες chart full of checkmarks για να μάθετε περισσότερα.

WebSockets

Τα WebSockets ανοίγουν το δρόμο στην άμεση αμφίδρομη επικοινωνία μεταξύ του Firefox και των διακομιστών. Έτσι, διευκολύνει τους δημιουργούς στην κατασκευή εφαρμογών επικοινωνίας μέσω μηνυμάτων σε πραγματικό χρόνο και δικτυακών παιχνιδιών γραμμένων σε HTML5 που θα έχουν καλύτερες αποκρίσεις και περισσότερα διαδραστικά στοιχεία από ποτέ.

Αναβαθμισμένες φόρμες

Κάντε την ενοχλητική και δύσκολη ανάπτυξη φορμών παρελθόν. Τα χαρακτηριστικά για φόρμες που είναι ενσωματωμένα στον Firefox, όπως τα list attributes και HTML validation σας παρέχουν πια τα εργαλεία που χρειάζεστε.

Βιντεο WebM και HD

Σαν πρωτοπόροι που είμαστε στην υποστήριξη των πρότυπων βίντεο της HTML5, ο Firefox υποστηρίζει και τη μορφή WebM για βίντεο, ώστε να μπορείτε να βλέπετε βίντεο ποιότητας HD κωδικοποιημένο με ανοικτά πρότυπα.

Parser

Ο Firefox περιλαμβάνει ένα parser έτοιμο να υποστηρίξει την HTML5. Ο parser περιλαμβάνει μερικές νέες δυνατότητες, όπως για παράδειγμα την υποστήριξη ενσωματωμένων SVG, αλλά επίσης βελτιώνει τις επιδόσεις αφού εκτελεί τον αλγόριθμο parsing σε ξεχωριστή διεργασία.

IndexedDB

Ο Firefox περιλαμβάνει μια πρώιμη έκδοση του IndexedDB για να δώσει στους προγραμματιστές μια πρώτη γεύση και την ευκαιρία να πειραματιστούν με local storage στον περιηγητή.

Γραφικά νέας γενιάς

Θα πρέπει να το δείτε για να το πιστέψετε. Χάρη στην αναβαθμισμένη μας μηχανή γραφικών, η απεικόνιση κειμένου και γραφικών, συμπεριλαμβανομένων των SVG και Canvas, είναι πιο γρήγορη και ευκρινής από ποτέ. Ο Firefox διορθώνει αυτόματα το χρώμα στις εικόνες (όπως κάνει και η φωτογραφική σας μηχανή) για να εμφανίζονται πάντα τα χρώματα τους όπως ακριβώς περιγράφονται στα δεδομένα τους.

Scratchpad

Το Scratchpad είναι ένα από τα νέα, ενσωματωμένα, εργαλεία για δημιουργούς που επιτρέπει το πέρασμα και τη δοκιμή τμημάτων κώδικα JavaScript άμεσα στον περιηγητή, εξοικονομώντας χρόνο και ελαχιστοποιώντας την πληκτρολόγηση.

Βελτιωμένο API File

Με την προσθήκη υποστήριξης για .click() στο File upload control, .slice και file API urls, ο Firefox προσφέρει ακόμα περισσότερο έλεγχο για τη διαχείριση αποστολών αρχείων.

Σχεδίαση σελίδων με CSS3

Η υποστήριξη νέων χαρακτηριστικών CSS3 όπως border-radius, box-shadow, Transitions και Transformations κάνει εύκολη υπόθεση την προσθήκη κομψών εφέ κινούμενων εικόνων σε ιστοσελίδες.

API Audio

Ο Firefox αλλάζει τον τρόπο που τα πολυμέσα ενσωματώνονται στο διαδίκτυο. Η HTML5 διευκολύνει τους προγραμματιστές στη χρήση JavaScript για να διαβάσουν και να γράψουν δεδομένα ήχου. Αποκαλύπτουμε τα ακατέργαστα δεδομένα ήχου για να μπορείτε να ενσωματώσετε κορυφαίες ηχητικές δημιουργίες!

Υποστήριξη Multi-touch

Ο Firefox τώρα ενσωματώνει υποστήριξη για multi-touch σε Windows 7, επιτρέποντας σας έτσι να αλληλεπιδράτε με τον περιηγητή σας με ένα νέο τρόπο.

Web Console

Θέλετε να δείτε πως χτυπάει η καρδιά των πιο σύγχρονων ιστοσελίδων; Ο Firefox σας επιτρέπει να δείτε στα ενδότερα δυναμικών σελίδων μ' αυτό το πειραματικό εργαλείο ανάλυσης.

Μάθετε περισσότερα για τα νέα τεχνολογικά χαρακτηριστικά του Firefox.

Οικουμενική πρόσβαση

Θαυμαστές γραμματοσειρές

Ο Firefox υποστηρίζει ανοικτές μορφές γραμματοσειρών όπως οι WOFF, TrueType και OpenType, πράγμα που σημαίνει ότι όσοι σχεδιάζουν ιστοσελίδες δεν είναι αναγκασμένοι να περιορίζονται πια σε μια επιλεγμένη δράκα έτοιμων γραμματοσειρών. Αφού ο Firefox εξασφαλίζει την σωστή τυπογραφική απεικόνιση και προβολή ακόμα κι αν δεν έχει κάποιος εγκατεστημένη μια γραμματοσειρά στον υπολογιστή του, μπορείτε να δείτε μια πληθώρα γραμματοσειρών στην καθημερινή σας περιήγηση.

Η γλώσσα του Firefox

Ο Firefox μιλάει τη γλώσσα σας όσο κανένας άλλος περιηγητής. Το διαδίκτυο είναι ένα παγκόσμιος πόρος και επειδή πιστεύουμε ότι αυτός ο πόρος ανήκει σε όλους ανθρώπους, διαθέτουμε τον περιηγητή μας σε όσες περισσότερες γλώσσες είναι εφικτό. Προσφέρουμε πάνω από 70 διαφορετικές μεταφράσεις, από Γαλλικά και κινέζικα ως Φιλανδικά και Τσέχικα. Βρείτε τη γλώσσα σας (Πλήρης λίστα διαθέσιμων μεταφράσεων στον κεντρικό ιστότοπο -Αγγλικά).

Καλύτερη απεικόνιση κειμένου

Διαβάστε τη γλώσσα σας όπως γράφεται στην πραγματικότητα. Η μηχανή γραφικών μας απεικονίζει αραβικά, περσικά κι άλλες γλώσσες με δύσκολες γραμματοσειρές με αξιοθαύμαστα ακριβή τρόπο.

Έμφαση στις βοηθητικές τεχνολογίες

Μειώνοντας την ανάγκη πληκτρολόγησης, επιτρέποντας στους χρήστες να εστιάζουν σε ιστοσελίδες και υποστηρίζοντας οπτικούς αναγνώστες, ο Firefox προσφέρει ευκολία στη χρήση του από άτομα με κινητικά προβλήματα ή περιορισμένη όραση.

Οικουμενική προσαρμογή

Πέρα από μια απλή μετάφραση, σας προσφέρουμε μια προσαρμοσμένη στην κουλτούρα σας εμπειρία στο δίκτυο που να έχει νόημα. Ντόπιοι δημιουργοί που ξέρουν τι είναι καλύτερο για την κοινότητα τους, έχουν προσαρμόσει τον Firefox με λεξικά στη γλώσσα σας, μηχανές αναζήτησης, δυναμικούς σελιδοδείκτες και αρχεία ροής. Ανάλογα με το που ζει ο καθένας μπορεί να βλέπει περιεχόμενο όπως πχ νέα από το BBC που αφορούν την περιοχή του, ή λειτουργίες της κινέζικης μηχανής αναζήτησης Baidu κλπ.
