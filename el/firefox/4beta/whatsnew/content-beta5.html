Τι νέο υπάρχει στον

Ο Firefox αλλάζει τον τρόπο ενσωμάτωσης πολυμέσων στον ιστό: Το νέο API για τον ήχο αποκαλύπτει ασυμπίεστα τα δεδομένα ήχου που υπάρχουν μέσα σε στοιχεία <video> και <audio> της HTML5, διευκολύνοντας έτσι τους δημιουργούς στο να χρησιμοποιήσουν JS για να διαβάσουν και να γράψουν δεδομένα ήχου. Επιτάχυνση γραφικών (Μόνο σε Windows)

Δείτε φοβερά αποτελεσματική επιτάχυνση γραφικών με χρήση Direct2D, που τώρα είναι ενεργοποιημένη σαν προεπιλογή, σε μηχανήματα που υποστηρίζουν DirectX 10.

Ασφαλείς συνδέσεις σε διακομιστές

Κάνοντας χρήση του HSTS, οι σελίδες μπορούν τώρα να ζητήσουν από τον Firefox να πραγματοποιήσει αυτόματα ασφαλή σύνδεση στους διακομιστές τους, εμποδίζοντας έτσι επιτήδειους από το να υποκλέψουν σημαντικά δεδομένα που μπορεί να κινδυνεύουν να υποκλαπούν κατά τη διάρκεια φόρτωσης της σελίδας.
