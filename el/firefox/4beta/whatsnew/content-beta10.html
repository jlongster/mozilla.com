Χρειαζόμαστε περισσότερα σχόλια!

Χάρη σε εσάς, έχουμε ήδη πάρει πάνω από ένα εκατομμύριο σχόλια για τον Firefox 4 Beta. Όσο περισσότερα όμως, τόσο καλύτερα! Κάντε απλά κλικ στο κουμπί σχολιασμού στην πάνω δεξιά γωνία του περιηγητή για να μας πείτε τι σας αρέσει και τι όχι (και μην ξεχνάτε να βάζετε και URL).

  • Πείτε μας τα σχόλια σας »

Πως εμφανίζονται οι αγαπημένες σας σελίδες;

Φορτώνονται γρήγορα; Μήπως βλέπετε κάποιες ασυνήθιστες γραμμές ή πλαίσια; Μπορείτε να εισάγετε δεδομένα σε φόρμες; Δουλεύει το Flash; Μπορείτε να δείτε βίντεο; Πείτε μας αν κάποια σελίδα έχει πρόβλημα.

Μπορείτε να παίξετε το αγαπημένο σας διαδικτυακό παιχνίδι;

Βοηθήστε μας να επιβεβαιώσουμε ότι σελίδες που χρησιμοποιούν Flash κι άλλα στοιχεία πολυμέσων, δουλεύουν σωστά στον Firefox 4. Δουλεύουν πιο γρήγορα; Μήπως τα παιχνίδια προκαλούν καταρρεύσεις; Πείτε μας ποια παιχνίδια δουλεύουν και ποια όχι.

Ποια πρόσθετα έχετε δοκιμάσει;

Είναι συμβατά τα αγαπημένα σας πρόσθετα με τη δοκιμαστική έκδοση του Firefox 4; Ποια δεν δουλεύουν; Σας βοηθάει ο νέος διαχειριστής πρόσθετων στην οργάνωση τους; (Σημείωση: Για να δοκιμάσετε πρόσθετα με τη δοκιμαστική έκδοση του Firefox 4, ψάξτε για πρόσθετα που περιλαμβάνουν την τιμή «4.0.*» στο πεδίο «Δουλεύει με τον».) Πείτε μας την άποψη σας.

Αν θέλετε να βοηθήσετε ακόμα περισσότερο, εγκαταστήστε το πρόσθετο Add-ons Compatibility Reporter. Με αυτή την επέκταση, αν παρατηρήσετε ότι κάποιο από τα πρόσθετα σας δεν φαίνεται να δουλεύει όπως δούλευε στις προηγούμενες εκδόσεις του Firefox, απλά ανοίγετε το διαχειριστή πρόσθετων και κάνετε κλικ στο κουμπί «Compatibility» δίπλα στο πρόσθετο για να στείλετε μια αναφορά στους υπεύθυνους του Mozilla.

Έχετε δημιουργήσει λογαριασμό στο Sync;

Αν έχετε ήδη λογαριασμό Sync, πως σας φάνηκε η διαδικασία ρύθμισης; Μπορείτε να συγχρονίσετε τα δεδομένα σας με κάποια συσκευή Android που έχετε τη δοκιμαστική έκδοση για κινητά ή σε iPhone μέσω του Firefox Home; Αν δεν έχετε φτιάξει λογαριασμό Sync, μπορέσατε τουλάχιστον να εντοπίσετε εύκολα το χαρακτηριστικό στον περιηγητή σας; Πείτε μας τα σχόλια σας για το Sync.
