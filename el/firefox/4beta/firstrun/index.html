Πείτε μας την άποψη σας

Σαν χρήστες της δοκιμαστικής έκδοσης, είστε μέλη της ομάδας που φτιάχνει τον Firefox 4. Πείτε μας τι είναι αυτό που σας αρέσει ή δεν σας αρέσει κάνοντας κλικ στο κουμπί σχολιασμού. Θα σας παρακαλούσαμε να πάρετε μέρος στις ανώνυμες μελέτες χρήσης ή να συμπληρώσετε τυχόν ερωτηματολόγια που θα εμφανιστούν.

Πως λειτουργούν οι μελέτες χρήσης;

Οι μελέτες χρήσης συγκεντρώνουν ανώνυμα στατιστικά στοιχεία για το πως χρησιμοποιείτε τον Firefox, χωρίς να καταγράφεται τι κάνετε και τι βλέπετε στο διαδίκτυο σε καμία περίπτωση. έχετε τον απόλυτο έλεγχο των δεδομένων και τίποτα δεν αποστέλλεται στους διακομιστές του Mozilla Test Pilot χωρίς την έγκριση σας. Όλες οι πληροφορίες θα χρησιμοποιηθούν αποκλειστικά για να βοηθήσουν στο σχεδιασμό του Firefox 4.

Τι γίνεται αν δεν θέλω να συμμετάσχω;

Θα ενημερωθείτε πριν ξεκινήσει να συλλέγει δεδομένα οποιαδήποτε μελέτη και θα ειδοποιηθείτε ξανά πριν γίνει οποιαδήποτε αποστολή δεδομένων στο Mozilla. Μπορείτε να επιλέξετε την αποχώρηση σας από οποιαδήποτε μελέτη ή να απενεργοποιήσετε όλες τις μελέτες μέσα από τις επιλογές που θα βρείτε κάνοντας κλικ στο κουμπί σχόλια.
