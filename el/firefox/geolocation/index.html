Περιήγηση με βάση την τοποθεσία

Ο Firefox μπορεί να γνωστοποιήσει την τοποθεσία σας σε ιστοσελίδες ώστε να σας παρέχουν πιο χρήσιμες και σχετικές πληροφορίες. Έτσι γίνεται πιο «έξυπνος» ο Ιστός – και γίνεται με ένα τρόπο που σέβεται απόλυτα το ιδιωτικό σας απόρρητο. Δοκιμάστε το!

Συχνές ερωτήσεις

Τι είναι η περιήγηση με βάση την τοποθεσία;

Οι ιστοσελίδες που χρησιμοποιούν την περιήγηση με βάση την τοποθεσία, θα ρωτούν την τοποθεσία σας για να μπορούν να σας παρέχουν πιο σχετικές πληροφορίες, ή για να ελαχιστοποιήσουν το χρόνο αναζήτησης. Ας πούμε πως ψάχνετε για μια πιτσαρία στην περιοχή σας. Μια ιστοσελίδα μπορεί να σας ζητήσει να γνωστοποιήσετε την τοποθεσία σας ώστε απλά αναζητώντας τη λέξη «πίτσα» να έχετε τις απαντήσεις που θέλετε, χωρίς περαιτέρω διευκρινήσεις ή πληκτρολόγηση.

Ή ακόμα, αν προσπαθείτε να χαρτογραφήσετε τη διαδρομή σας για κάποιο προορισμό, η ιστοσελίδα θα ξέρει από που ξεκινάτε οπότε το μόνο που έχετε να κάνετε είναι να δώσετε τον προορισμό σας.

Αυτή η υπηρεσία είναι εντελώς προαιρετική – ο Firefox δεν γνωστοποιεί την τοποθεσία σας χωρίς την άδεια σας – και πραγματοποιείται με τον μέγιστο σεβασμό στο ιδιωτικό σας απόρρητο. Και όπως όλα τα συστατικά στοιχεία του Firefox, υλοποιείται πάνω σε ανοικτά πρότυπα ώστε να διευκολύνει την υιοθέτηση της από τους δημιουργούς εφαρμογών για τον Ιστό.

Πως δουλεύει;

Όταν επισκέπτεστε μια ιστοσελίδα που χρησιμοποιεί την περιήγηση με βάση την τοποθεσία, ο Firefox θα σας ρωτήσει αν θέλετε να γνωστοποιήσετε την τοποθεσία σας.

Αν συμφωνήσετε, ο Firefox θα συλλέξει πληροφορίες για κοντινά σημεία ασύρματης πρόσβασης και τη διεύθυνση IP του υπολογιστή σας. Μετά, ο Firefox θα αποστείλει τις πληροφορίες αυτές τον προεπιλεγμένο παροχέα υπηρεσιών τοποθεσίας, την υπηρεσία Google Location Services, για να πάρει μια εκτίμηση για την τοποθεσία σας. Αυτή η εκτίμηση της τοποθεσίας σας θα αποσταλεί μετά στην ιστοσελίδα που τη ζήτησε.

Αν φυσικά δεν συμφωνήσετε, ο Firefox δεν θα κάνει καμία ενέργεια.

Πόσο ακριβείς είναι οι τοποθεσίες;

Η ακρίβεια διαφέρει από τοποθεσία σε τοποθεσία. Σε μερικά μέρη, οι πάροχοι της υπηρεσίας μπορεί να μας παρέχουν στοιχεία με ακρίβεια μερικών μέτρων. Σε άλλες όμως περιοχές η ακριβής τοποθεσία μπορεί να απέχει πολύ περισσότερο. Όλες οι πληροφορίες τοποθεσίας που παρέχονται από τους παρόχους μας είναι απλά εκτιμήσεις και δεν εγγυόμαστε την ακρίβεια τους. Παρακαλούμε μην τις χρησιμοποιείτε για καταστάσεις έκτακτης ανάγκης. Χρησιμοποιήστε πάντα την κοινή λογική.

Τι πληροφορίες αποστέλλονται και σε ποιόν; Πως προστατεύεται το ιδιωτικό μου απόρρητο;

Για εμάς, το ιδιωτικό σας απόρρητο είναι εξαιρετικά σημαντικό και ο Firefox δεν γνωστοποιεί την τοποθεσία σας ποτέ χωρίς την έγκριση σας. Όταν επισκέπτεστε κάποια ιστοσελίδα που ζητάει αυτή την πληροφορία, θα ερωτηθείτε πριν αποσταλεί οποιαδήποτε πληροφορία προς την ιστοσελίδα αυτή ή τον πάροχο της υπηρεσίας αυτής (τρίτο μέρος).

Σαν προεπιλογή, ο Firefox χρησιμοποιεί την υπηρεσία Google Location Services για να καθορίσει την τοποθεσία σας αποστέλλοντας:

  • την διεύθυνση IP του υπολογιστή σας,
  • πληροφορίες για κοντινά σημεία ασύρματης πρόσβασης και
  • ένα τυχαίο αριθμό ταυτοποίησης πελάτη, ο οποίος αποδίδεται από την Google και έχει ισχύει για δυο εβδομάδες.

Για μια ολοκληρωμένη περιγραφή των πληροφοριών που συλλέγονται και χρησιμοποιούνται από τον Firefox, παρακαλούμε δείτε την πολιτική απορρήτου του Firefox (Αγγλικά).

Μετά, η υπηρεσία Google Location Services επιστρέφει την τοποθεσία σας κατά προσέγγιση (δηλ. γεωγραφικό πλάτος και μήκος). Για μια ολοκληρωμένη περιγραφή των πληροφοριών που συλλέγονται και χρησιμοποιούνται από την Google, παρακαλούμε δείτε την πολιτική απορρήτου Google Geolocation (Αγγλικά).

Οι πληροφορίες ανταλλάσσονται με κρυπτογραφημένη σύνδεση για να προστατευθεί το απόρρητο σας. Όταν ο Firefox αποκτήσει τις πληροφορίες για την τοποθεσία σας, τις μεταφέρει στην ιστοσελίδα που τις ζήτησε. Δεν αποστέλλονται στην υπηρεσία Google Location Services σε καμία περίπτωση πληροφορίες για την ιστοσελίδα που επισκέπτεστε ή την τοποθεσία του, ούτε cookies..

Ούτε το ίδρυμα Mozilla ούτε η Google θα χρησιμοποιήσουν ποτέ τις πληροφορίες που συλλέγονται από την υπηρεσία Google Location Services για να σας ταυτοποιήσουν ή να σας κατασκοπεύσουν.

Για πληροφορίες σχετικά με τη χρήση των πληροφοριών τοποθεσίας από την ιστοσελίδα που τις ζήτησε, παρακαλούμε δείτε την πολιτική απορρήτου της ιστοσελίδας αυτής.

Για περισσότερες πληροφορίες σχετικά με το ιδιωτικό σας απόρρητο, διαβάστε επίσης:

  • Πολιτική απορρήτου Mozilla Firefox (Αγγλικά)
  • Πολιτική απορρήτου Google (Αγγλικά)

Παρακολουθούμαι όταν περιηγούμαι στο δίκτυο;

Όχι. Ο Firefox ζητάει την τοποθεσία σας μόνο όταν κάποια ιστοσελίδα του τη ζητήσει και την γνωστοποιεί μόνο αν ο χρήστης εγκρίνει το αίτημα. Ο Firefox δεν παρακολουθεί ούτε απομνημονεύει την τοποθεσία σας κατά την περιήγηση.

Πως μπορώ να αναιρέσω την άδεια που έχω δώσει σε μια ιστοσελίδα;

Αν ρυθμίσετε τον Firefox ώστε να γνωστοποιεί πάντα την τοποθεσία σας σε κάποια ιστοσελίδα και μετά αλλάξετε γνώμη, μπορείτε να αλλάξετε εύκολα αυτή τη ρύθμιση. Ιδού πως:

  • Πηγαίνετε στην ιστοσελίδα που έχετε δώσει το δικαίωμα
  • Πηγαίνετε στο μενού «Εργαλεία» και μετά επιλέξτε «Πληροφορίες σελίδας»
  • Επιλέξτε από τον διάλογο την καρτέλα «Δικαιώματα»
  • Αλλάξτε τη ρύθμιση για την «Γνωστοποίηση τοποθεσίας»

Πως μπορώ να εκκαθαρίσω τον «τυχαίο αριθμό ταυτοποίησης πελάτη»;

  • Πηγαίνετε στο μενού «Εργαλεία» και επιλέξτε «Εκκαθάριση ιστορικού...»
  • Στο διάλογο «Εκκαθάριση πρόσφατου ιστορικού», κάντε κλικ στο βέλος του «Λεπτομέρειες»
  • Σιγουρευτείτε πως είναι επιλεγμένα τα «Cookies»
  • Κάντε κλικ στο «Εκκαθάριση τώρα»

Πως μπορώ να απενεργοποιήσω την περιήγηση με βάση τοποθεσίες οριστικά;

Η περιήγηση με βάση την τοποθεσία είναι πάντα προαιρετική στον Firefox. Δεν αποστέλλεται ποτέ καμία πληροφορία τοποθεσίας χωρίς την άδεια σας. Αν θέλετε να απενεργοποιήσετε εντελώς αυτό το χαρακτηριστικό, παρακαλούμε ακολουθήστε τα παρακάτω βήματα:

  • Πληκτρολογήστε about:config στην εργαλειοθήκη διευθύνσεων
  • Πληκτρολογήστε geo.enabled
  • Κάντε διπλό κλικ στο φίλτρο geo.enabled
  • Η περιήγηση με βάση τοποθεσίες έχει πια απενεργοποιηθεί.

Παρουσιάστηκε σφάλμα κατά την προσπάθεια εντοπισμού της τοποθεσίας σας.
