부가 기능

Firefox는 웹 서비스 이용시 온라인 사용자 경험을 증진시키기 위한 기능들을 가지고 있습니다. 독특한 요구 사항을 충족시키기 위해 사용자들이 직접 만든 5,000여 가지의 부가 기능을 이용하시면 더 재미있고 창의적인 인터넷 여행을 경험하실 수 있습니다.

바로 만나 보고 싶으시다면 부가 기능 사이트를 방문해 보세요. 한국어 사용자가 자주 사용하는 부가 기능 목록도 참고하세요.
