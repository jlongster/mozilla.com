위치 인식 서비스

Firefox에서는 여러분이 이용하고 있는 웹 사이트가 현재 위치 정보를 얻어 유용한 서비스를 제공할 수 있도록 하는 기능을 추가하였습니다. 이는 웹 서비스가 좀 더 스마트해 지도록 하는데 도움을 줌과 동시에 개인 정보는 철처하게 관리될 것입니다. 테스트 하기

자주 질문 하는 사항(FAQ)

위치 인식 서비스란 무엇인가요?

위치 인식 서비스가 가능한 웹 사이트는 여러분의 현재 이용 위치를 기반으로 하여 적절하고 유용한 정보를 제공하게 됩니다. 만약 여러분 주위의 피자 가게를 찾고자 한다면 웹 사이트가 여러분의 위치를 알게 됨으로서 '피자'라고 검색만 함으로서 가게를 쉽게 찾아서 알려 주게 됩니다.

또는, 여러분이 어떤 곳으로 갈려고 할 때 웹 서비스가 여러분의 현재 출발점을 알고 있기 때문에 원하는 곳만 입력하면 바로 최단 경로를 찾아 줄 수 있습니다.

본 서비스는 전적으로 선택 가능합니다. Firefox는 여러분의 허가 없이 절대로 웹 사이트에 정보를 제공하지 않습니다. 사용자의 개인 정보 보호를 가장 중요하게 생각 함과 동시에 공개 표준을 채택함으로서 웹 개발자들이 쉽게 구현 할 수 있습니다.

어떻게 동작하나요?

위치 인식 기반 웹 서비스에 바문하면 Firefox는 현재 당신의 위치를 웹 사이트에 제공할 것인가를 물어 보게 됩니다.

만약 여러분이 허가 하면, Firefox는 주변의 무선 접속기 혹은 PC의 아이피 주소 정보를 얻은 후 이를 기본 위치 인식 서비스 제공자(Geolocation service provider)인 구글 위치 서비스(Google Location Services)로 보내어 현재 위치를 추정하게 됩니다. 추정된 위치 정보(위경도 좌표)가 웹 사이트로 전달 되게 됩니다.

여러분이 허가 하지 않으면 Firefox는 아무 작업도 하지 않게 됩니다.

어떻게 위치를 정확하게 파악 하나요?

정확도는 지역에 따라 매우 다릅니다. 어떤 지역은 서비스 제공자가 수미터 이내로 정확하게 파악할 수 있는 반면 다른 곳은 그렇게 할 수 없는 곳도 많습니다. 서비스 제공자로 부터 받은 위치 정보는 확률적 추정에 의한 것으로 정확성을 보장하지 않습니다. 이 서비스를 긴급하거나 정확한 정보를 제공해야 할 경우에는 사용하지 마시고 일반적 상식 수준에서 이용해 주십시오.

무슨 정보가 누구에게 전달 되나요? 개인 정보가 어떻게 보호되나요?

여러분의 개인 정보는 매우 중요합니다. Firefox는 결코 여러분의 허가 없이 개인 정보를 공유하지 않습니다. 웹 페이지를 방문하면 정보를 공유할지 여부를 묻게 되고 질문에 답하기 전에는 제3자에게 정보가 제공되지 않습니다.

기본적으로 Firefox는 Google Location Services에 아래와 같은 정보가 전달 됩니다.

  • PC의 IP 주소
  • 근처의 무선 억세스 포인트의 정보
  • 무작위 클라이언트 인식자 (구글이 할당하며 2주 후 폐기됨)

Firefox가 수집하는 정보에 대해 Firefox 개인 정보 보호 정책(영문)을 자세히 살펴 보시기 바랍니다.

Google Location Services는 추정된 위치 정보 (예, 위경도 좌표)를 반환 하며 이에 대한 자세한 사항은 Google Geolocation Privacy Policy (영문)을 참가하시기 바랍니다.

모든 정보는 암호화된 연결을 통해 주고 받게 되고, 반환받은 위치 정보는 웹 사이트로 전달 됩니다. 이 때 구글 위치 서비스에 방문하는 웹사이트 정보나 쿠키 등은 절대로 전달 되지 않습니다.

Mozilla와 Google 모두 수집된 정보를 신원 확인이나 감시의 목적으로 절대 사용하지 않습니다.

요청한 웹 사이트는 여러분의 위치 정보를 어떻게 활용하는지 다양한 목적이 있을 수 있으므로 웹 사이트의 개인 정보 페이지를 참고해 주시기 바랍니다.

더 자세한 정보는 아래 웹 페이지를 참고하세요.

  • Mozilla 개인 정보 보호 정책 (in English)
  • Google 개인 정보 보호 정책 (in English)

나의 웹 서핑 기록이 추적 되지 않나요?

아닙니다. Firefox는 웹 사이트가 요청하는 위치 정보만을 제공 합니다. 여러분의 웹 사용 기록을 보내지 않습니다.

위치 서비스 허가 목록을 삭제하려면 어떻게 하나요?

만일 여러분이 허가한 웹 사이트에 더 이상 위치 정보를 제공하고 싶지 않다면 다음과 같이 하시면 됩니다.

  • 위치 정보를 제공하지 않으려는 웹 사이트로 갑니다.
  • 도구 메뉴의 페이지 정보를 선택 합니다.
  • 이용 권한을 선택 합니다.
  • 위치 정보 공유 체크를 해제합니다.

무작위 클라이언트 인식자를 어떻게 삭제 합니까?

  • 도구 메뉴에서 개인 정보 초기화를 선택합니다.
  • 상세 정보를 클릭합니다.
  • 쿠키를 선택합니다.
  • 지금 삭제를 클릭합니다.

위치 기반 서비스를 아예 쓰지 않으려면 어떻게 합니까?

위치 기반 서비스는 Firefox의 기본 설정으로 제공 됩니다. 만약 아예 서비스를 이용하고 싶지 않으려면 아래와 같은 방법을 이용하시면 됩니다.

  • 주소창에 about:config 라고 입력 합니다.
  • 검색창에 geo.enabled 라고 입력 합니다.
  • geo.enabled을 더블 클릭합니다.
  • 서비스 이용이 해제되었습니다.

이런! 위치 정보 이용에 오류가 생겼습니다.
