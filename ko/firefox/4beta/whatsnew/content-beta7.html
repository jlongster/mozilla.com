주요 기능

엄청난 속도 향상

Firefox 4는 주요 성능 향상을 가져왔습니다. 새롭게 개발된 JägerMonkey 자바스크립트 엔진과 함께 더 빨라진 구동 속도, 페이지 로딩 속도와 그래픽 표현 등 바로 느끼실 수 있을 것입니다.

웹 기반 3D 기술

WebGL은 Firefox에 3D 기술을 제공합니다. 웹 개발자들이 새로운 시각효과를 만들고 멋진 3D 게임을 개발할 수 있습니다.

글꼴을 자유자재로

Firefox 4의 OpenType 글꼴 기능은 웹 개발자들이 글꼴 크기나 장평들을 자유롭게 조절하여 더 미려한 웹 사이트로 만들 수 있습니다.

새로운 웹 폼

Firefox 4는 웹 폼 양식에 목록 기능을 제공하여 자동 완성을 제공하고, 전화번호 이메일 주소와 같은 경우 웹 브라우저가 직접 유효성 확인을 해주는 HTML5 Webform의 일부 기능을 제공하여 웹 개발자들의 개발 시간을 단축해 줍니다.
