주요 기능

Firefox 4!

Firefox 4 Beta는 여러분과 정식 출시를 위해 여러분과 같은 고급 사용자의 도움이 필요합니다. Firefox 4의 새로운 기능을 조금씩 이용해 보시고 문제가 있는지 웹 서핑에 도움이 되는지 알려 주시기 바랍니다. 여러분의 의견은 상단 오른쪽의 의견 보내기 버튼을 통해 보내 주시면 됩니다.

웹 사이트 호환성

여러분이 자주 가는 웹 사이트가 잘 동작 하시나요? Firefox 4를 통해 웹 사이트 로그인, 동영상 재생 그리고 글쓰기 등에 문제가 있으시면피드백이나 한국 사용자 포럼에 알려 주십시오.

동기화

아이폰용 Firefox Home 앱이나 안드로이드용 Firefox 4 Beta를 모바일폰에서 사용하고 계시면 지금 여러분의 북마크, 방문 기록, 탭 정보 등을 PC와 모바일 관계 없이 어디서나 이용 할 수 있습니다.

파노라마

파노라마 기능(탭 그룹)을 사용해 보셨나요. 오른쪽 상단의 파노라마 아이콘을 클릭하시면, 탭을 여러분의 입맛대로 분류해서 쉽게 관리하면서 사용할 수 있는 기능을 제공합니다. 사용해 보시고 많은 의견 주시기 바랍니다.

플러그인 호환성

플러그인은 동영상, 애니메이션, 게임 등 각종 웹 사이트의 추가 기능을 제공해 주지만, 때로는 Firefox를 느리게 하거나 문제를 일으키는 주요 원인이 됩니다. 플러그인이 잘 동작 하는지 알아보시고 문제가 있으시면 의견주시기 바랍니다.
