주요 기능

Firefox는 멀티미디어를 웹에서 다루는 방법을 변화시키고 있습니다. 새로운 Audio API를 통해 HTML5 <video>와 <audio>의 소스 데이터를 읽고 사용할 수 있는 자바스크립트 기능을 제공합니다. 자세한 것은 동영상에서 확인하세요.

최근 추가 기능

하드웨어 그래픽 가속(윈도용)

Direct2D 하드웨어 가속 기능을 통해 DirectX 10을 지원하는 PC의 경우 더 빠르게 그래픽 기능을 사용할 수 있습니다.

안전한 서버 연결

HSTS(HTTPS 강제 연결 기능)을 통해 이제 웹 서버 설정으로 Firefox에서 자동으로 SSL 서버로 접속할 수 있는 기능을 제공합니다. 해커가 중간에 HTTP서버로 변경하는 공격을 막아줌으로서 더 안전하게 웹 서핑을 즐길 수 있습니다. 자세히 보기(영문) »
