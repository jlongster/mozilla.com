중급 사용자

태그 이용하기

북마크에 자주 가는 사이트를 계속 추가하다 보면 관리하는 일이 매우 힘들어 집니다. 이를 효율적으로 관리하려면 각 사이트에 태그를 붙이는 것이 좋습니다. 주소창 오른쪽 별표를 한번만 누르면 북마크와 동시에 태그를 달 수 있습니다.

웹 페이지를 설명하는 키워드를 쉼표로 분리하여 원하는 만큼 입력할 수 있습니다. 나중에 주소창에 태크만 입력하면 북마크한 웹 페이지가 자동 완성으로 추천 됩니다. 예를 들어 "여행"이라고 주소창에 치면 여름 방학에 가려던 관광지를 "여행"이라고 키워드로 태그를 단 웹 사이트를 자동으로 가져 옵니다.

부가 기능 이용하기

Firefox 기본 버전에는 다양한 기능이 포함되어 있습니다만 여러분만의 특화된 기능을 사용할 수 있는 부가 기능도 제공하고 있습니다. 5,000여가지가 넘는 확장 기능 프로그램은 일반 사용자들이 직접 만들어 배포하는 것으로 여러분이 원하는 기능을 바로 찾으실 수 있습니다. '도구'→'부가 기능'의 부가 기능 관리자를 통해 직접 검색하고 설치함으로서 자기만의 Firefox로 업그레이드 할 수 있습니다.

똑똑한 검색 기능

Firefox 오른쪽 상단에 검색창을 이미 잘 아실 것입니다. 하지만 검색 엔진을 여러분이 원하는 것으로 바꿀 수 있다는 것은 알고 계시나요? 왼쪽 검색 아이콘을 클릭하고 원하는 검색 엔진을 선택하면 기본 설정이 바뀝니다.

뿐만 아니라 '검색 사이트 관리…' 메뉴를 통해 검색 엔진 순서 변경 및 추가/삭제 등이 가능 합니다. 특히, 각 검색 엔진의 키워드를 설정하면 주소창에서 바로 검색 결과를 얻을 수 있습니다. 예를 들어, 주소창에서 Google의 검색 결과를 얻으려면 'g'라고 키워드 설정을 하신 후, 주소창에서 'g 요리법'이라고 치면 됩니다.

향상된 찾기 기능

'빠른 검색'(Find As You Type)은 시간 절약을 위한 또 다른 강력한 기능입니다. 기존의 페이지 찾기 기능이 아니라 문서에서 바로 타이핑하는 순서 대로 문서 내 단어를 찾아줍니다. 커서가 타이핑 하는 단어가 나온 첫번째 위치로 바로 이동 합니다.

링크를 찾을 때도 사용할 수 있습니다. 예를 들어 마우스를 이용하지 않고도 바로 링크로 이동 가능 합니다. 이 기능은 '도구→설정→고급'에서 "입력을 시작하면 내용 검색하기"에서 사용 여부를 설정할 수 있습니다.

모든 정보 삭제하기

보안이나 개인 정보 보호 문제로 인해, 외부에 있는 공용 PC를 이용하게 된 경우 웹 사이트 이용 기록을 모두 삭제할 필요가 있습니다. Firefox는 웹 이용 기록, 다운로드 기록, 쿠키 및 이용 기록을 한번에 삭제할 수 있습니다. '도구 → 개인 정보 초기화…'를 선택하시면 됩니다. 또는 도구 → 설정 → 개인 정보에서 "Firefox를 종료할 때 개인 정보 모두 삭제" 설정을 이용하실 수 있습니다.

고급 사용자

즐겨 찾는 사이트 바로 가기

북마크한 사이트에 바로 접속할 수 있도록 키워드를 설정할 수 있습니다. 북마크 관리에서 각 웹 사이트의 키워드 항목에 간단한 단어를 설정하고 주소창에서 입력만 하면 웹 사이트로 바로 이동 합니다. 예를 들어, www.daum.net 이라는 웹 사이트의 경우 '메일'이라고 키워드를 설정해 바로 접속할 수 있습니다.

스마트 폴더 만들기

여러분이 고급 웹 사용자라면 특정 키워드에 대한 웹 사이트 목록을 보거나 정보 변경을 항상 염두해야 할 필요성이 있을 것입니다. Firefox 3에서 북마크에 웹 사이트가 추가될 때 마다 특정 검색어 포함되어 있으면 자동으로 표시해 주는 스마트 폴더라는 기능이 있습니다.

북마크 메뉴에서 '북마크 구성'을 선택 한 후 검색창에 특정 키워드를 입력 한후 저장 버튼을 눌러 스마트 폴더를 만들 수 있습니다.

다운로드 항목 관리

웹 상에서 다운로드를 자주 하신다면 다운로드 관리자를 통해 파일을 관리하실 수 있습니다. 다운로드 진행을 중단하거나 재개할 수도 있고 다운로드한 파일이나 파일의 저장 폴더를 바로 열 수도 있습니다.

다운로드 파일을 찾으시려면 도구 → 다운로드에서 관리자 창을 열어 검색창에 파일명 중 일부를 입력하시면 됩니다. 바로 파일을 열거나, PC에서 파일이 삭제된 경우 "다운로드 링크 복사"를 통해 다시 다운로드 가능합니다.

개발 도구로 사용하기

여러분이 웹 개발자라면 Firefox의 개발 도구를 통해 편하게 개발 하실 수 있습니다. Mozilla 부가 기능 사이트에서는 CSS, HTML 및 자바스크립트를 수정 및 디버깅 할 수 있는 Firebug 부가 기능을 비롯 HTTP/HTTPS 헤더 기록을 볼 수 있는 Tamper Data, HTML/CSS 요소를 조사할 수 있는 DOM Inspector 같은 부가 기능을 다운로드하여 사용할 수 있습니다.

자신이 원하는 웹 서비스 이용

웹 기반 프로토콜 핸들러를 이용하여 여러분이 원하는 웹 사이트에서 기능을 이용할 수 있습니다. 예를 들어 Firefox는 mailto: 라는 링크에 이메일 주소가 있을 경우 새로운 메일 작성을 위해 아웃룩 같은 외부 프로그램을 이용하지 않고 웹 메일 서비스 업체에서 바로 메일 쓰기가 가능해졌습니다. Firefox 3 한국어 버전에는 '한메일', '네이버 메일', '지메일', '야후!메일' 중 하나를 선택할 수 있습니다.

도구 → 설정 → 프로그램을 항목에서 각 프로토콜에 해당 하는 실행 방법을 "항상 물어보기" 또는 원하는 프로그램이나 웹 서비스로 바꾸실 수 있습니다.

여러분이 웹 개발자라면 웹 기반 프로토콜 핸들러를 지원하는 방법을 Mozilla 개발자 센터에서 확인 가능 합니다.
