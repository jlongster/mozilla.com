플러그인 검사기

여러분의 웹 브라우저의 보안 및 안정성을 위해 제 3자가 제공하는 플러그인의 최신 버전을 확인하고 업데이트해야 합니다.

FLASH가 없나요?

여러분의 보안을 위해 Firefox는 오래된 Flash 버전은 자동으로 차단합니다. >최신 버전으로 업그레이드하시기 바랍니다.


  • 새 버전이 나왔습니다. 여러분의 웹 브라우저를 지금 업그레이드 하십시오.
  • 이 페이지는 아래 웹 브라우저에서 보실 수 있습니다: Firefox 3.6+, Opera 10.5, Safari 4, Chrome 4, 또는 IE 8
  • 플러그인 검사기는 IE에서 제한적으로 사용 가능합니다. 도구 > 부가 기능 관리를 참고하세요
  • 1단계: 플러그인을 업데이하려면 업데이트를 선택하세요.
  • 2단계: 웹 브라우저를 재시작하기 전에 모든 업데이트를 완료하시기 바랍니다.

플러그인 검사 오류

플러그인을 검사하는데 오류가 생겼습니다. 다시 한번 더 시도해 주시기 바랍니다.

자주 묻는 질문

플러그인이란 무엇입니까?

  • 플러그인은 동영상, 애니메이션, 게임 등을 가능하게 합니다.
  • Firefox가 아닌 외부 회사가 만듭니다. 예를 들어, >Adobe Systems나 >Apple.
  • 플러그인은 항상 자동 업데이트 되지 않습니다.

왜 플러그인을 업데이트 해야 합니까?

  • 오래된 플러그인은 작동이 늦어서 여러분의 시간을 낭비할 수 있습니다.
  • 오래된 플러그인은 악성 코드, 바이러스 등 보안 취약점 때문에 위험을 증가 시킵니다.
  • 최신 플러그인은 웹을 더 좋고 안전한 곳으로 만듭니다.

Firefox를 어떻게 도울 수 있나요?

앞으로 Firefox는 플러그인 업데이트 정보를 제공하고자 합니다. 그 때까지 주기적으로 본 웹페이지를 방문하셔서 플러그인을 업데이트 하십시오.

플러그인은 어디에 있나요?

본 웹 페이지는 자동으로 Firefox에 설치된 플러그인 목록을 아래와 같은 방법으로 찾습니다.:

  1. 도구 메뉴 열기.
  2. 부가 기능 선택하기.
  3. 플러그인 탭 선택하기

How do I disable a plugin?

In Firefox:

  1. 도구 메뉴 열기.
  2. 부가 기능 선택하기.
  3. 플러그인 탭 선택하기
  4. 현재 페이지에서 플러그인 클릭
  5. 차단 부튼 누르기.

주의: 플러그인을 사용 중지 하면 더 이상 사용할 수 없습니다. 예를 들어 플래시를 차단하면, 다른 동영상 사이트에서 플래시 동영상은 볼 수 없습니다.
