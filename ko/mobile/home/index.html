왜 아이폰에서 Firefox Home을 써야 하나요?

이동 시 편리한 환경

여러분이 자리에서 일어나서 이동을 할 때 아이폰에서 기다림 없이 편안히 바>로 하던 작업을 계속 할 수 있습니다. 북마크, 방문기록, 탭 정보, 스마트 주소창 등>을 바로 이용할 수 있기 때문입니다.

편리한 입력 환경

휴대폰에서 또 다시 정보를 저장할 필요가 없습니다. PC에서 작업하던 그대로 바로 하고 싶은 일을 할 수 있습니다.

안전하고 편리한 서비스

여러분의 개인 정보를 철저하게 보호할 보안 서비스를 갖추고 있으며 암호 및 암호키를 이용하여 이중으로 암호화 합니다.

시작하기

  1. 무료 Firefox Sync 부가 기능을 PC에 설치한다.

    Firefox Home은 Firefox Sync 부가 기능의 동기화 기능을 이용하여 여러 디바이스로 부터 같은 데이터를 이용할 수 있도록 해 줌으로서 여러분의 iPhone에서도 이용 가능합니다.

  2. Firefox를 재시작 하여 아이디, 계정을 만들어서 암호 및 암호키를 입력하여 로그인한다.

    Firefox Home은 암호키를 이용하여 정보 보호를 위해 모든 정보를 암호화 합니다.

  3. Firefo Sync는 서비스간 동기화를 시작합니다. 초기 작업일 경우 약 20분 정도가 소요됩니다.

    Firefox Sync는 사생활 보호 모드를 제외하고 항상 동작합니다. 초기 동기화가 완료된 후 도구 > Sync 메뉴에 최근 업데이트를 보실 수 있습니다.

  4. PC 설치를 완료하면 아이폰에 설치된 프로그램에 사용자명, 암호 및 암호키를 입력하고 완료를 선택합니다.

    만약 로그인 시 문제가 있으시면 도움말을 보시기 바랍니다.

  5. 설정이 완료되면 아이폰에서 PC의 북마크, 방문 기록, 탭 정보를 보실 수 있습니다.

    테이터가 보이지 않으신다면 도움말을 참고하시기 바랍니다.
