Firefox 4 베타 동기화 설정 소개
여러분의 컴퓨터나 휴대폰을 가지고 언제 어디서나 Firefox 동기화 설정을 통해
북마크, 방문 기록, 암호 및 작업하던 탭들 모든 정보를
빠르고 쉽게 접근하실 수 있습니다.
재미있는 웹 사이트를 발견했는 데 급한 일로 나가셔야 할 때
동기화 설정으로 집에 왔을 때 또는 모바일 폰으로도 그 웹 사이트를 바로 다시 보실 수 있습니다. 놀랍지요?
동기화를 가능하게 하려면 우선 계정을 만드셔야 합니다. 시작해 보겠습니다.
우선 Firefox 4 최신 베타 버전을 다운로드 받으시고 Firefox Sync 계정을 만들 수 있습니다.
이미 Firefox Sync를 만드셨으면 '다른 컴퓨터에서 동기화 사용'을 선택하시고 로그인을 하시면 됩니다.
처음 사용하는 것이면 click '처음 동기화 사용'을 선택합니다.
사용자 이름과 암호 그리고 메일 주소를 입력합니다.
그리고 암호키를 입력합니다.
12자 이상의 문구를 넣어셔야 하며 데이터를 암호하여 전송하는 데 이용되며 꼭 기억하셔야 합니다.
잃어버리시면 데이터를 다시 찾을 수 없습니다.
자체 서버에 데이터를 저장하기 원하시지 않으시면 '다음'을 선택하십시오.
보이는 단어를 그대로 입력하세요.
'완료'를 선택하시면 Firefox Sync 계정이 만들어졌습니다!
이제부터 Firefox 데이터들은 자동으로 백그라운드로 동기화 됩니다.
처음 동기화를 하신다면 시간이 좀 걸릴 수 있습니다. 하지만 웹 서핑은 계속하셔도 됩니다.
일단 첫 동기화가 완료되면 회전하는 오른쪽 아래 아이콘을 보실 수 있을 것입니다.
지금 부터 동기화는 주기적으로 여러분의 데이터를 계속 동기화 작업을 하게 됩니다.
간단한 Firefox Sync 설정으로 언제 어디서나 Firefox를 끊김없이 사용하실 수 있습니다.
