Kiterjesztések fejlesztése

A Maemo és Windows Mobile (beta) platformon futó Firefox is támogatja a kiegészítéseket. Tulajdonképpen ugyanaz a kiegészítő használható ugyanazon a platformon. A leghasznosabb tudnivalók listája:

  • Kiterjesztések alapjai
  • Leírások a kiterjesztésekhez
  • Leírások a felhasználói felületekhez
  • Architektúraáttekintés
  • Kódrészletek

Segítsen a Firefox for Mobile készítésében

A Firefox sikeréhez az Ön segítségére is szükség van. Csatlakozzon a Mobile IRC-csatornához (irc.mozilla.org, #mobile), a szerdai fejlesztői megbeszéléshez, vagy küldjön hibabejelentést.
