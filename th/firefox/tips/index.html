เพิ่มที่คั่นหน้าเว็บในคลิกเดียว

วิธีติดตามเว็บไซต์ที่คุณชอบได้ดีกว่า (และประหยัดเวลา) ด้วยที่คั่นหน้าเว็บในคลิกเดียว เมื่อคุณเข้าเยี่ยมหน้าที่คุณต้องการจะคั่นไว้ แค่คลิกไอคอนรูปดาวในแถบที่อยู่ Firefox จะทำการเพิ่มเว็บนั้นลงไปในรายการที่คั่นหน้าเว็บของคุณ ซึ่งจะทำให้คุณสามารถค้นหามันได้อย่างง่ายดาย

  • เว็บไซต์มีข้อมูลการระบุตัวตนเจ้าของเว็บแบบเต็มขั้นที่ผ่านการพิสูจน์แล้ว และมีการป้องกันข้อมูลของคุณจากผู้ประสงค์ร้าย
  • เว็บไซต์มีข้อมูลการระบุตัวตนขั้นพื้นฐาน และมีการป้องกันการดักข้อมูลข้อมูลของคุณจากผู้ประสงค์ร้าย
  • เว็บไซต์ไม่มีข้อมูลการระบุตัวตนใด ๆ

รู้ว่าคุณกำลังติดต่ออยู่กับใคร

การหลอกล่อของเว็บเจ้าเล่ห์ทั่ว ๆ ไปคือการตั้งเว็บหลอก (ที่รู้จักกันในนามเว็บล่อเหยื่อ หรือ phishing site) ด้วยการแกล้งทำตัวเป็นธนาคารที่ใช้คุณงานประจำ เว็บค้าขายทางอินเทอร์เน็ตที่คุณชื่นชอบ ฯลฯ โชคดีที่ Firefox ช่วยคุณได้ด้วยตรวจสอบเว็บที่คุณกำลังใช้งานอยู่ ถ้าเว็บนั้นบอกว่าเขาคือใคร เพียงแค่คลิกที่ไอคอนของไซต์เพื่อดูภาพรวมของข้อมูลเว็บ

เว็บมุ่งโจมตีหรือเว็บหลอก การให้ข้อมูลส่วนตัว หรือเพียงแค่เข้าดูเว็บก็อาจเป็นอันตรายได้

ผู้ใช้ระดับกลาง

เล่นกับป้าย

เมื่อคุณทำการสร้างรายการที่คั่นหน้าเว็บจำนวนมาก การติดตามรายการทั้งหมดอาจจะทำให้คุณสับสนได้ คุณสามารถกำหนดป้ายให้กับเว็บที่ถูกคั่นไป เพื่อที่จะทำให้รายการที่คั่นหน้าเว็บเป็นระเบียบอยู่เสมอ เพียงแค่ดับเบิ้ลคลิกบนไอคอนรูปดาว แล้วคุณก็จะสามารถใส่ป้ายได้ทันที

คุณสามารถใส่ป้ายไปยังเว็บไซต์ได้มากเท่าที่คุณต้องการ (อย่าลืมใส่เครื่องหมาย , คั่นระหว่างป้าย) และยังสามารถทำการค้นหาได้อย่างง่ายดายด้วยการพิมพ์ชื่อป้ายในแถวที่อยู่ ตัวอย่างเช่น พิมพ์คำว่า "ท่องเที่ยว" ลงในแถบที่อยู่ จะเป็นการแสดงทุกเว็บไซต์ที่คุณทำการคั่นหน้าเว็บไซต์ด้วยป้าย "ท่องเที่ยว"

เรียนรู้เพิ่มเติมเกี่ยวกับส่วนเสริม

Firefox รุ่นมาตรฐานนั้นเต็มไปด้วยความสามารถพิเศษ แต่มันยังเพิ่มเติมได้มากกว่านั้น? มีส่วนเสริมอีกมากกว่า 5,000 ส่วนเสริม (ส่วนพิเศษเล็ก ๆ ที่ช่วยปรับแต่ง Firefox) ที่คุณสามารถดาวน์โหลดเพื่อเสริมงานใด ๆ ก็ตามที่คุณสามารถนึกถึงได้ เพียงแค่ไปยัง เครื่องมือ → ส่วนเสริมเพื่อเปิดตัวจัดการส่วนเสริมและค้นพบทุกหนทางที่สามารถทำให้ Firefox เป็นอย่างที่คุณต้องการ

การค้นหาที่ฉลาดขึ้น

คุณคงจะคุ้นเคยกับกล่องค้นหาใน Firefox แล้ว แต่คุณรู้หรือไม่ว่าคุณสามารถปรับแต่งมันให้เป็นตัวค้นหาใด ๆ ก็ได้ที่คุณต้องการ? แค่คลิกบนไอคอนทางด้านซ้ายของกล่องค้นหาเพื่อดูตัวเลือกต่าง ๆ ที่กำหนดมาให้

ที่ดียิ่งกว่านั้น ลองกดที่จัดการตัวค้นหา… เพื่อจัดลำดับรายการใหม่, เพิ่มตัวเลือกเพิ่มเติม หรือแม้กระทั่งกำหนดปุ่มกดทางลัดต่าง ๆ ให้กับตัวค้นหาที่คุณชื่นชอบ ตัวอย่างเช่น คุณสามารถกำหนดทางลัด "G" ให้กับ Google เมื่อคุณต้องการที่จะทำการค้นหาตำราอาหาร เพียงแค่พิมพ์ "G ตำราอาหาร" ลงในแถวที่อยู่ คุณจะสามารถที่จะค้นหาได้อย่างง่ายดาย

ค้นหาแบบสายฟ้าแลบ

ความสามารถ "ค้นหาข้อความเมื่อเริ่มต้นพิมพ์" ช่วยประหยัดเวลาได้มาก แทนที่จะใช้ แถบ "ค้นหา" เพื่อทำการค้นหาคำบนหน้าเว็บ แค่คลิกที่ใดก็ตามบนหน้าเว็บและพิมพ์คำที่คุณต้องการค้นหา เคอร์เซอร์ของคุณจะกระโดดไปยังตัวอักษรชุดแรกของคำนั้น

คุณสามารถใช้ความสามารถนี้กับลิงก์ได้เช่นกัน ตัวอย่างเช่น แทนที่จะใช้วิธีเลื่อนเมาส์ข้ามหน้ามาเพื่อคลิกลิงก์ "เรียนรู้เพิ่มเติม" แค่เริ่มพิมพ์คำและเมื่อเคอร์เซอร์พบคำนั้นก็กด Enter

กำจัดร่องรอยให้หมด

ด้วยเหตุผลทางความปลอดภัยและความเป็นส่วนตัว มันไม่ค่อยดีนักที่จะทิ้งร่อยรอยไว้ว่าคุณได้ท่องไปยังเว็บไหนบ้าง (โดยเฉพาะเมื่อคุณใช้คอมพิวเตอร์ร่วมกับผู้อื่น) Firefox อนุญาตให้คุณทำการลบประวัติการเข้าเว็บและดาวน์โหลด ล้างแคชและลบคุ้กกี้ในคลิกเดียว เพียงแค่ไปยังเครื่องมือ → ล้างข้อมูลส่วนตัว… หรือถ้าต้องการระมัดระวังยิ่งกว่านั้น ไปที่เครื่องมือ → ปรับแต่ง → ความเป็นส่วนตัว และเลือกตัวเลือก "ล้างข้อมูลส่วนตัวของฉันทุกครั้งที่ปิด Firefox"

ผู้ใช้ระดับสูง

ไปยังเว็บที่คุณชื่นชอบโดยตรง

คุณสามารถเพิ่มคำสำคัญไปยังที่คั่นหน้าเว็บของคุณเพื่อการเข้าถึงที่ง่ายและเร็วขึ้น เพียงแค่เพิ่มคำสำคัญสั้น ๆ ในช่องคำสำคัญลงในห้องสมุด จากนั้นคุณจะสามารถเข้าถึงที่คั่นหน้าเว็บนั้นได้เพียงแค่พิมพ์คำสำคัญนั้นลงในแถบที่อยู่ ตัวอย่างเช่น คุณให้คำสำคัญกับเว็บ del.icio.us ว่า "ลิงก์" เมื่อคุณทำการพิมพ์ "ลิงก์" ในแถบที่อยู่คุณจะสามารถเข้าถึงเว็บนั้นได้ทันที

สร้างโฟลเดอร์อัจฉริยะ

ถ้าคุณเป็นนักท่องเว็บขั้นสูงที่ต้องการบันทึกและติดตามเว็บต่าง ๆ ที่หลากหลายบ่อย ๆ การใช้ห้องสมุดของ Firefox 3 สามารถช่วยประหยัดเวลาคุณได้ เพียงคุณสร้างและบันทึกการค้นหาต่าง ๆ ลงในโฟลเดอร์ที่จะทำการปรับปรุงข้อมูลของเว็บไซต์ที่คุณเพิ่มลงในที่คั่นหน้าเว็บโดยอัตโนมัติ

เริ่มด้วยการเปิดห้องสมุดโดยการเลือกตัวเลือก "จัดการที่คั่นหน้าเว็บ" จากเมนูที่คั่นหน้าเว็บ แล้วกรอกคำค้นในกล่องค้นหา แล้วคลิกปุ่มบันทึกเพื่อทำการสร้างโฟลเดอร์อัจฉริยะ

จัดการการดาวน์โหลดของคุณ

ถ้าคุณเป็นนักดาวน์โหลดตัวยง คุณสามารถใช้หน้าต่างตัวจัดการดาวน์โหลดเพื่อติดตามการดาวน์โหลดทั้งหมดของคุณ คุณสามารถหยุดพักและดาวน์โหลดไฟล์ต่อได้และคุณสามารถเปิดไฟล์จากตัวจัดการดาวน์โหลดได้โดยตรงเพื่อประหยัดเวลา

ถ้าคุณต้องการติดตามการดาวน์โหลดที่ผ่านมา ไปที่ เครื่องมือ → ดาวน์โหลดและใช้กล่องค้นหาเพื่อค้นหาไฟล์ของคุณ เมื่อคุณพบไฟล์ที่คุณต้องการ คุณสามารถเปิดไฟล์ได้ด้วยการดับเบิ้ลคลิก หรือ คลิกขวา และเลือก "คัดลอกลิงก์ดาวน์โหลด"

ค้นพบเครื่องมือนักพัฒนา

ถ้าคุณเป็นนักพัฒนาเว็บ เครื่องมือนักพัฒนาของ Firefox จะทำให้ชีวิตคุณง่ายขึ้น เว็บไซต์ส่วนเสริมของ Mozilla มีเครื่องมือมากมายเพื่อเสริมประสิทธิภาพขั้นตอนการพัฒนา รวมไปถึง Firebug ที่ใช้ในการ แก้ไข, ดีบั๊ก และสำรวจ CSS, HTML และ JavaScript แบบสด ๆ บนเว็บไซต์ใด ๆ หรือจะเป็น Tamper Data ที่ใช้ในการดูและแก้ไขเฮดเดอร์ของ HTTP/HTTPS และพารามิเตอร์ POST และตัวตรวจสอบ DOM ที่ใช้ในการตรวจสอบ HTML และ CSS ใด ๆ เพียงแค่คลิกขวา

ตั้งค่าเว็บไซต์ให้เป็นอย่างที่คุณชื่นชอบ

คุณสามารถใช้งานตัวจัดการโปรโตคอลบนเว็บ เพื่อเข้าใช้งานเว็บแอปพลิเคชันที่คุณชื่นชอบได้รวดเร็วยิ่งขึ้น ตัวอย่างเช่น คุณสามารถตั้งให้ Firefox เปิดหน้าต่างเขียนอีเมลใหม่กับเว็บเมลที่คุณเลือก แทนที่จะเป็นโปรแกรมอีเมลบนเครื่องของคุณเมื่อใดก็ตามที่คลิกลิงก์ mailto: (หมายเหตุ: คุณสามารถใช้ความสามารถนี้ได้กับบางผู้ให้บริการอีเมลบางราย ที่ทำการลงทะเบียนไว้กับ Firefox 3 เท่านั้น)

ไปยัง เครื่องมือ → ปรับแต่ง → แอปพลิเคชัน เพื่อเลือกแอปพลิเคชันปริยายสำหรับแต่ละโพรโตคอลหรือเลือก "ถามเสมอ" ถ้าคุณต้องการเลือกแอปพลิเคชันด้วยตัวเองทุกครั้ง

ถ้าคุณเป็นนักพัฒนาเว็บ เข้าไปดูข้อมูลเพิ่มเติมเกี่ยวกับการเพิ่มการรองรับ Web-based protocol handlers ใน Mozilla Developer Center.
