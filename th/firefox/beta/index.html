Firefox 4 Beta มาพร้อมกับส่วนเสริม Feedback ซึ่งให้คุณเข้าร่วมในการศึกษาผู้ใช้ จะไม่มีข้อมูลใดถูกส่งให้ Mozilla โดยไม่ได้รับความยินยอมจากคุณ ข้อมูลเพิ่มเติม

ช่วยกันบอกต่อ: ยิ่งมีผู้เข้าร่วมทดสอบ Firefox 4 มาก รุ่นจริงก็จะยิ่งดีขึ้นไปด้วย เรียนรู้เพิ่มเติมเกี่ยวกับรุ่นทดสอบ

Firefox 4 Beta มาพร้อมกับส่วนเสริม Feedback ซึ่งมีพื้นฐานอยู่บน Mozilla Test Pilot

เมื่อคุณติดตั้ง Firefox 4 Beta คุณจะสังเกตเห็นปุ่ม Feedback ที่มุมขวาบน ซึ่งคุณสามารถใช้เพื่อส่งคำแนะนำได้ทุกเมื่อ คุณยังจะได้รับการแจ้งเตือนเกี่ยวกับการศึกษาผู้ใช้ (ทำงานโดย Mozilla Test Pilot) ซึ่งจะเก็บข้อมูลเฉพาะที่เกี่ยวกับว่าคุณใช้ Firefox อย่างไร โดยไม่เก็บว่าคุณเข้าเว็บไซต์ไหนบ้าง คุณควบคุมการมีส่วนร่วมของคุณได้และสามารถเลือกที่จะออกจากการศึกษาใดๆ หรือดูได้ว่าข้อมูลใดถูกเก็บไปบ้าง คุณจะถูกถามก่อนที่ข้อมูลใดๆ จะถูกส่งไปยัง Mozilla โปรดดู นโยบายความเป็นส่วนตัวของส่วนเสริม Feedback สำหรับข้อมูลเพิ่มเติม
