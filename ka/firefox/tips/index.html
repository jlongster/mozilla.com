სასურველი საიტების სანიშნეების შენახვა (და დროის დაზოგვა) ერთი დაწკაპვით შეგიძლიათ. თუ საიტის დამახსოვრება გსურთ, უბრალოდ დაწკაპეთ ვარსკვლავი მისამართების პანელში. Firefox ბრაუზერი მას სანიშნეებში დაამატებს, სადაც მისი ხელახლა მოძებნა ადვილად შეგეძლებათ.

  • საიტი იძლევა წარმომავლობის გადამოწმებულ ინფორმაციას და იცავს თქვენს პირად მონაცმებს მოთვალთვალეებისგან.
  • საიტი იძლევა წარმომავლობის ძირითად ინფორმაციას და იცავს თქვენს პირად მონაცმებს მოთვალთვალეებისგან.
  • საიტი არ იძლევა ინფორმაციას წარმომავლობის შესახებ.

იცოდეთ ვისთან გაქვთ საქმე

ვებ ყალთაბანდების ზოგადი ტაქტიკა ყალბი საიტების შექმნაა, რომლებიც საკუთარი თავის ბანკის ან კომერციულ ან სხვა სახის საიტებად გასაღებად ცდილობენ. საბედნიეროდ, Firefox ბრაუზერი აიოლებს ასეთი სახის საიტების დადგენას – არიან თუ არა ისინი რადაც თავი მოაქვთ. ამისათვის უბრალოდ დაწკაპეთ საიტის ხატულა წარმომავლობის მყისიერად დასადგენად.

გამოცდილთათვის

დაჭდევება

თუ ბევრი სანიშნე დაგიგროვდათ, მათი თვალმიდევნება გართულდება. დასახარისხებლად შეგიძლიათ მათთვის ჭდეების მინიჭება: ორმაგად დაწკაპეთ ვარსკვლავის ხატულა და მიუთითეთ სასურველი ჭდეები.

საიტისთვის იმდენი ჭდის მინიჭება შეგიძლიათ, რამდენსაც ისურვებთ (არ დაგავიწყდეთ მათი მძიმეებით დაყოფა) და შემდეგ იოლად მოძებნოთ საიტები მისამართების პანელში ჭდის მითითებით. მაგალითად, თუ მისამართების პანელში მიუთითებთ ჭდეს "მოგზაურობა", თქვენ მიიღებთ ყველა საიტს, რომლებსაც შესაბამის ჭდე მიანიჭეთ.

ყველაფერი მოდულების შესახებ

Firefox ბრაუზერის სტანდარტულ ვერსიას უამრავი კომპონენტი გააჩნია, მაგრამ ღირს ამაზე გაჩერება? არსებობს 5,000-ზე მეტი მოდული (მცირე პაკეტები სხვადასხვა დანიშნულებით), რომლის საშუალებითაც თითქმის ნებისმიერი ამოცანის გადაჭრა შეგიძლიათ. პაკეტების მმართველის გასახსნელად შეარჩიეთ მენიუდან ხელსაწყოები → პაკეტები და აღმოაჩენთ უამრავ გზას, Firefox ბრაუზერის თქვენი მოთხოვნილებისამებრ გასამართად.

სხარტი ძიება

თქვენ უკვე იცნობთ Firefox ბრაუზერის საძიებო პანელს. მაგრამ იცით თუ არა, რომ საძიებო სისტემების მორგებაც შეგიძლიათ? დაწკაპეთ ხატულა საძიებო პანელის მარცხნივ ნაგულისხმები პარამეტრების მენიუს გამოსატანად.

მეტიც - დაწკაპეთ «საძიებო სისტემების მართვა» სიის დასახარისხებლად, ახლების დასამატებლად და კლავიშების კომბინაციის მისათითებლად თქვენი რჩეული საძიებო სისტემების გამოსაძახებლად. მაგალითად, Google საძიებო სისტემისთვის შეგიძლიათ მიუთითოთ "G" - შემდეგ, თუ გსურთ მაგალითად, კულინარიული რეცეპტების მოძებნა, მისამართების პანელში უბრალოდ მიუთითეთ "G კულინარიული რეცეპტები" შესაბამისი საიტების სრული სიის მისაღებად.

იპოვნეთ მყისიერად

პოვნა ტექსტის შეტანისას დროის დაზოგვის კიდევ ერთი საშუალებაა. ძიების პანელით სარგებლობის ნაცვლად, სიტყვის გვერდზე მოსაძებნად დაწკაპეთ ნებისმიერ გვერდის ადგილას და დაიწყეთ საძიებო სიტყვის შეტანა. თქვენი კურსორი დაუყოვნებლივ გადაადგილდება ამ სიტყვის პირველივე თანხვედრაზე.

იგივე წარმატებით შეგიძლიათ ბმების მოძებნაც. მაგალითად, თაგუნას კურსორის ბმაზე "დაწვრილებით" გადაადგილების ნაცვლად, უბრალოდ დაიწყეთ სიტყვის შეტანა და როცა კურსორი მას მიადგება დააჭირეთ კლავიშს Enter.

ყველა კვალის წაშლა

უსაფრთხოებისა და კონფიდენციალობის უზრუნველსაყოფად ინტერნეტში მოგზაურობის კვალის დატოვება (განსაკუთრებით საზიარო კომპიუტერზე) რეკომენდებული არაა. Firefox ბრაუზერი საშუალებას გაძლევთ წაშალოთ მონახულებული საიტებისა და ჩამოტვირთვების ისტორია, გაასუფთავოთ ბუფერი და წაშალოთ ნაჭდევები ერთი დაწკაპვით. ამისათვის მენიუდან შეარჩიეთ ხელსაწყოები → პირადი მონაცემების წაშლა. ან, მეტისმეტად ფრთხილებისთვის, გადადით ხელსაწყოები → პარამეტრები → პრივალტულობა და ჩართეთ პარამეტრი "ჩემი პირადი მონაცემების ყოველთვის წაშლა Firefox ბრაუზერის დახურვისას".

სპეციალისტებისთვის

უშუალოდ თქვენს რჩეულ საიტებზე გადასვლა

სწრაფი და იოლი წვდომის უზრუნველსაყოფად სანიშნეებს შეგიძლიათ საკვანძო სიტყვებიც დაუმატოთ. დაუმატეთ მოკლე საკვანძო სიტყვა შესაბამის ველში და ამ სანიშნის გამოძახებაა საკვანძო სიტყვის მისამართების პანელში აკრეფით შეძლებთ. მაგალითად, მაგალითად თქვენს del.icio.us ანგარიშს შეგიძლიათ მიანიჭოთ საკვანძო სიტყვა “ბმები”. ამის შემდეგ მისამართების პანელზე სიტყვა “ბმები”-ს აკრეფა უშუალოდ ამ გვერდზე გადაგიყვანთ.

სხარტი დასტების შექმნა

თუ ინტერნეტის გამოცდილი მომხმარებელი ხართ, რომელსაც ხშირად სჭირდება გარკვეული სახის საიტების თვალმიდევნება, თქვენთვის არსებობს სასარგებლო და დროის დასაზოგი ფუნქცია : Firefox 3-ის ბიბლიოთეკაში შეგიძლიათ ძიების შედეგების დასტების შექმნა და შენახვა, რომლებიც ავტომატურად განახლდება საიტების სანიშნეებსა და ისტორიაში შეტანისას.

თავდაპირველად გახსენით ბიბლიოთეკა სანიშნეების მენიუს პარამეტრიდან "სანიშნეების მართვა". შემდეგ საძიებო დაფაზე მიუთითეთ საზიებო კონტექსტი. შემდეგ დაწკაპეთ ღილაკი შენახვა სხარტი დასტის შესაქნელად.

თქვენი ჩამოტვირთვების მართვა

თუ ჩამოტვირთვებს ხშირად ახორციელებთ, მათ თვალმისადევნებლად ჩამოტვირთვების მმართველის სარკმლით სარგებლობა შეგიძლიათ. შეგიძლიათ ჩამოტვირთვების შეჩერება და დროის დაზოგვა ფაილების უშუალოდ მმართველიდან გახსნით.

თუ ძველი ჩამოტვირთვის მოძებნა გსურთ გადადით მენიუდან პუნქტზე ხელსაწყოები→ჩამოტვირთვები და ფაილის მოსაძებნად ძიების დაფა გამოიყენეთ. პოვნის შემდეგ ორმაგად დაწკაპეთ ფაილი გასახსნელად ან Ctrl-დაწკაპვით შეარჩიეთ "ჩამოტვირთვის ბმის კოპირება".

აღმოაჩინეთ თქვენთვის პროგრამისტის ხელსაწყოები

თუ ვებ პროგრამისტი ხართ, Firefox ბრაუზერის პროგრამისტის ხელსაწყოები ბევრად გაგიადვილებთ ცხოვრებას. Mozilla მოდულების საიტი გთავაზობთ ხელსაწყოების ფართო არჩევანს პროგრამირებისთვის. მათ შორისაა: Firebug რომელიც ვებსაიტების CSS, HTML და JavaScript კოდების რედაქტირების, გამართვისა და და თვალმისადევნებად გამოიყენება, Tamper Data გამოიყენება HTTP/HTTPS თავსართებისა და POST პარამეტრების საჩვენებლად და შესაცვლელად, ხოლო DOM ინსპექტორი ამოწმებს ნებისმიერ HTML ან CSS ელემენტს მარჯვენა ღილაკზე დაწკაპვით.

ახლა უკვე შეგიძლიათ ინტერნეტ ოქმების სამარჯვებით სარგებლობა თქვენი რჩეული ვებ პროგრამების მხარდაჭერის უზრუნველსაყოფად. მაგალითად, შეგიძლიათ გამართოთ Firefox ბრაუზერი ისე, რომ ნებისმიერ საიტზე mailto: ბმაზე დაწკაპვისას გაიხსნება სარკმელი სასურველი ელფოსტის პროვაიდერისთვის ნაცვლად ელფოსტის ადგილობრივი საფოსტო პროგრამისა თქვენს კომპიუტერში (შენიშვნა: ეს კომპონენტი მისაწვდომია მხოლოდ Firefox 3 ბრაუზერში რეგისტრირებული ელფოსტის მომსახურებებისთვის).

თითოეული ოქმისთვის ნაგულისხმები პროგრამის შესარჩევად გადადით ხელსაწყოები→პარამეტრები→პროგრამები ან თუ პროგრამის შერჩევა თავად გსურთ, მიუთითეთ პარამეტრი “ყოველთვის დამეკითხე”.

თუ ვებ პროგრამისტი ხართ, თქვენთვის სასარგებლო იქნება გაეცნოთ თუ როგორ შეიძლება ინტერნეტ ოქმების სამარჯვების მხარდაჭერის დამატება Mozilla პროგრამირების ცენტრში.
