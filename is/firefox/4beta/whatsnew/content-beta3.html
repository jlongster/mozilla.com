Nýjustu hlutirnir

Firefox 4 byrjar stórkostlega vel. Hérna eru nokkrir af nýjustu hlutunum sem þú vilt alls ekki missa af:

Mýkri grafík

Nýrri JavaScript hlutir gera Firefox kleift að keyra þungar reiknivinnslur mun betur, sem skilar sér í mýkri grafík.

SQLite 3.7.0

Stórkostlega staðsetningarsláin, niðurhalsstjóri og IndexedDB eru nú hraðari og mýkri þökk sé uppfærslu á SQLite, sem er SQL gagnagrunnsvél.
