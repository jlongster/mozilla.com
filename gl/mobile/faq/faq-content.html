Preguntas máis frecuentes

Sobre Firefox

Por que Firefox para teléfonos móbiles?
Traer Firefox aos teléfonos móbiles é o seguinte paso para completar a misión de Mozilla de dar acceso a unha Rede á que todos poidan acceder, sen importar o dispositivo ou a localización. Ademais, Mozilla ten como obxectivo proporcionar unha plataforma rica para os desenvolvedores poderen crear contido atractivo e aplicacións innovadoras. O navegador máis avanzado existente, Firefox, é rápido, seguro, personalizábel e sincronízase á perfección co Firefox do teu escritorio.
Isto ten algún custo?
Non. Firefox foi creado como parte do compromiso sen fin de lucro da Mozilla Foundation de contribuír á apertura, innovación e oportunidade na Internet. Cremos que os produtos de software libre e de código aberto como Firefox son fundamentais para manter a Internet como un recurso público global. Obtén máis información.
Que diferencia o navegador Firefox do navegador móbil que teño agora?
Mozilla trae a experiencia de navegar con Firefox aos dispositivos móbiles. Construído co motor de navegación máis moderno dispoñíbel, Firefox complétase con funcións xa coñecidas como a "Barra fantástica", o Xestor de contrasinais, os Marcadores cun toque, a Navegación sensíbel á localización e máis. Firefox é tamén o primeiro navegador web que ofrece complementos, dando a posibilidade aos usuarios de personalizaren o seu navegador estean onde estiveren.
Está Firefox dispoñíbel no meu idioma?
Firefox para dispositivos móbiles está dispoñíbel en máis de 25 idiomas e hai máis traducións de camiño. Consulta a lista completa de idiomas dispoñíbeis.
Quen fai Firefox?
Firefox foi producido sen fin de lucro pola Mozilla Foundation coa axuda da nosa comunidade mundial. Grazas ás contribucións da comunidade podemos engadir novas actualizacións e complementos para facer Firefox máis rápido, máis seguro e máis potente. Queres axudar? Visita a páxina Desenvolve para móbil.

Seguranza

Como me protexe Firefox mentres navego co meu móbil?
O proceso de código aberto de Mozilla válese da experiencia de centos de expertos en seguranza de todo o mundo, e Firefox inclúe funcións como ID instantáneo do sitio web, que comproba os sitios sospeitosos para asegurarse de que son o que din ser. Os axustes de seguranza personalizábeis para xestionar contrasinais e cookies, cargar imaxes e instalar complementos danche un control total sobre a túa experiencia de navegación. Obtén máis información.

Descargas e dispositivos

Como descargo Firefox no meu teléfono?
Visita a páxina de Mozilla.com Mobile para enviar unha mensaxe de texto cunha ligazón para descargares Firefox no teu teléfono. Ou, simplemente vai a mozilla.com/m co teu navegador móbil actual e descarga Firefox directamente no teu teléfono.
Vai funcionar Firefox co meu teléfono móbil?
Agora mesmo Firefox é compatíbel cos dispositivos móbiles de Nokia N810 e N900, así como con algúns teléfonos específico con Windows Mobile 6.0 e versións superiores. Consulta a lista completa de dispositivos compatíbeis.
Podo descargar Firefox no meu teléfono aínda que non sexa un dispositivo compatíbel?
Podes executar Firefox nun teléfono Maemo ou Windows Mobile 6.0 e superior aínda que non estea na nosa listaxe de dispositivos probados, aínda que non garantimos funcionalidade completa. Animámoste a probar Firefox nun dispositivo compatíbel para que poidas experimentar as súas capacidades completas. Consulta a lista completa de dispositivos compatíbeis.
Podo descargar Firefox para o iPhone?
De momento Firefox non está dispoñíbel para o iPhone.

A usar Firefox

Como uso a interface de usuario de Firefox para navegar na Rede?
Para aforrar espazo na pantalla do teu teléfono, os controis do navegador están ocultos nas beiras da pantalla. Podes acceder a eles deslizando o teu dedo pola pantalla cara á esquerda ou á dereita. Os teus separadores están no lado esquerdo da pantalla, mentres que os marcadores e os axustes están no dereito. Para que poidas ver máis do sitio web que esteas a visitar, a barra de enderezos desaparece cando baixas coa barra de desprazamento, mais volve aparecer deslizando cara arriba ou cara o lado.
Como aumento ou reduzo o zoom?
Dá dous toques co dedo na pantalla para aumentar o zoom. Para reducir o zoom volve dar dous toques.
Como marco unha páxina web?
Vai ao sitio que queres marcar. Toca o botón de marcadores coa "estrela" no lado dereito da túa pantalla.
Como abro un novo separador?
Toca o botón de novo separador co "máis" no lado esquerdo da túa pantalla para abrir un novo separador. Os separadores aparecen como miniaturas, así poderás ver facilmente que páxina queres seleccionar.

Características

Que ten de especial a Barra fantástica?
A Barra fantástica lévate rapidamente a onde queres ir. Toca a Barra fantástica e comeza a escribir. Ao mesmo tempo que escribes amosarache páxinas web que visitas con frecuencia ou que visitaches ou marcaches recentemente. A Barra fantástica aprende ao mesmo tempo que a usas, adaptándose ás túas preferencias e facendo a túa navegación máis persoal. Tamén podes darlle un toque á Barra fantástica para introducir un URL ou un termo de busca e empezar a explorar a Rede. Ou dar un toque nun dos motores de busca na parte inferior da pantalla para chegar ata a información desexada. 
Cales son os complementos de Firefox e como podo obtelos no meu dispositivo móbil?
Os complementos permítenche personalizar o Firefox engadindo funcións e personalizacións que che axudan a facer teu o navegador. Selecciona o botón de axustes co "mecanismo" na parte inferior dereita da túa pantalla, logo selecciona o botón de complementos coa "peza de quebracabezas" e busca e instala directamente no teu teléfono os complementos máis útiles para ti.
Como podo compartir os meus marcadores, o meu historial, os meus separadores e outras cousas de Firefox entre o meu computador e o meu móbil?
O complemento Firefox Sync tráeche a experiencia Firefox ao teu teléfono móbil. Primeiro, crea unha conta de Firefox Sync no teu computador. Logo vai ao Firefox no teu móbil e busca e instala o complemento Firefox Sync. Cando introduzas a túa información na conta de Firefox Sync e sincronices, o teu computador e o teu móbil compartirán as túas preferencias, historial e demais de Firefox. Para obteres máis información, visita a páxina sobre sincronización.
É necesario ter o complemento Firefox Sync para usar Firefox no meu teléfono?
Non. Aínda que nós cremos que sincronizar Firefox entre o teu computador e o teu móbil é super útil, é un servizo opcional que podes escoller usar ou non. Non che fai falta para seguires desfrutando do Firefox no que confías e que coñeces. Firefox Sync permíteche compartir o teu historial, marcadores, preferencias e máis, aumentando a túa experiencia global de navegación onde sexa que estiveres.
Que é a navegación sensíbel á localización? Fai Firefox un seguimento de onde estou?
A navegación sensíbel á localización afórrache tempo permitindo aos sitios web preguntarche onde estás. Se elixes compartir a túa localización cun sitio web, este pode usar esta información para atopar puntos próximos de interese e devolverche información adicional e útil, como mapas da túa zona. É todo opcional: Firefox non accede á túa localización sen o teu permiso. Obtén máis información sobre Navegación sensíbel á localización e consulta a política de privacidade do Servizo de localización de Google.
Como de precisas son as localizacións?
A precisión varía moito dunhas localizacións a outras. Nalgúns lugares, os nosos provedores de servizo talvez poidan proporcionar unha localización a uns poucos metros, mais noutras zonas pode ser moito máis. Todas as localizacións devoltas polos nosos provedores de servizo son estimacións e non garantimos a exactitude das localizacións proporcionadas.

Preguntas de desenvolvedores

Pasarase ao móbil un complemento de Firefox que creei para o computador? Se non, cómpre que o mude?
Aínda que o Firefox para escritorio e para móbil están construídos na mesma plataforma hai pequenas diferenzas que requirirán que os desenvolvedores fagan cambios nos seus complementos. Normalmente estas mudanzas son sinxelas, visita a páxina Desenvolve para móbil para obter máis información.
Que motor de navegación utiliza Firefox?
Tanto a versión de Firefox de escritorio como a de móbil usan o motor de renderización Gecko. A versión móbil de Firefox usa a versión 1.9.2 de Gecko, que é o motor de navegación de mellor rendemento que Mozilla xamais liberou e inclúe apaixonantes funcionalidades como un rendemento radicalmente mellorado de JavaScript, soporte para xeolocalización, fíos de traballo e moito máis.
Que motor JavaScript usa Firefox?
Firefox inclúe o novo motor JavaScript “TraceMonkey”, un avanzado compilador en tempo de execución que usa tecnoloxía de rastrexo. O motor TraceMonkey fai as aplicacións web máis rápidas ao compilar o código á medida que as executas, permitindo que moitos tipos de aplicación que fan un uso intensivo da CPU se executen a velocidade case nativa.
Como desenvolvo un sitio web sensíbel á localización?
Firefox implementou o primeiro borrador público da Especificación de xeolocalización para a W3C. Unha visión xeral sobre como desenvolver sitios web sensíbeis á localización aquí.
É Firefox compatíbel con HTML 5 no móbil?
HTML 5 é a próxima revisión a fondo do HTML. A especificación HTML 5 é un traballo en curso, mais hai varias características de HTML 5 e relacionadas que posibilitarán aos desenvolvedores crearen sitos web e aplicacións máis innovadores. Firefox inclúe importantes funcionalidades como a Navegación sensíbel á localización, as etiquetas <canvas> e <svg>, Web workers e almacenamento sen conexión.
Admite plugins Firefox no meu teléfono?
Pois en principio non. Deshabilitamos o soporte para plugins no Firefox para Maemo debido ao baixo rendemento das páxinas co plugin Flash de Adobe, algo especialmente notorio de webs con múltiples instancias do plugin. Porén os usuarios avanzados poden habilitar o uso de plugins pero só con propósito experimental. Estamos a traballar agora para permitir ao usuario escoller cando habilitar ou non esta posibilidade que nalgúns casos, como en YouTube, funciona bastante ben. Seguramente incorporaremos esta posibilidade como un complemento do navegador.
