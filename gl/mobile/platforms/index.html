Non está aquí o teu teléfono?

Se o teu teléfono non aparece nesta lista, sempre podes descargar e usar Firefox no teu dispositivo móbil. Pode que o teu dispositivo móbil non teña suficiente memoria (128 MB ou máis) para executar Firefox. Dito isto, animámoste a descargar Firefox nun dispositivo que xa fose probado para así experimentar as funcionalidades completas.

Tamén podes experimentar con Firefox, facer suxestións, escribir complementos e involucrarte instalando unha versión de Firefox para o teu computador de escritorio.
