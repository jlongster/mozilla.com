Firefox Flicks 常見問題

Firefox Flicks 是什麼？
Firefox Flicks 是一個邀請全世界所有 Firefox 粉絲與製片人拍攝短片（稱之為「flick」）來敘述 Firefox 的故事的競賽。
比賽將在什麼時候開始？
Firefox Flicks 競賽網站將於 2012 年二月開站。您將可以從該時間開始上傳影片，公開投票將於同年度稍後開始進行。我們將在近期貼出更多資訊，請記得常常回來看看。
我可以為多部短片報名嗎？
你想投多少就投多少！但是您必須獨立上傳每部短片。
可以組隊拍片嗎？
當然！每組隊伍最多可以有十個人。雖然我們還是只會送出一包 Firefox 大禮包讓隊員分享。同時，請指派一位組長負責聯繫事宜。另外每個作品最多只會贏得一個獎項。
這個比賽是否接受全世界的作品？
是的。這個比賽開放給世界上的所有人參加。然而每個獎項，我們將會從北美洲、拉丁美洲、亞洲與紐澳，以及歐洲與非洲四個區域各選出一個得獎者（以最靠近您的區域為準）。
評分標準是什麼？
影片將會依照拍攝的品質、剪接，以及多切中 創意說明 來評分。

比賽將有下列獎項:

  • 最佳 30 秒短片
  • 最佳動畫
  • 最佳技術使用（例如使用 Popcorn.js、全程使用行動裝置拍攝、3D 影片）
  • 最佳公益廣告

評審團將自經由大眾投票選出的作品當中挑選出大獎得獎作品。

評審有哪些人？
我們正在邀請廣告、製片等不同領域的專家加入國際評審團隊。在未來幾個月當中會有更多評審加入。
我可以贏得什麼？
只要參加比賽，您的短片就將能被全世界數以百萬計的 Firefox 粉絲看到，讓您的影片可以曝光給全球觀眾看見。你不但能以世界上最大的品牌之一為題創作，成果也會在國際廣告專家與製片權威前展現，這是一個在作品集當中超棒的履歷。

不只是這樣！每個區域（北美洲、拉丁美洲、歐洲、亞洲與紐澳）當中的大獎得獎者將可以任選價值五千美金的攝影器材或電影學校獎學金，每個類型的前兩名得主將可以得到價值一千美金的攝影器材；每個區域的各分類首獎得獎者以及大獎得主將可以免費參加 2012 年舉辦的 Firefox Flicks 活動之旅。所有競賽作品將可以得到一份 Firefox 大禮包。最重要的是，一但您得獎了，您的影片將可能被用來與世界分享 Firefox 的精采故事。

我要如何報名加入比賽？
我們將會在 2012 年初更新關於如何報名的資訊。您將可以把作品上傳到一個投票網站供網友投票，但您還需要送出高解析度版本給 Mozilla。請密切注意詳情。
若我還有問題，應該聯絡誰？
請寄信到 firefoxflicks@mozilla.org，我們將會盡快回覆您的問題。
我可以把作品當作學校專題製作嗎？
當然可以！我們覺得 Firefox Flicks 是一個讓學生參與實務競賽的大好機會，這個競賽可以讓很多人看到，且由電視電影與廣告等產業的專家進行評分。歡迎跟您的教授談論如何把拍攝影片做為作業之一。
我可以在短片當中使用 Firefox 的商標與圖示嗎？
可以。若您是以拍攝這個比賽要用的短片為目的的話，您可以使用 Firefox 商標與圖示。若需要更多關於商標使用的資訊，請參考 Mozilla 商標使用政策。
我要如何才能取得高解析度的 Firefox 名稱與圖示圖檔？
若需要高解析度的圖檔還有使用規範的話，請看看我們的 Brand Toolkit 網頁。
我要到哪裡才可以找到音樂或其他素材放到短片當中？
若您需要更多素材，不論是音樂、鏡頭素材、美工圖片等等，我們建議您使用公有領域的媒體素材。創用 CC 提供了一個不錯的搜尋服務讓您可以尋找公有領域作品。您可以到 維基百科 取得更多關於公有領域的資訊。
我可以在短片當中使用第三方商標或受著作權保護的素材嗎？
不可以，未經這些素材的擁有者授權前請勿使用。
我可以用非英語影片報名嗎？
可以。對於非英語的作品，您需要送出完整且精確的字幕或腳本。若您沒有辦法翻譯您的短片，請 聯絡我們。我們可以幫忙。
如果我想撤回我的作品怎麼辦？
如果您想要放棄報名，請寫信到 firefoxflicks@mozilla.org 聯絡我們，我們將會自比賽中移除您的作品。
我的作品應該使用什麼長寬比例製作？
我們接受任何長寬比例的影片，但我們偏好且推薦您以 16:9 製作。
報名的影片將會向大眾公開嗎？
您的短片將會被張貼在投票網站，這樣網友們才能幫忙投票選出大獎得獎者。
