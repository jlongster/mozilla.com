Firefox Flicks 常見問題集

Firefox Flicks 是什麼？
Firefox Flicks 是一個邀請全世界所有 Firefox 粉絲與製片人拍攝短片（稱之為 "flicks"）來敘述 Firefox 的故事的競賽。
比賽將在什麼時候開始？
Firefox Flicks 競賽網站將於 2012 年二月開站。您將可以從該時間開始上傳影片，公開投票將於同年度稍後開始進行。我們將很快會貼出更多資訊，請記得常常回來看看。
我可以為超過一部短片報名嗎？
每個人不限報名的影片數量，但是每次只能為一部影片報名。
可以組隊拍片嗎？
當然！每組隊伍最多可以有十個人。然而我們將會送出比較大包的 Firefox 大禮包讓隊員分享。同時，請指派一位組長。這位組長需要負責所有與我們的聯繫，且每個作品最多只會得到一個獎項。
這個比賽是否接受全世界的作品？
是的。這個比賽開放給世界上的所有人參加。然而針對每個獎項，我們將會從北美洲、拉丁美洲、亞洲澳洲與紐西蘭，以及歐洲與非洲四個區域各選出一個得獎者。
評分標準是什麼？
影片將會依照拍攝的品質、剪接，以及多切中 創意簡介 來評分。

比賽將有下列獎項:

  • 最佳 30 秒短片
  • 最佳動畫
  • 最佳技術使用（例如使用 Popcorn.js、全程使用行動裝置拍攝、3D影片）
  • 最佳公益廣告

評審團將自經由大眾投票選出的作品當中挑選出大獎得獎作品。

評審有哪些人？
我們正在邀請廣告、製片等不同領域的專家加入國際評審團隊。在未來幾個月當中會有更多評審加入。
我可以贏得什麼？
只要參加比賽，您的短片就將能被全世界數以百萬計的 Firefox 粉絲看到，讓您的影片可以曝光給全球性的觀眾。您將可以讓您的影片被世界最大品牌當中之一的國際廣告公司高層與製片權威所看見。這是一個在作品集當中超棒的履歷。

不只是這樣！每個區域（北美洲、拉丁美洲、歐洲、亞洲與紐澳）當中的大獎得獎者將可以得到價值五千美金的攝影器材或電影學校的獎學金，每個作品分類的前兩名將可以得到價值一千美金的攝影器材。每個區域的每個分類的第一名得獎者以及大獎得獎者將可以免費參加 2012 年當中舉辦的 Firefox Flicks 活動。所有報名參加競賽的短片作者將可以得到一份 Firefox 大禮包。最重要的是，一但您得獎了，您的影片將可能被用來與世界分享 Firefox 的精采故事。

我要如何報名加入比賽？
我們將會在 2012 年初更新關於如何送交短片給我們的資訊。您將可以送出您的作品到一個投票網站供網友投票，但您還需要送出高解析度版本給 Mozilla。請密切注意詳情。
若我還有問題，應該聯絡誰？
請寄信到 firefoxflicks@mozilla.org，我們將會盡快回覆您的問題。
我可以把作品當作學校作業嗎？
當然可以！我們覺得 Firefox Flicks 是一個超棒的機會能讓學生參與現實世界的活動，這個競賽可以讓很多人看到，且由電視電影與廣告等產業的專家進行評分。歡迎跟您的教授談論把影片做為作業之一。
我可以在短片當中使用 Firefox 的商標與圖示嗎？
可以。以拍攝這個比賽要用的短片為目的的話，您可以使用 Firefox 商標與圖示。若需要更多關於商標使用的資訊，請參考 Mozilla 商標使用政策。
我要如何才能取得高解析度的 Firefox 名稱與圖示圖檔？
若需要高解析度的圖檔還有使用規範的話，請看看我們的 Brand Toolkit 網頁。
我要到哪裡才可以找到音樂或其他素材放到短片當中？
若您需要更多素材，不論是音樂、鏡頭素材、美工圖片等等，我們建議您使用公有領域的媒體素材。創用 CC 提供了一個不錯的搜尋服務讓您可以尋找公有領域作品。您可以到 Wikipedia 取得更多關於公有領域的資訊。
我可以在短片當中使用第三方商標或受著作權保護的素材嗎？
不可以，未經這些素材的擁有者授權前請勿使用。
我可以用非英語影片報名嗎？
可以。對於非英語的作品，您需要一定送出完整且精確的字幕或腳本。若您沒有辦法翻譯您的短片，請 聯絡我們。我們可以幫忙。
如果我想撤回我的作品怎麼辦？
如果您想要放棄報名，請寫信到 firefoxflicks@mozilla.org 聯絡我們，我們將會自比賽中移除您的作品。
我的作品應該使用什麼長寬比例製作？
我們接受任何長寬比例的影片，但我們推薦您用 16:9 製作。
報名的影片將會向大眾公開嗎？
您的短片將會被張貼在投票網站，這樣網友們才能幫忙投票選出大獎得獎者。
