個人化的瀏覽體驗

自訂您的 Firefox，第一個在行動裝置上提供擴充能力的瀏覽器。

親手打造屬於您個人的瀏覽器

Mobile 附加元件網站幫助您在移動時也能方便使用網路。您可以安裝附加元件，透過社群網站分享您的最新消息、自動修正打錯的網址、或者安裝附加元件提供翻譯功能……等等，就在您的行動裝置上。附加元件免費且易於安裝，所以現在就試試看吧！

直接在手機上安裝

您可以直接在手機上搜尋並且安裝 Firefox Mobile 附加元件。選擇螢幕右下側的設定按鈕（齒輪），並選擇附加元件按鈕（拼圖）即可。

想要開發 Mobile 附加元件嗎？

您的創新帶領 Firefox 邁向成功。我們已試著讓開發 Mobile 附加元件的方法更加簡單。想了解更多資訊，請閱讀 Mobile 開發網頁。
