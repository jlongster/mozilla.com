為什麼要在您的 iPhone 上使用 Firefox Home？

隨身帶著走

隨時隨地讓您喜歡的網站在 iPhone 上等著您：您的歷史資訊、開啟分頁，以及書籤。用法就和您習慣的智慧位址列一模一樣。

不用從頭開始

不需要輸入冗長的網址，或是浪費時間來搜尋您最愛的網站。就讓 Firefox 的歷史紀錄在您打開 iPhone 的時候幫您代勞吧。

甜蜜的家

家是很重要的，所以您需要保全來保護您的私密資料。Firefox Home 也一樣，您的加密金鑰可以保證只讓您存取您所儲存的資訊。

開始使用

  1. 在您的電腦安裝 免費的 Firefox Sync 附加元件。

    Firefox Home 使用了 Firefox Sync 附加元件的同步功能以存取您的瀏覽器資訊，並安全地將這些資訊傳送到您的 iPhone。

  2. 重新啟動 Firefox 之後，請依照畫面的說明輸入密碼、加密金鑰來建立您的帳號。

    Firefox Home 使用您的加密金鑰在本地端進行加密，保護您的資訊。

  3. Firefox Sync 將會開始安全地同步您的資料。第一次的同步可能需要最多二十分鐘。

    Firefox Sync 將只會在您不在私密瀏覽模式時工作。當您完成第一次同步時，您會在工具選單當中的 Sync 選單看到最後更新時間。

  4. 要完成設定，請在您 iPhone 上的 Firefox Home 也輸入帳號、密碼、加密金鑰，並且按下「完成」。

    如果您登入時發生問題，請 點一下這裡以取得幫助。

  5. 當您完成設定之後，您將可以在 iPhone 上存取您電腦的書籤、歷史資訊，以及開啟的分頁。

    如果您的資料沒有出現，請 點一下這裡以取得幫助。
