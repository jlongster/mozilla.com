Firefox 功能

來看看 Firefox 如何將網路帶到您的行動裝置上。

獨特功能

真實本色

忘了那些綁手綁腳的行動版瀏覽器吧。Firefox Mobile 帶著您所期待的效能、安全、以及更多您在桌面版 Firefox 熟悉且喜愛的功能而來。

大飽「眼福」

分頁與瀏覽按鈕輕巧的收納在螢幕側邊；從此您可以清楚看見完整網頁，不再被工具列遮蓋。享受全新視野吧！

雙手萬能

透過直覺的手勢讓你簡單加入書籤、選擇分頁與控制螢幕縮放。

智慧位址列「Awesome Bar」

忘記那些落落長的網址吧！讓智慧位置列指引您快速到達想去的網站。

個人化的瀏覽體驗

帶著最愛的附加元件一起旅行吧！首創行動瀏覽器附加元件功能，使您得以打造個人的隨身瀏覽體驗。

密碼管理員

輸入密碼這種事在手機上麻煩至極，何必一再重複？首次登入後選擇「記住密碼」，再也不必重複又難又煩的輸入動作。

不論身在何處，都是識途老馬

使用地理定位功能來瀏覽地圖、取得在地的內線消息。

起而行！

別忘了帶 Firefox 一起出門。下載 Firefox Sync，同步您的 PC 與行動上網生活。

安全性 & 隱私保護

即時檢視網站身分資訊

想要在購物前再次確認網站身分嗎？按下網站圖示立刻顯示網站身份資訊，再次按下以顯示更多資料：您已瀏覽的次數、是否有已存密碼等等。如此來避開詐騙及有害網站，讓您確定網站的真實身分。

密碼管理員

輸入密碼這種事在手機上麻煩至極，何必一再重複？首次登入後選擇「記住密碼」，再也不必重複又難又煩的輸入動作。

清除最近的歷史記錄

輕輕點選即可清除您所有的個人隱私資訊，或是過去幾個小時的瀏覽記錄。您可以完整掌握要刪除什麼資訊，且刪除的資訊會永久消失。就是這麼簡單。

自訂安全性設定

設定您要 Firefox 多麼仔細審查網站的安全，並設定白名單。指定個別網站的密碼紀錄、Cookies、載入圖片與安裝附加元件權限，使您在行動網路上也能擁有完整控制權。

自動更新

我們的開放源碼安全策略，讓我們能在最短時間內找出、並且修復程式的安全漏洞，使 Firefox 成為上網最安全的瀏覽器。您可以在收到更新通知時立即更新 Firefox，或者等候方便的時刻再進行。

阻擋彈出型廣告視窗

阻擋網頁彈出廣告視窗，讓您在網路旅程上與其從此不見。您也能選擇想要開啟的彈出視窗、或是建立允許使用彈出型視窗的網站清單。

方便自訂

自訂的瀏覽體驗

附加元件現在可以帶著走了，所以開始依您所需自訂瀏覽器吧！安裝附加元件透過社群網站分享您的最新消息、自動修正打錯的網址、或者安裝附加元件提供翻譯功能……等等，就在您的行動裝置上。

有了 Firefox Sync，您可隨時起而行！

使用安裝 Firefox Sync 的 PC 來上網吧，當你離開電腦時，行動裝置上的 Firefox Sync 會幫您備齊所有的最新資訊：瀏覽記錄、分頁、書籤以及網址列紀錄，就像您從未中斷瀏覽一般。Firefox Sync 讓您生活更順心！了解更多。

單鍵快速新增書籤

輕鬆快速管理您的書籤。瀏覽至想要新增書籤的網頁，再點選螢幕右側的星星按鈕即可。欲尋找某個書籤，只要在位址列輸入書籤名稱或是標籤；您越常透過位址列使用書籤，Firefox 越快適應您的使用偏好。

智慧位址列「Awesome Bar」

您只管輸入，讓 Awesome Bar 幫您找出您想要的頁面。Awesome Bar 記得您曾瀏覽、加入書籤與標籤的網頁。簡單的說，您只要簡單輸入幾個字，就能快速找到目標（現在您知道我們為何叫它「智慧位址列」了吧！）

增進效率

雙手萬能

輕鬆點擊將您最愛的網頁加入書籤，手指劃過螢幕以選擇分頁及附加元件，雙點螢幕即可縮放。就是這麼聰明簡單！

下載管理員

無縫且安全的下載檔案！輕鬆下載您的檔案，下載管理員會顯示進度、讓您直接在下載清單中透過網址及檔案名稱來尋找檔案。

拼字檢查

檢查按錯鍵或拼字錯誤。內建的拼字檢查功能，能讓您在網頁上撰寫部落格文章、或者輸入電子郵件時，不必擔心拼字錯誤。不拐彎抹角、直接在頁面上進行。

記憶體管理

輕巧快速，Firefox 執行時會使用更少的記憶體。內建新的記憶體管理功能，Firefox 會節約使用記憶體，保持您瀏覽網頁時的速度。

美妙分頁

打開您的掌心，在分頁間翱翔網路。只要向右輕劃螢幕，您就能快速在縮圖間找到您想要瀏覽的分頁。

搜尋

多重搜尋引擎

從位址列直接使用您最愛的搜尋引擎。您可以安裝選用多重搜尋引擎，或是只固定使用某一個。從預設選單中選擇想要的搜尋引擎。

普及存取

增強的文字佈局功能

所見如所寫，用生活中的寫法來顯示您的語言。全新圖形佈局引擎能正確處理顯示阿拉伯文、波斯文、旁遮普文等語言。

網頁內嵌字型

Maemo 平台上已提供內嵌字型顯示功能。網站設計師可以為自己的作品設計特殊的字型，在網頁中提供更廣泛的字型選擇。

在地特色

軟體在地化不只是翻譯畫面訊息而已。我們的在地化團隊志工知道什麼最適合您，為您準備了在地的翻譯／拼字檢查字典、搜尋引擎、即時書籤與消息來源。您會看到v像是當地的 BBC 頭條新聞，或者百度（中國流行的搜尋引擎）所提供的內容等等各式地域性資訊。

各種語言版本的 Firefox

Firefox 比其他瀏覽器懂得更多的語言。我們相信網際網路為全人類所共有，因此我們竭盡所能的提供各種語言版本。目前有 30 種不同語言的 Firefox Mobile，還有更多語言版本陸續推出。

領先群雄

位置感知的瀏覽體驗

想尋找城裡最棒的 Live House 或者通往高速公路的捷徑嗎？使用地理定位功能來瀏覽地圖、取得在地的內線消息。這個功能是全然自願性的——在未獲得您的同意前，Firefox 不會隨意透露您的地點。

大飽「眼福」

分頁與瀏覽按鈕輕巧的收納在螢幕側邊；從此您可以清楚看見完整網頁，不再被工具列遮蓋。享受全新視野吧！

離線瀏覽

即便您的手機沒有網路連線，您還是可以隨手取得想要的資訊。某些具備離線功能的網站，例如網頁郵件網站，可以讓您隨時隨地讀取訊息。

新世代的圖形格式

您必須眼見為憑。藉由我們的新繪圖引擎，文字與圖形（包含 SVG 與 Canvas）的顯示，從未曾如此快速清晰。Firefox 會應用圖檔中的校色資訊，修正顯示的顏色，確保您的數位照片色彩如真。

最新網路科技

Firefox 包含了最新與增強的網路技術，讓網站作者與開發者可以開發更進階的功能。最新支援的技術包含：

  • CSS2.1
  • CSS3
  • HTML5
  • SVG 過濾器
  • 原生 JSON 處理
  • MathML
  • 動態 PNG (APNG)
  • 跨網站 XMLHttpRequest
  • Web Worker 執行緒
