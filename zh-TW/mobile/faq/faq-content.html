Mobile 常見問題集

對於 Firefox Mobile 的常見問題

關於 Firefox

為何要推出行動裝置用 Firefox？
在行動裝置上提供 Firefox，使 Mozilla 朝向目標邁進：讓您在任何地點或任何裝置，都可以擁有相同的網際網路體驗。另外，Mozilla 也立志於提供網路開發者更豐富的平台來建立創新與引人入勝的網路應用。Firefox 身為目前最先進的行動版瀏覽器，不但速度快、安全、可充分自訂以個人化，還能與您的桌面瀏覽器完美同步。
我需要花錢購買嗎？
不需要！Firefox 由非營利的 Mozilla 基金會推出，我們承諾為網際網路帶來開放、創新與更多機會。我們認為開放源碼軟體，例如 Firefox 能夠促使網際網路繼續做為全球大眾共有的公共資源。了解更多。
Firefox 跟我現有的行動版瀏覽器有什麼不同？
Mozilla 將您的 Firefox 瀏覽體驗帶到行動裝置上。Firefox Mobile 內建最新的瀏覽器引擎，以及您早已熟悉的智慧位址列「Awesome Bar」、密碼管理員、單擊新增書籤、地理定位瀏覽等功能。Firefox 也是行動裝置上第一個具有擴增能力的瀏覽器，讓使用者隨時自訂瀏覽體驗。
Firefox 使用我的語言嗎？
行動裝置用 Firefox 目前提供超過 25 種語言的版本，即將推出更多語言。您可以參考我們的完整語言列表。
Firefox 是由誰所製作的呢？
Firefox 由非營利的 Mozilla 基金會推出，由來自全球各地的社群成員共同製作。因為來自於社群的貢獻，我們才能為 Firefox 加上更多更安全、更快、且功能更強大的功能升級與附加元件。想要協助我們嗎？請參考 Mobile 開發網頁。

安全性

Firefox 如何在我使用行動裝置瀏覽時確保我的網上安全？
Mozilla 的公開源碼開發過程，使來自世界各地的網路安全專家得以參與，且 Firefox 使用了使用 Web site ID 檢查可疑網站，並提供可自訂的安全性設定，讓您管理密碼、Cookie、圖片載入動作與附加元件，完整掌控個人的瀏覽體驗。了解更多。

下載安裝 & 行動裝置

如何在我的行動裝置下載安裝 Firefox？
請於 PC 上開啟 Mozilla.com Mobile 網頁，輸入您的電話號碼以傳送內含下載鏈結的簡訊至手機上。您也可以使用現有的行動版瀏覽器，至 mozilla.com/m 直接下載 Firefox。
Firefox 可以在我的行動裝置上運作嗎？
Firefox 目前相容於 Nokia N810、N900。查看完整的支援列表。
即便裝置不支援，我還可以下載安裝 Firefox 嗎？
即便您的裝置沒有列在我們的已測試清單中，您還是可以試著在 Maemo 上安裝 Firefox，但我們不保證它的功能性。我們建議您使用支援的行動裝置，以完整體驗所有功能。查看完整的支援列表。
可以在 iPhone 上面使用 Firefox 嗎？
目前 Firefox 並沒有供 iPhone 使用的版本。

使用 Firefox

要如何使用 Firefox 的介面來瀏覽網路？
為了增加螢幕的可用空間，Firefox 把控制按鈕收納於螢幕側邊。用手指向左或是向右劃過畫面即可顯示。分頁放在左側、右側則有書籤與設定按鈕。往下捲動網頁時位址列會自動隱藏，向上劃過螢幕就會再度出現。
如何縮放網頁？
用手指輕點螢幕兩下即可放大網頁，再次輕點螢幕兩下即可縮小。
如何將網頁加入書籤？
請開啟該網頁，再點選螢幕右側的星星按鈕即可。
如何開啟新分頁？
新增分頁的加號按鈕在螢幕左側，點選即可開新分頁。分頁會顯示為縮圖，讓您可以清楚辨別、便於點選。

功能

全能位址列「Awesome Bar」是什麼特別的功能？
Awesome Bar 讓您能快速開啟想要的網頁。點選 Awesome Bar 然後開始輸入文字，當您輸入時它會即時顯示您最常瀏覽的網頁或是書籤。Awesome Bar 擁有學習能力，在使用的過程中會了解您的習慣，提供更適合您的瀏覽體驗。您也可以直接在 Awesome Bar 輸入關鍵字來搜尋網路。
什麼是 Firefox 附加元件？如何在行動裝置上安裝？
您可以安裝附加元件來新增 Firefox 功能。點選螢幕右下側的設定按鈕（齒輪），並選擇附加元件按鈕（拼圖），即可在您的行動裝置上直接搜尋、安裝附加元件。
要如何在我的行動裝置、手機和 PC 之間同步 Firefox 的書籤、瀏覽記錄及開啟分頁等資訊？
「Firefox Sync」這個附加元件把您桌面的個人體驗帶到行動裝置上。首先在 PC 上建立一個 Firefox Sync 帳號，接著在行動裝置上安裝 Firefox Sync 附加元件並輸入您的帳號、點選同步，您即可在 PC 和手機與行動裝置間共用 Firefox 偏好設定及瀏覽記錄等。
我一定要使用 Firefox Sync 才能在行動裝置上使用 Firefox 嗎？
不一定要用。雖然我們認為透過 Firefox Sync 來同步您的行動裝置、手機和 PC 是超級好用的功能，您仍然可以自行決定要不要使用。沒有 Firefox Sync 您還是可以享受您所信任、期待的 Firefox。無論您身在何處，Firefox Sync 都能讓您同步瀏覽記錄、書籤、個人偏好設定等等，提昇您整體的瀏覽體驗。
什麼是地理定位功能？Firefox 會追蹤我在哪裡嗎？
地理定位功能能告知網站您的所在位置，以節省您輸入的時間。如果您同意透露地點資訊，網站就能提供您附近的相關訊息，像是周遭的地圖。這個功能是全然自願性的 – 在未獲得您的同意前，Firefox 不會隨意透露您的地點。了解更多關於位置感知的瀏覽體驗與 Google Location Service 的 隱私權保護政策。
地理位置會有多準確？
隨著地點不同，精準程度會有很大的差異。在某些地方，我們的地理定位服務可以精確至數公尺，然而在其他地方誤差也可能遠大於此。我們的地理服務所回傳的定位資訊都是估計值，我們並不保證準確性。

開發相關

我撰寫的桌面版 Firefox 附加元件可以轉移到行動裝置上嗎？如果不行，要怎麼修改？
雖然桌面版與行動裝置版 Firefox 使用同樣的平台，兩者還是有一些小差異，需要開發者修改附加元件。這些變更通常都很直覺，請閱讀 Mobile 開發網頁以了解詳情。
Firefox 使用哪個佈局引擎？
桌面版與行動裝置版 Firefox 同樣使用 Gecko 佈局引擎。Firefox Mobile 使用 Gecko 1.9.2 版，是 Mozilla 最新、效能最好的佈局引擎，包含增強的 JavaScript 效能、地理定位支援，Work Threads 等等。
Firefox 使用哪個 JavaScript 引擎？
Firefox 使用新的「TraceMonkey」JavaScript 引擎，一個使用 Tracing 技術的高等 JIT（just in time）編譯器。TraceMonkey 在執行時即時編譯程式碼，讓網路應用程式執行時更為快速，且讓許多仰賴 CPU 密集運算的應用程式已接近原生的速度執行。
我要如何開發使用 Firefox 地理定位技術的網路應用？
Firefox 實做了第一版的 W3C 地理定位規格書第一版公開草稿。您可以在該處看到地理定位技術的概觀。
Firefox Mobile 支援 HTML 5 嗎？
HTML 5 是 HTML 的下一代修訂版本。HTML 5 規格書還在修改中，但是已有許多 HTML 5 與相關規格可讓開發者做出更多更有創新的網路應用。Firefox 包含重要的新功能像是 地理定位瀏覽、<canvas> 與 <svg> 標籤、Web Workers 與 離線儲存。
我手機上的 Firefox 支援外掛（Plug-in）嗎？
No, not initially. Support for plugins in Firefox for Maemo has been disabled. On most Web pages that use the Adobe Flash plugin, the performance of the plugin didn't meet our standards, and the interactivity and performance of the entire Web page was negatively impacted. This was especially felt on pages with multiple instances of the plugin. Advanced users can enable plugins for experimentation and testing purposes only. We are working on ways for the user to have control of which sites to enable plugins for, as some sites, like YouTube, do work quite well. That capability will most likely be packaged as a browser add-on.
