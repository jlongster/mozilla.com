功能櫥窗

Firefox 4 有了良好的基礎。這裡還有一些您不會想要錯過的新功能:

平滑的圖型表現

全新的 JavaScript Values 代表 Firefox 可以更有效的執行大量的數值運算，得到更平滑的圖形表現。

SQLite 3.7.0

聰明絕頂的智慧位址列、下載管理員，以及 IndexedDB 功能可以更快、更順暢，歸功於升級後的 SQLite，一個 SQL 資料庫引擎。
