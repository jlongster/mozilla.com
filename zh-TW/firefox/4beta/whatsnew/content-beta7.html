繫上安全帶

Firefox 4 現在加上了大量的效能改進，包含全新的 JägerMonkey JavaScript 引擎。從更快的啟動時間、圖形渲染，到改善過的頁面載入時間，您馬上就可以發現與以往不同之處。

網際網路上的 3D

WebGL 將 3D 圖形帶來了 Firefox，同時也為開發者開了一扇門以建立生動的遊戲、全新的網頁視覺效果與體驗。

驚人的字體效果

Firefox 4 的 OpenType 字型功能提供設計師與開發者更多的字型內容控制—像是字距調整（kerning）或是合字（ligaturees）—來建立華麗的網站。

升級的表單功能

Firefox 4 讓您可以美觀且簡單地實作表單。新的表單 API 將提供您工具來透過列表屬性啟用自動完成以及 HTML5 的表單驗證等功能，讓惱人的表單開發成為過去式。
