略通一二

玩樂標籤

當您收集了一大堆書籤，要整理它們就變成是相當困擾的一件事。要維持書籤的整齊，您可以對每個書籤下標籤：只要在星星上點兩下就能輸入您想設定的標籤了。

您可以同時下多個標籤（每個標籤用逗點分隔），未來在網址列直接輸入標籤就能直接找到該書籤。例如，在網址列輸入「旅行」就能找到所有您之前計畫旅遊行程時所儲存的書籤。

了解附加元件

標準版的 Firefox 含有常用的一切功能，但功能永遠不嫌多，不是嗎？超過 5000 種附加元件（能讓您自訂 Firefox 的附加小程式）可供您下載，只要是您想的到的需求幾乎都有對應的元件可用。請到「工具＞附加元件」開啟「附加元件管理員」，探索 Firefox 的無限可能。

聰明搜尋

或許您已經熟知 Firefox 內建的搜尋功能，但是您知道可以自訂搜尋列，加上任何您想要的搜尋引擎嗎？只要按搜尋列左邊的圖示，就能看到預設引擎供您選擇。

更棒的是，按「管理搜尋引擎」還可以重排搜尋引擎順序、新增其他搜尋引擎，甚至幫你常用的搜尋引擎設定快捷鍵。例如：你可以為 Google 指定快捷鍵「G」－若之後您想搜尋餅乾食譜，只要在位址列打「G 餅乾食譜」就能搜尋。

如閃電般尋找文字

「隨打即找」功能是另一個省時間的絕技。除了用尋找列尋找網頁上的文字外，只要在網頁空白處輕按一下，就能直接輸入您想尋找的文字。游標會馬上跳到該詞第一次出現的位置。

您也可以用「隨打即找」功能搜尋鏈結。例如：直接輸入「Learn More」後按 Enter 就能直接開啟，再也不需要花時間拉捲軸、移動滑鼠尋找鏈結。

走過不留痕跡

由於安全性和保護個人隱私之故，當您瀏覽時留下足跡是不太好的（特別是您正在用多人共用的電腦時。）在 Firefox 裡，只要一個按鈕就能清除瀏覽記錄、下載記錄、網頁快取以及 Cookie。只要按「工具＞清除隱私資料」即可。若您還是放不下心，到「工具＞選項＞個人隱私」後選擇「結束 Firefox 時清除隱私資料」。

超強高手

直達您最愛的網站

您可以在書籤上加入關鍵字，如此一來您就可以更方便、更快速地使用書籤。您只要在收藏庫中輸入關鍵字，您就可以通過在網址列輸入關鍵字直達網站。例如，您可以給您的「del.icio.us」書籤的關鍵字欄中打上「links」，那麼下次您只要在網址列打上「links」，您就可以直達「del.icio.us」。

建立智慧書籤夾

如果您是一位時常需要儲存和紀錄各式各樣網頁的重度上網者，那麼這裡有一個節省時間的好功能：Firefox 3 的收藏庫。您可以建立和儲存各種搜尋條件為智慧書籤夾，如此一來，每當您增加網頁到您的書籤或瀏覽記錄的同時，資料夾也會根據搜尋條件自動更新。

首先，從書籤選單中選取「顯示所有書籤…」開啟收藏庫。然後，在搜尋欄打上搜尋條件。最後按一下儲存，智慧書籤夾就完成了。

管理您的下載

如果您時常下載檔案，請利用下載管理員追蹤所有下載項目。您可以直接在下載管理員裡暫停、續傳或開啟下載檔案。

如果您需要找出之前的下載項目，請到「工具＞下載管理員」中，於搜索列尋找您的檔案。一但您找到想找的檔案，您可以按兩下開啟該檔案或按 Ctrl+左鍵後選擇「複製下載鏈結」。

探索開發者工具

如果您是網頁設計者，Firefox 的開發者工具會讓您的生活更輕鬆。Mozilla 附加元件網站 提供許多工具使開發過程更為流暢，例如：Firebug 可以在任何網站即時編輯、除錯、查看 CSS、HTML 與 JavaScript；Tamper Data 可以查看 HTTP/HTTPS 檔頭與 POST 參數；DOM Inspector 可以透過簡單的右鍵點擊檢查任何的 HTML 或 CSS 元件。

依您喜好設定網路

現在您可以透過網路程式協定處理器提供您快速存取您喜好的網路應用程式。例如：您可以設定 Firefox 點選 mailto: 連結時開啟您喜好的電子郵件網站，而非電腦上預設的郵件處理程式。（附註：此功能只能選用已在 Firefox 3 中註冊的電子郵件網站）

打開 「工具＞偏好設定＞應用程式」 以選擇各種通訊協定預設的應用程式，若選擇「每次詢問」則您每次都需要選擇要以何種程式處理。

如果您是網路應用程式設計者，可以到 Mozilla 開發者中心 看看有關如何支援網頁通訊協定的詳細資訊。
