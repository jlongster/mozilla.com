Firefox 4 Beta 包含以 Mozilla Test Pilot 為基礎的回饋附加元件

您安裝 Firefox 4 Beta 測試版之後將會發現視窗右上角新增的意見回饋按鈕。您可以隨時使用這個功能提供立即的回饋，您也將會在透過 Mozilla Test Pilot 所進行的使用者研究開始前收到通知。使用者研究將會收集關於您如何使用 Firefox 的資訊，但不包含您瀏覽過哪些網站。您有完全的控制權，可以選擇離開任何一份研究或是看看已經被收集了那些資料。您將會在資料被送出到 Mozilla 之前再被詢問一次。請參考 回饋資料的隱私權保護政策 以取得更多資訊。
