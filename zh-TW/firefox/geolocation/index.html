位置感知的瀏覽體驗

Firefox 可以向網站透露您所在的地理位置，讓他們提供與您更有用的資訊。這讓網路更加 聰明 —— 而且過程完全尊重您的個人隱私。 試試看！

常見問題

什麼是地理定位功能？

使用地理定位功能的網站可以詢問您所在的地理位置，以提供更有用的資訊，或是節省您搜尋的時間。例如，您在找一間附近的比薩店，網站可以直接向您詢問地理位置，這樣您只要搜尋「比薩」就能找到解答 ... 不需輸入更多細節或是其他關鍵字。

或是，您在規劃如何到某處的路線，網站會知道您要從哪出發，您只要輸入您要去哪裡即可。

這個服務完全是自願性的 —— Firefox 不會沒有得到您的允許就透露您的地理位置 —— 完全尊重您的個人隱私。而且，就像 Firefox 的其他元件一樣，這個功能也是使用開放標準開發，使網站開發者能輕易的採用。

它如何運作？

當您瀏覽到具有地理感知功能的網站時，Firefox 會向您詢問您是否想要透露您的地理位置。

若您同意，Firefox 會收集附近的無線網路狀況與您的電腦的 IP 位址。接著，Firefox 會將此資訊傳送給預設的地理定位服務提供者 Google Location Services 供其估計您的地理位置。估計結果會再傳送給詢問的網站。

如果您不同意，Firefox 不會進行任何動作。

地理位置估計會有多準確？

準確程度隨著地點不同有很大的差異。在某些地方，地理定位服務可以定位到數公尺的誤差。然而，在其他地方準確度會遠大於此。所有服務提供者所回傳的地理定位資訊都是估計值，且我們不保證它的準確性。請不要使用此服務作為緊急使用。請別忘了用常識保持判斷地點。

什麼資訊被傳出去，傳給誰？你們怎麼保護我的隱私？

您的個人隱私對我們來說至關重要，且 Firefox 不會沒有得到您的允許就透露您的地理位置。當您瀏覽到要求您的個人資訊的網頁，我們會在任何資訊傳送到網站與第三方服務之前就徵求您的同意。

根據預設值，Firefox 藉由傳送下列資訊給 Google Location Services 來決定您的地理位置：

  • 您電腦的 IP 位址，
  • 鄰近的無線網路 AP 的資訊，以及
  • 一個隨機產生、兩個星期後過期的客戶端辨識 ID，由 Google 指定。

要了解 Firefox 如何使用與收集資訊的細節，請閱讀 Firefox 隱私權保護政策頁面（英文）。

Google Location Services 接著回傳所估計的您的地理定位（例如，經緯度。）要了解 Google如何使用與收集資訊的細節，請閱讀 Google Geolocation 隱私權保護政策

這些資訊會經由加密連線傳送，以保護您的個人隱私。得到您的地理定位資訊後，Firefox 會將此資訊傳送給提出要求的網站。Firefox 決不會將您瀏覽網站的名稱或是網址，或是任何Cookie，傳送給 Google Location Services。

Mozilla 與 Google 也絕對不會使用 Google Location Services 收集到的資訊來辨識或是偷窺您。

要了解要求您的地理位置資訊的網站如何保護您的個人隱私，請瀏覽該網站的隱私權保護政策。

要了解更多關於您的個人隱私保護，您也可以閱讀：

  • Mozilla Firefox 隱私權保護政策 （英文）
  • Google 隱私權保護政策（英文）

我瀏覽網路的時候會被跟蹤嗎？

不會。Firefox 只有在網站要求地理定位資訊的時候才會向您詢問，且只在使用者同意時才會透露您的地理位置。Firefox 在您瀏覽時，不會追蹤或是記錄您的地理位置。

我要如何取消給與某個網站的權限？

如果您曾經給予 Firefox 總是透露您的地理位置給特定網站的權限卻又改變了您的想法，您可以很輕易的撤回那項權限：

  • 開啟您曾經給予權限的網站
  • 開啟工具選單，選擇頁面資訊
  • 選擇權限頁面
  • 修改透露地理位置設定

我要如何清除「客戶端辨識 ID」？

  • 開啟工具選單，選擇清除最近的歷史記錄
  • 點選細節箭頭
  • 確定已選擇Cookie項目
  • 點選立即清除

如何永久關閉地理定位功能？

地理定位功能在 Firefox 是可取消的選項。在沒有您的同意之前，不會傳送任何地點資訊。如果您想要永久停用此功能，請依以下步驟操作：

  • 在位址列輸入 about:config
  • 輸入 geo.enabled
  • 連點兩下 geo.enabled 偏好設定
  • 至此，地理定位功能已停用
