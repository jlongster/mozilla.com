機會與挑戰

在今日，我們的生活當中更常待在線上。這帶給了 Mozilla 打造自由與讓使用者在生活中更多不同領域團結再一起的機會。同時，隨著行動科技的進步，讓我們可以往一些被不同的巨型商業組織所控制的不同的平行世界前進。

我們已經遇過這種威脅了。Mozilla 就是在這種環境下誕生的，作為一個有滿滿的承諾與潛力的線上世界的反應，然而馬上就被一些限制使用者選擇、可互相操作性，以及網路的活力與生命的大公司所威脅。我們面對並贏得了挑戰，然後，我們必須在這個新紀元下再度面對它。

一個競爭的世界

Firefox 展示了瀏覽器有多重要，現在其他的對手也已經進入市場了。競爭是好的，這讓我們可以做出最棒的東西。但我們必須記得要打造網頁瀏覽器的原因已經完全與以往不同了。Mozilla 的獨特之處就是在於我們打造 Firefox 來提供一個獨立的產品，並聚焦於每個人的體驗與網路上美好的事物。

「Mozilla 的獨特之處就是在於我們打造 Firefox 來提供一個獨立的產品，並聚焦於每個人的體驗與網路上美好的事物。」

為了在此處保持競爭力，Mozilla 有以下的作法:

首先是讓 Firefox 變得更好。從前 Mozilla 需要大約 18 個月才能釋出新功能。從今年開始我們縮短為六週一次的開發循環，讓我們的產品能有效地被更新。我們正在送出創新的網頁技術，不是每年，而是每幾個禮拜。我們在破紀錄的時間當中將像是 CSS 動畫、Web Sockets、tear-off animation，以及 WebGL 等技術釋出給數以億計的使用者。還有更多工作需要完成才能確保新的時程對每個人都有效，但我們已經克服了在第一個十年當中能正常使用的偉大障礙。

我們也開始將 Android 加入至一級平台與我們的桌面平台一起開發。這個重要的轉移過程幫我們擴展 Mozilla 的使命到下個世代的網路。在這個新的設定上，Firefox 不再只是您桌面上的瀏覽器，而是一個受信任的環境，讓使用者可以對他或她的體驗有一致的控制，不論裝置。

作為目前正在努力讓使用者可以控制她們網路體驗的的一部分，我們在 Firefox 桌面版與行動版當中開發了 「不要追蹤我」 功能。業界的支援馬上就跟上了。Apple 與 Microsoft 也在他們的瀏覽器中實作了類似的解決方案。美國聯邦貿易委員會公開地支援了不要追蹤我功能，歐盟執行委員會的副主席也將在 2012 年將此功能採納。我們刻意要在這個領域當中打造這些功能，也將會持續努力倡導使用者探索新的讓網路以及收集使用者資料變得更透明的方式。

Firefox 目前正，也將會持續透過 Mozilla 最重要的開放、透明，與可互相操作性等價值作為最基礎的扳手，帶領並倡導網路的進步。

改善網路功能

Mozilla 一直都在前線，讓網路成為一個更強大、更豐富，以及更引人注目的平台。我們在今日也將持續這個領導地位。

網路身份

現在的網路需要一個開放原始碼，以標準為基礎的平台讓全世界地時都可以使用，且去中心、去自訂化的網路 身分識別。Mozilla 已經打算好要開始打造了。透過 BrowserID 專案作為第一步，Browser ID 是一個安全、去中心化、開放原始碼、跨瀏覽器的方式，讓您可以透過您的電子郵件地址登入不同網站。

應用程式

App 是一個全新、便捷的方式讓您與網際網互動，但它缺乏了關於美好的網路的 一些新功能。Mozilla 開放應用程式生態圈將會讓使用者可以讓它們在不同的平台和裝置間隨身帶著走。它將會從網路上的不同提供者間橋接聯絡人清單與社交圖。這將會允許使用者用開放且彈性的方式探索應用程式，就像是我們在網路上探索其他內容一樣。

教育

Mozilla 教育計劃的目的是來啟發新一代的網頁作者。School of Webcraft 將會透過同儕間的合作、社群來教導開放的網頁開發技巧。Mozilla Hackasaurus 計畫將會教導年輕人打造網頁技術當中的一磚一瓦與持續開發。Mozilla 也持續在發展新的面向，以透過我們的 Open Badges 專案進行線上學習。

媒體

Mozilla 支持能讓網路媒體帶來開放性與創新的專案與工具。這包含了 Web Made Movies，一個開放的視訊實驗室，讓網頁開發者跟影片導演結合起來。Mozilla 與 Knight Foundation 的夥伴關係也同樣的將網路創新者、記者，以及新聞組織結合在一起，帶來新聞學中的創新。也支援像是 Universal Subtitles、Open Video Alliance 以及 Bay Area Video Coalition 等計畫。

WebFWD

WebFWD (Web Forward) 是 Mozilla 的育成計畫。我們將 WebFWD 同伴們，包含了讓下一代網路更進一步的工人與創新者集合起來，並在 Mozilla 提供他們一個空間，透過提供許多技術領域當中的領導者來指導，以及其他世界級的資源，一起幫助讓網路更進一步。

Boot to Gecko

Mozilla 最近啟動了 Boot to Gecko 計畫，旨在讓網路成為行動裝置的底層平台。目的在於透過開放的網頁技術來打造一個完整且獨立的作業系統，並為未來的行動與桌面應用程式建立一個更好的基礎。
