Non trovi il tuo telefono?

Se il tuo telefono non è presente nell'elenco, puoi comunque scaricare e utilizzare Firefox sul tuo dispositivo mobile, a patto che disponga di sufficiente RAM (128 MB o più) per l'esecuzione di Firefox. In ogni caso ti consigliamo di scaricare Firefox su uno dei dispositivi già testati, così sarai in grado di sperimentare a pieno tutte le sue funzionalità.

Puoi anche provare Firefox, fornire feedback, realizzare componenti aggiuntivi e collaborare installando sul tuo computer una versione desktop di Firefox:

  • Mac OS X
  • Linux
