Bienvenue à Sync dans Firefox 4 Beta
Avec Sync, vous avez accès instantanément à toutes vos données Firefox importantes…
…comme les marque-pages, l'historique, les mots de passe enregistrés et même les onglets ouverts…
…où que vos utilisiez Firefox, sur tous vos ordinateurs et un nombre grandissant de téléphones portables.
Vous êtes pressé de partir mais vous venez juste de trouver un site Web intéressant ?
Avec Sync, vous pouvez reprendre où vous vous étiez arrêté avant de partir de chez vous, au travail ou sur le trajet – C'est tout simplement magique.
Pour activer Sync, vous devez créer un compte. Commençons.
Quand vous téléchargez la dernière version de Firefox 4 bêta, vous aurez une invitation pour créer un compte Firefox Sync.
Si vous utilisiez déjà Firefox Sync, cliquez simplement sur « J'utilise déjà Sync sur un autre ordinateur ». Et le tour est joué.
Si c'est la première fois que vous utilisez Sync, cliquez sur « Je n'ai jamais utilisé Sync ».
Saisissez un nom d'utilisateur… un mot de passe… et une adresse électronique.
Créez maintenant votre phrase secrète.
Cette phrase de 12 caractères ou plus est semblable à la clé de votre coffre-fort auquel vous seul avez accès.
Ne la perdez pas car elle protège vos données.
Cliquez sur « Suivant » à moins que vous ne vouliez héberger les données sur votre propre serveur.
Saisissez les mots tels que vous les voyez.
Enfin, cliquez sur « Terminer ». Vous disposez à présent d'un compte Firefox Sync !
Vos données Firefox vont commencer à se synchroniser en tâche de fond.
Si vous paramétrez Sync pour la première fois, cela pourrait prendre un peu de temps, mais vous pouvez continuer à naviguer sur le Web.
Lorsque la synchronisation initiale est terminée, l'icône en rotation se transforme en l'icône de Sync.
Dès ce moment, Sync mettra à jour périodiquement vos données pour que vous soyez toujours synchronisé.
N'oubliez pas de paramétrer Firefox Sync sur tous vos ordinateurs.
