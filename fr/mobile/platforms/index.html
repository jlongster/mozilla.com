Votre téléphone n'est pas dans la liste ?

Si vous ne voyez pas votre téléphone dans la liste, vous pouvez tout de même télécharger et utiliser Firefox sur votre mobile. Votre mobile doit avoir assez de mémoire RAM (128 MB ou plus) pour exécuter Firefox. Ceci dit, nous vous encourageons à télécharger Firefox pour un des mobiles testés pour profiter au mieux de toutes les fonctionnalités.

Vous pouvez également essayer Firefox, fournir des retours d'informations, développer des modules complémentaires et contribuer en installant Firefox sur votre ordinateur de bureau :

  • Mac OS X
