Merci d'avoir choisi Firefox !

Suivez ces étapes pour télécharger Firefox, découvrir Firefox Sync, et mettez en route

  1. Regardez la boîte de réception de votre mobile pour consulter le SMS que nous venons de vous envoyer.
  2. Ouvrez le SMS et sélectionnez le lien qu'il contient pour télécharger Firefox directement sur votre mobile.

Firefox est gratuit ; cependant, des frais de connexion peuvent vous être facturés par votre fournisseur de réseau. Veuillez vérifier les conditions de votre forfait mobile.

Pendant le téléchargement…

  1. Paramétrez Firefox Sync et emportez votre Firefox partout avec vous. Partagez votre historique de navigation, vos marque-pages, vos mots de passe, et vos onglets entre votre ordinateur et votre mobile.
  2. Installez le module Firefox Sync sur votre ordinateur et enregistrez-vous sur Firefox Sync. Une fois Firefox téléchargé sur votre mobile, suivez les étapes que nous vous suggérons et sélectionnez Installer le module Firefox Sync pour terminer l'installation. Sinon, vous pouvez également sélectionner manuellement le bouton Préférences puis sélectionnez le bouton Modules pour rechercher et installer le module Firefox Sync sur votre mobile.
