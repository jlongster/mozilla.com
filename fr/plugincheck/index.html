Qu'est ce qu'un plugin ?

  • Les plugins offrent des vidéos, animations et jeux.
  • Ils sont développés par d'autres entreprises que Firefox comme Adobe Systems et Apple.
  • Les plugins ne se mettent pas toujours à jour automatiquement.

Pourquoi dois-je mettre à jour mes plugins ?

  • Les plugins qui ne sont pas à jour peuvent perturber votre navigation sur Internet et vous faire perdre du temps.
  • Les plugins qui ne sont pas à jour augmentent les risques d'attaque par des logiciels malveillants, des virus et toute autre menace de sécurité.
  • Des plugins à jour permettent de rendre Internet plus sécurisé et performant pour vous.

Comment Firefox peut-il m'aider ?

Dans un futur proche, Firefox mettra automatiquement à jour les plugins. En attendant, vous devez régulièrement vérifier via cette page s'ils sont à jour et le cas échéant les mettre à jour afin de préserver votre sécurité sur le Web au maximum.

Quels sont les plugins installés ?

Nous pouvons automatiquement détecter vos plugins, vous pouvez aussi facilement voir la liste de tous les plugins installés :

  1. Ouvrir le menu Outils.
  2. Choisir Modules complémentaires.
  3. Cliquer sur Plugins.

Comment désactiver un plugin ?

  1. Ouvrir le menu Outils.
  2. Choisir Modules complémentaires.
  3. Cliquer sur Plugins
  4. Cliquer sur un plugin dans la liste.
  5. Cliquer sur le bouton Désactiver.

Attention : désactiver un plugin signifie que vous ne pourrez plus faire certaines choses. Par exemple, si vous désactivez Flash, vous ne pourrez plus regarder des vidéos sur YouTube.
