Quoi de neuf dans

Firefox lance une autre façon d'intégrer les médias dans les pages Web : la nouvelle API audio expose les données brutes audio incluses dans les éléments <video> et <audio> du HTML5, ce qui permet aux développeurs d'utiliser facilement JavaScript pour lire/écrire des données audio. L'accélération graphique (Windows seulement)

Profitez de l'accélération graphique ultra-rapide avec Direct2D, désormais activée par défaut, avec le matériel compatible DirectX 10.

Connexions sécurisées aux serveurs

En utilisant le protocole HSTS, les sites peuvent maintenant demander à Firefox d'établir automatiquement des connexions sécurisées à leurs serveurs, empêchant les escrocs d'intercepter les données sensibles qui peuvent être en danger pendant qu'on accède au site. Pour en savoir plus »
