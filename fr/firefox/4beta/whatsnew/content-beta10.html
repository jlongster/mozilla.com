Nous avons besoin de plus de retours d'utilisation !

Grâce à vous, nous avons déjà reçu plus d'un million de retours d'utilisation pour Firefox 4 Bêta. Cependant, plus de retours d'utilisation ne font pas de mal ! Il suffit de cliquer sur le bouton « Réagir » dans le coin supérieur droit de votre navigateur pour nous dire ce que vous avez aimé ou pas (assurez-vous de joindre aussi l'adresse Web si vous nous signalez un site non-fonctionnel).

Vos sites favoris se comportent-ils normalement ?

Se chargent-ils rapidement ? Constatez-vous des lignes ou des boîtes de dialogue inhabituelles ? Pouvez-vous remplir les formulaires en ligne ? Pouvez-vous visionnez les vidéos ? Signalez-nous les sites qui vous posent problème.

Pouvez-vous jouer à votre jeu en ligne favori ?

Aidez-nous à nous assurer que les sites qui utilisent Flash ou d'autres composants multimédia fonctionnent correctement avec Firefox 4. Est-ce plus rapide ? Y a t'il un plantage ? Signalez-nous les jeux qui ne fonctionneraient pas avec cette bêta.

Quels modules avez-vous essayés ?

Vos modules complémentaires favoris sont-ils compatibles avec Firefox 4 Bêta ? Quels sont ceux qui ne fonctionnent pas ? Le nouveau gestionnaire de modules vous semble-t-il pratique ? Donnez-nous votre avis.

Si vous désirez bêta-tester des modules en plus grand nombre, installez Add-on Compatibility Reporter (le Rapporteur de compatibilité des modules). Grâce à cette extension, if vous remarquez que l'un de vos modules ne semble pas marcher aussi bien qu'avec les versions précédentes de Firefox, il vous suffit d'ouvrir le gestionnaire de modules complémentaires et de cliquer sur le lien 'Compatibilité' à coté du nom de ce module pour le faire savoir à Mozilla.

Avez-vous créé un compte Sync ?

Si vous aviez un compte Sync, que pensez-vous du processus de paramétrage de celui-ci dans Firefox ? Vous a t'il été possible de synchroniser vos données avec Firefox bêta pour mobile sur votre appareil sous Android ou bien avec Firefox Home sur votre iPhone/iPodTouch ? Si vous n'aviez pas de compte Sync existant, la création d'un compte dans votre navigateur a t'elle été aisée ? Partagez vos retours d'expérience concernant Sync.
