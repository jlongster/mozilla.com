Alerte de sécurité Firefox

Pour des raisons de sécurité, nous vous recommandons fortement de télécharger la dernière version de Firefox. La version que vous utilisez n'est plus maintenue, ce qui signifie que votre ordinateur sera potentiellement exposé à diverses menaces si vous continuez à utiliser cette version.

Pour optimiser votre sécurité en ligne, et pour apprécier les nombreuses améliorations délivrées par Firefox 2, cliquez sur « Récupérer la nouvelle version » ci-dessous.
