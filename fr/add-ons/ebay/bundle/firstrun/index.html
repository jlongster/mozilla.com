Vous venez de télécharger l’édition Firefox eBay et vous pouvez à présent vous lancer dans la recherche, les enchères et suivre les affaires sur eBay directement depuis votre navigateur Web Firefox.

Des millions de personnes à travers le monde ont choisi Firefox, séduits par une meilleure expérience en ligne, plus rapide et plus sûre. Firefox est développé et distribué par Mozilla, une communauté mondiale dont l’objectif est de faire du Web un espace plus ouvert optimisé pour tous.

Pour lancer l’utilisation du Firefox Companion pour eBay, il vous suffit simplement de reproduire les étapes suivantes:

  1. Cliquer sur l’icône eBay sur votre barre d’outils Firefox. Une barre latérale apparaît dans votre navigateur.
  2. Connectez-vous sur votre compte eBay en cliquant sur le bouton "Ouvrir une session " dans la barre latérale, puis entrez votre pseudo et votre mot de passe dans la fenêtre principale du navigateur.
  3. Si vous êtes un nouvel utilisateur eBay, cliquez sur le bouton d’inscription pour commencer.
  4. Vous y êtes ! Vous allez maintenant pouvoir rechercher ce que vous voulez parmi les millions d'objets disponibles sur eBay.

Pour plus de renseignements sur les fonctionnalités et les questions fréquentes sur Firefox Companion pour eBay, consultez la page sur les fonctionnalités et la foire aux questions.

Grâce à eBay Companion et Firefox, vous vous sentez en sécurité. Notre communauté et les 10 000 volontaires pour les tests qui la composent, travaillent chaque jour pour vous protéger contre toutes les dernières menaces du Web.

Choisir Firefox comme navigateur Web par défaut

Afin que l'expérience soit optimale, cliquez sur « Outils, puis option » dans le menu Firefox de votre navigateur et assurez-vous que l'option "Toujours vérifier au démarrage que Firefox est le navigateur par défaut" est sélectionnée. Vous profiterez ainsi de tous les avantages de Firefox à tout moment
