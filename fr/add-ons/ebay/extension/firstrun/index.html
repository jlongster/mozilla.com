Le Firefox Companion pour eBay a été installé. Vous allez bientôt pouvoir effectuer vos recherches, vos enchères parmi les objets eBay et les surveiller directement depuis votre navigateur Web Firefox.

Pour lancer l’utilisation du Firefox Companion pour eBay, il vous suffit simplement de suivre les étapes suivantes :

  1. Cliquez sur l'icône eBay sur votre barre d’outils Firefox et une barre latérale apparaît dans votre navigateur.
  2. Connectez-vous sur votre compte eBay en cliquant sur le bouton Ouvrir une session dans la barre latérale, puis saisissez les informations concernant votre compte dans la fenêtre principale du navigateur. Si vous êtes un nouvel utilisateur eBay, cliquez sur le bouton d'inscription pour commencer.
  3. Vous y êtes ! Vous allez maintenant pouvoir rechercher et surveiller parmi les millions d'objets disponibles chaque jour sur eBay.

Pour plus de renseignements sur les fonctionnalités et les questions fréquentes concernant Firefox Companion pour eBay, consultez la page décrivant ses fonctionnalités et la foire aux questions.

Grâce à eBay Companion et Firefox, vous vous sentez en sécurité. Notre communauté, forte de plus de 100.000 volontaires pour les tests, travaille chaque jour pour vous protéger contre les toutes dernières menaces du Web.

Choisir Firefox comme navigateur Web par défaut

Pour que l'expérience soit optimale, cliquez sur l'option Préférences dans le menu supérieur de Firefox et assurez-vous que l'option Toujours vérifier au démarrage que Firefox est le navigateur par défaut est sélectionnée. Vous profiterez ainsi de tous les avantages de Firefox à tout moment, même en ayant cliqué sur un lien dans un courriel.

Démarrer Firefox sur vos sites préférés

Savez-vous que vous pouvez démarrer Firefox avec vos sites préférés déjà ouverts dans des onglets séparés, prêts à être visités ? Cliquez ici pour que Firefox démarre avec la page d’accueil d’eBay en deuxième onglet.
