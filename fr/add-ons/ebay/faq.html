Comment ajouter le Firefox Companion pour eBay à mon navigateur Web Firefox ?

C'est très facile. Si vous disposez déjà de Firefox 2.0 ou d'une version supérieure, téléchargez et installez [lien vers la page d'extensions - site Companion] le Firefox Companion comme toute autre extension. Après l'installation, il vous est demandé de redémarrer Firefox.

Si Firefox 2.0 n'est pas déjà installé, vous pouvez télécharger un fichier unique [lien vers la page d'extensions - site Companion] comprenant à la fois le navigateur Firefox et l'eBay Companion.

Le Firefox Companion s'ouvre dans une barre latérale, à l'intérieur de votre navigateur Firefox.

Comment se connecter sur eBay avec l'extension Companion ?
Si la barre latérale du Companion n'est pas ouverte, cliquez sur le bouton eBay dans la barre d'outils : cela ouvrira la barre latérale. Cliquez sur le bouton « Ouvrir une session ». Cela vous dirige vers une page d'autorisation sécurisée eBay où l'on vous demande de saisir vos pseudo et mot de passe eBay. Après cette saisie, le Companion se connectera automatiquement sur eBay à chaque ouverture de votre navigateur.
Comment empêcher l'eBay Companion de se connecter automatiquement à chaque fois que j'ouvre Firefox ?
Cliquez sur le bouton de menu déroulant à droite du bouton eBay dans la barre d'outils. Sélectionner « Options… ». Cliquez sur l'onglet « Plus » et vérifier « Déconnecter le Firefox Companion pour eBay à la fermeture de Firefox ».
Avec quel type d'ordinateur et de système d'exploitation fonctionne l'extension Companion ?
Le Firefox Companion pour eBay fonctionne avec n'importe quel type d'ordinateur sur lequel Firefox est installé. Ceci inclut les plates-formes Apple Mac, Linux et Windows. Vous devez avoir un statut d'administrateur pour pouvoir ajouter l'eBay Companion à votre navigateur Web Firefox.
Comment ouvrir l'eBay Companion ?
Cliquez sur l'icône du Firefox Companion pour eBay dans votre barre d'outils Firefox et ceci ouvre une barre latérale dans votre navigateur Web Firefox.
Comment rechercher un objet à partir du eBay Companion ?
Utilisez simplement la barre de recherche à l'intérieur du eBay Companion pour trouver les objets qui vous intéressent. Vous êtes directement amené sur le site d'eBay où vous pouvez faire une enchère sur l'objet de votre choix.
Est-il possible de (sur)enchérir à partir du eBay Companion ?
Oui. Il suffit de cliquer sur le bouton « enchérir » à côté de l'objet pour lequel vous souhaitez augmenter votre enchère. Cela vous dirige immédiatement vers la page d'eBay où vous pouvez faire une enchère.
Pourquoi dois-je ouvrir une session dans eBay alors que l'eBay Companion est connecté sur eBay ?
Pour garantir votre sécurité, il vous est demandé d'ouvrir une session sur le site d'eBay indépendamment du statut de connexion du eBay Companion.
Le Firefox Companion pour eBay est-il sécurisé ?
Oui, avec l'eBay Companion, vous savez que vous êtes sur une vraie page eBay et pas sur un site usurpé ou piège similaire. En outre, l'utilisation de Firefox diminue les risques de fenêtres publicitaires, de logiciels espions et autres menaces Internet.
Comment désinstaller le Firefox Companion pour eBay?

Voici les étapes pour désinstaller le eBay Companion

  1. Dans votre bar de navigation, allez dans outils, puis modules complémentaires
  2. Dans la nouvelle fenêtre, cliquez sur « extensions »
  3. Sélectionnez « Firefox Companion pour eBay »
  4. Cliquez sur « désinstaller »
  5. Redémarrez Firefox
Comment surveiller d'autres objets que ceux que je suis en train de regarder ?
Sélectionnez un objet quelconque sur lequel vous avez enchéri, ou que vous surveillez ou vendez actuellement en utilisant le menu déroulant dans la partie supérieure du eBay Companion.
Puis-je savoir combien de personnes suivent un objet que j'ai mis en vente ?
Oui. Passez simplement votre souris sur l'objet qui vous intéresse et une infobulle apparaît avec des détails sur le nombre de personnes qui suivent l'objet et combien de personnes ont vu votre objet
Puis-je vendre des objets à partir du Companion ?
Non. Vous pouvez toutefois utiliser le menu activé à partir du bouton eBay dans la barre d'outils pour accéder rapidement à un raccourci qui vous redirige vers la page de vente eBay.
Puis-je changer la taille de la barre latérale du Firefox Companion pour eBay »?
Oui. Placez simplement le curseur sur le bord de barre latérale, cliquez dessus et faites glisser jusqu'à la largeur souhaitée.
À quels pays est destiné le Firefox Companion pour eBay ?
Initialement l'Allemagne, la France et le Royaume-Uni et éventuellement d'autres pays par la suite.
Si je ne me trouve pas dans l'un de ces pays, puis-je quand même utiliser l'eBay Companion ?
Oui, vous pouvez. Toutefois, l'assistance n'est disponible que pour l'Allemagne, la France et le Royaume-Uni.
Le code source du Firefox Companion pour eBay est-il libre ?
Nous envisageons un code source libre dans la mesure de ce qui est possible en ayant développé le code source sous MPL. Nous avons encore du travail en termes de concession de licence et de révision de code avant de sauter le pas, mais nous allons commencer sous peu. Toutefois, les graphiques, les marques, certaines fonctionnalités et les clés d'interface API utilisés pour concevoir Firefox Companion sont couverts par des licences concédées à des tiers et ne seront donc pas proposés avec un code source libre.
