Notes (First offered to Beta channel users on November 9th, 2011)

Check out what’s below, and find answers to frequently asked questions. As always, you’re encouraged to tell us what you think, or file a bug in Bugzilla.

What’s New

  • NEW
    Anti-Aliasing for WebGL is now implemented (see bug 615976)
  • NEW
    Accelerated layers are now supported via OpenGL ES
  • CHANGED
    It's now even easier to set up Firefox Sync - see the "Set up Sync" link on the home tab
  • HTML5
    New <bdi> element for bi-directional text isolation, along with supporting CSS properties (see bugs 613149 and 662288)
  • DEVELOPER
    We've added IndexedDB APIs to more closely match the specification
  • FIXED
    There is an extremely rare case where the browser may become unable to load web pages or close tabs (676780)

Known Issues

  • Unresolved
    Searches from the URL bar do not function reliably for some locales
  • Unresolved
    Some recent changes in the Google layout and CSS may cause issues in non-webkit browsers like Firefox
  • Resolved
    Blinking characters in form fields
  • Unresolved
    Unable to zoom in/out a webpage on header/footer when a double tap gesture is made
  • Unresolved
    Sync Now button is active even if the Sync is in progress
  • Resolved
    There is an extremely rare case where the browser may become unable to load web pages or close tabs
  • Unresolved
    For some users HTML5 video may load slowly and display rendering artifacts on the screen during playback
  • Resolved
    Reloading a page increases the zoom level
  • Resolved
    Adobe Flash and other plugins are not supported
