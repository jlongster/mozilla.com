Notes (First offered to Aurora channel users on 2011-12-22)

Check out what’s new and known issues for this version of Firefox for mobile below. As always, you’re encouraged to tell us what you think, or file a bug in Bugzilla.

What’s New

  • NEW
    We've made exciting changes to the latest Firefox Aurora for mobile! Find out more info in the Future of Firefox Blog Post
  • NEW
    Faster start-up and page load times!
  • NEW
    The user interface has been completely re-designed!
  • NEW
    "Request Desktop Site" menu option added
  • NEW
    Re-designed start page with Top Sites
  • NEW
    The CSS text-size-adjust property is now supported
  • CHANGED
    Top Sites, Bookmarks, and History are now more accessible from the Awesomebar
  • HTML5
    The outerHTML property is now supported on HTML elements
  • DEVELOPER
    SPDY protocol support for faster page loads is now testable
  • DEVELOPER
    XMLHttpRequest now supports HTML parsing
  • DEVELOPER
    Files can now be stored in IndexedDB (see bug 661877)
  • DEVELOPER
    Websockets has now been unprefixed
  • FIXED
    Blinking characters in form fields (see 665374)
  • FIXED
    Reloading a page increases the zoom level (see 628269)
  • FIXED
    Adobe Flash and other plugins are not supported (see 687265)

Known Issues

  • Unresolved
    Some recent changes in the Google layout and CSS may cause issues in non-webkit browsers like Firefox (see 668288)
  • Unresolved
    Sync Now button is active even if the Sync is in progress (see 656594)
  • Unresolved
    Firefox for Android is English-only regardless of any device-specific language settings (see 695565)
  • Unresolved
    Forms are not yet fully functional (see 705201)
  • Unresolved
    Some text may appear too large (see 710808)
  • Unresolved
    Opening links from external apps may not open properly. (see 711515)
  • Unresolved
    Your previous tab session may not be re-opened on launch in some instances (see 697858)
  • Unresolved
    Some Android 2.1 devices may crash on startup (see 705874)
