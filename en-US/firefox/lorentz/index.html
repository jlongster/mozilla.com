Keep Plugin Crashes Under Control

Firefox "Lorentz" provides uninterrupted browsing for Windows and Linux users when there is a crash in the Adobe Flash, Apple Quicktime or Microsoft Silverlight plugins.

If a plugin crashes or freezes, it will not affect the rest of Firefox. You will be able to reload the page to restart the plugin and try again.

Firefox Lorentz will become part of a future release of Firefox 3.6.

Help Test Lorentz

You can help Mozilla by downloading and using a Lorentz beta build. Then take care to visit websites which use plugins. If you encounter a crash, please submit a crash report. If you notice any other problems while you browse with this beta, please let Mozilla know by filing a bug or sending us feedback.

For more information on known issues, please see the release notes.
