Frequently Asked Questions

When will Firefox 4 be available?

Firefox 4 is currently in beta. To become a Firefox 4 Beta tester, please visit the Firefox 4 Beta page. The Firefox 4 Beta software will update every few weeks so you can test new features or changes, and it will eventually update to the final release of Firefox 4. We’re excited to receive your feedback and look forward to hearing from you!

On what platforms is the Firefox 4 Beta available?

The Firefox 4 Beta is currently available on Mac, Windows and Linux. Firefox 4 Beta for mobile is currently available on Maemo and Android.

What is Firefox Sync?

Firefox Sync synchronizes browser settings, history, passwords, bookmarks and even open tabs across multiple browser instances. If you have an iPhone you can get access to your Firefox Sync data using the application Firefox Home. Find out more information on Sync.

What is Panorama?

Panorama is the next evolution of tabbed browsing. With a single keystroke, it gives you an overview of all your open tabs, so you can organize them into groups and switch between them with ease. For example, you might create separate groups for tabs related to vacation, work, recipes, games and social networking – whatever makes sense to you. Take a closer look at Panorama.

Why does Firefox look different than it used to?

We updated the default look of Firefox to make it easier to focus on Web content and to control the tools in your Web browser. Also, we think it’s prettier looking. This blog post explains what we did to streamline the interface and why Tabs are on Top in Firefox 4.

Can I still customize the user interface?

Of course! The Firefox UI remains completely customizable. You can switch the locations of tabs, whether or not a menu bar, bookmark bar or status bar is visible.

How did stability improve in Firefox 4 Beta?

We hate crashes. When Firefox crashes, we try to get you back on your feet as quickly as possible, but we’d much rather you not crash in the first place. Mozilla recognizes that third-party plugins provide important functionality in many of today’s websites. At the same time, plugins can lead to problems for users as they browse. With the ability to alert users when they have out of date plugins, and now crash protection on all operating systems, Firefox allows users to experience all the content they love without any of the hassles.

How does Firefox 4 Beta protect my privacy?

Firefox puts your privacy first. Firefox 4 fixes flaws in the CSS Web standard that allowed your browser history to be exposed. Future releases will provide easy to use tools that let you control your privacy.

Is the Firefox 4 Beta safe to test?

Yes! Mozilla developers have been using Firefox 4 preview builds as their primary Web browser for months.

My existing Add-ons don’t work with Firefox 4 Beta. What can I do?

You can install the Add-on Compatibility Reporter and test your Add-ons!

Is Firefox 4 available in my language?

Firefox 4 Beta is currently available in more than 50 languages and we will add more soon.

Where can I get a quick overview of Firefox 4?

We’ve created a Firefox 4 Beta Features page. This is the best place to learn all about the exciting new and enhanced features in Firefox 4. You can also get a guided video tour of Firefox 4 Beta and see the new features in action.

Where can I see some cool demos of Firefox 4?

You can always check out the Mozilla Hacks site.

Should I expect Firefox 4 to look just like Firefox 4 Beta?

Firefox 4 Beta is now feature complete but there will still be some changes as we work to tweak and polish the final product. Your feedback is important to help us get Firefox 4 ready for the final release. Please see the Feedback page for more information on how to participate in the Firefox 4 Beta test.

How can I give feedback on my experience with Firefox 4 Beta?

Using the Feedback button in the upper right corner of the browser. Tell us anytime something makes you happy or sad!

What is the Feedback Add-on?

Firefox 4 Beta comes with the Feedback Add-on, which adds a button in the upper-right corner. You can use this button to provide instant feedback at any time. You will also be notified about user studies (run by Mozilla Test Pilot), which collect information about how you use Firefox, but not which websites you visit. You control your participation, and can choose to opt out of any study or see what data has been collected. You will be asked before any data is sent to Mozilla, and that data is subject to the Feedback and Test Pilot privacy policies.

When should we expect the next beta?

We plan on iterating quickly and shipping new versions of the beta every few weeks. Once you download a beta, it will automatically update whenever a new version is released.

What can we expect in the next beta? Or future betas?

Firefox 4 Beta is now feature complete. We are still looking at ways to improve performance and user experience as we prepare Firefox 4 for final release.

How does Firefox 4 Beta improve performance?

Firefox 4 Beta includes the enhanced high-performance SpiderMonkey JavaScript engine with TraceMonkey and JägerMonkey JITs. You’ll notice the speed in everything from faster start-up time and improved page-load speed to performance of Web apps. Firefox 4 Beta also has hardware acceleration, which enables websites to load and respond faster to rich and interactive content like Web games, apps or photos. Hardware acceleration is on by default on all supported hardware (release notes).
