Frequently Asked Questions

What is Mozilla?

Mozilla is a global, nonprofit organization dedicated to making the Web better. We emphasize principle over profit, and believe that the Web is a shared public resource to be cared for, not a commodity to be sold. We work with a worldwide community to create free, open source products like Mozilla Firefox, and to innovate for the benefit of the individual and the betterment of the Web. The result is great products built by passionate people and better Web choices for everyone. For more information, visit www.mozilla.org.

When will Firefox 4 be available?

The Firefox 4 Release Candidate (RC) is currently available for testing. Firefox 4 RC will automatically update to the final release of Firefox 4 when it is available. We’re excited to receive your feedback about the Firefox 4 RC.

On what platforms is the Firefox 4 Beta available?

Firefox 4 RC is currently available on Mac, Windows and Linux.

Firefox 4 is currently in Beta for Google Android and Nokia Maemo platforms.

What is Firefox Sync?

Firefox 4 RC includes Firefox Sync to synchronize your Firefox history, bookmarks, open tabs, passwords and form data across multiple computers and mobile devices. The new Firefox Sync user interface also makes it even easier to set up, helping go from work to home to mobile and back with ease.

What is Panorama?

Panorama is the next evolution of tabbed browsing. With a single keystroke, it gives you an overview of all your open tabs, so you can organize them into groups and switch between them with ease. For example, you might create separate groups for tabs related to vacation, work, recipes, games and social networking – whatever makes sense to you. Take a closer look at Panorama.

Why does Firefox look different than it used to?

We updated the default look of Firefox to compliment the way people use the Web today. The new look makes it easier to focus on Web content and to control your Firefox tools. For more details, this blog post explains what we did to streamline the interface and why Tabs are on Top by default in Firefox 4.

Can I still customize the user interface?

Of course! The Firefox user-interface (UI) remains completely customizable. You can switch the locations of tabs and whether or not a menu bar, bookmark bar or status bar is visible.

How else can I customize Firefox?

Firefox is the most customizable browser on the planet with a growing library of hundreds of thousands of free Firefox Add-ons and themes to choose from to add features and functionality to your Web experience.

Are my Firefox Add-ons compatible with Firefox 4 RC?

You can install the Add-on Compatibility Reporter to test your Firefox Add-ons. The Firefox development community is fast at work updating additional add-ons and we expect you’ll see lot’s more compatibility soon. You can go here to learn more about Firefox Add-ons.

Will the final Firefox 4 have the same look and features as Firefox 4 RC?

Firefox 4 RC is feature and UI complete, but there may be some changes as we work to polish the final product.

How did stability improve in Firefox 4 Beta?

Firefox provides uninterrupted browsing when there is a crash in the Adobe Flash, Apple QuickTime or Microsoft Silverlight plugins. If one of these commonly-used plugins crashes or freezes, it will not affect the rest of the page or Firefox. You can simply reload the page to restart the plugin and try again.

How does Firefox 4 RC improve performance?

Firefox 4 RC includes the enhanced high-performance SpiderMonkey JavaScript engine with TraceMonkey and JägerMonkey JITs. You'll notice the speed in everything from faster start-up time and improved page-load speed to the performance of Web applications. Firefox 4 RC also has hardware acceleration, which enables websites to load and respond to rich and interactive content like Web games, apps or photos faster. Hardware acceleration is on by default on all supported hardware.

How did privacy and security improve in Firefox 4?

Mozilla Firefox puts your privacy first. Firefox 4 RC builds on existing ways to protect your privacy in Firefox, including browser privacy controls and private browsing mode. Firefox 4 RC adds the following privacy and security improvements to Firefox:

Do Not Track The Do Not Track feature in Firefox tells websites that you would like to opt-out of tracking used for behavioral advertising. Firefox is leading the Web towards a universal standard with this feature. Users will not notice any difference in their browsing experience until sites and advertisers start to respond to the header.

Content Security Policy (CSP) The Content Security Policy in Firefox is designed to shut down cross-site scripting attacks by providing a mechanism for sites to explicitly tell the browser which content is legitimate. The browser can then disregard any content which has not been blessed by the site.

HTTP Strict Transport Security (HSTS) Websites can now help keep your sensitive data safe from interception during the log-in process by telling Firefox to automatically establish secure connections to their servers.
