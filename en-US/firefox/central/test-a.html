What’s new with 3.6?

  • Available in more than 70 languages—get your local version.
  • Support for a new type of theme called Personas. Change Firefox’s appearance with a single click.
  • Protection from out-of-date plugins keeps users safer as they browse.
  • Open, native video can now be displayed full screen and supports poster frames.
  • Faster startup time and improvements in overall browser responsiveness and JavaScript performance.
  • Fewer crashes through improved integrations with third-party software.
  • Faster page load times with functionality for developers to indicate that scripts should run asynchronously.
  • Support for new Web page experiences: gradients, background sizing and pointer events.
  • Continued support for downloadable Web fonts.
