Firefox 3.5 Preview

released June 8th, 2009 This is a preview of Firefox 3.5, the upcoming version of the Firefox web browser. It has not yet gone through a full set of quality assurance tests, and should not be considered as a release candidate. However, we appreciate your time and help in testing it by using it as your regular browser.

We expect that the Firefox 3.5 Release Candidate to be issued shortly. At that time you will be automatically upgraded to that version.

About this Preview

While this preview release has not undergone a full set of quality assurance tests, it has proven to be stable for the daily use of our nightly user audience, and we expect it to function properly. It is intended for developers and members of our testing community to use for early evaluation and feedback. Users of the latest released version of Firefox should not expect all of their add-ons to work properly with this preview.
