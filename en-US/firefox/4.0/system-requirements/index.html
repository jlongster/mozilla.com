Firefox 4 System Requirements

Windows

Operating Systems
  • Windows 2000
  • Windows XP
  • Windows Server 2003
  • Windows Vista
  • Windows 7

Please note that while the 32-bit and 64-bit versions of Windows Vista and Windows 7 can be used to run Firefox 4, only 32-bit builds of Firefox 4 are supported at this time.

Recommended Hardware
  • Pentium 4 or newer processor that supports SSE2
  • 512MB of RAM
  • 200MB of hard drive space

Mac

Operating Systems
  • Mac OS X 10.5
  • Mac OS X 10.6
Recommended Hardware
  • Macintosh computer with an Intel x86 processor
  • 512 MB of RAM
  • 200 MB hard drive space

Linux

Software Requirements
Please note that Linux distributors may provide packages for your distribution which have different requirements.
  • Firefox will not run at all without the following libraries or packages:
    • GTK+ 2.10 or higher
    • GLib 2.12 or higher
    • Pango 1.14 or higher
    • X.Org 1.0 or higher (1.7 or higher is recommended)
    • libstdc++ 4.3 or higher
  • For optimal functionality, we recommend the following libraries or packages:
    • NetworkManager 0.7 or higher
    • DBus 1.0 or higher
    • HAL 0.5.8 or higher
    • GNOME 2.16 or higher
