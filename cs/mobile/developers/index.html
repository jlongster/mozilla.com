Vyvíjejte inovativní doplňky

Jak Firefox pro Maemo, tak Firefox pro Windows Mobile (beta) podporují doplňky. To znamená, že stejné doplňky mohou být použity na obou platformách. Takže začněte a buďte mobilní. Zde je seznam užitečných zdrojů:

  • Základy tvorby rozšíření
  • Nejlepší postupy pro psaní rozšíření
  • Nejlepší postupy pro tvorbu uživatelského rozhraní
  • Přehled architektury
  • Ukázky kódu

Pomáhejte tvořit Firefox pro mobilní zařízení

Vaše pomoc je pro úspěch Firefoxu zásadní. Řekněte nám své názory na IRC kanálu (irc.mozilla.org, #mobile), navštivte naše středeční setkání vývojářů či nahlašte chyby.
