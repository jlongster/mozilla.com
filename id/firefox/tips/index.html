Buat Bookmark dengan Sekali Klik

Ketahui jejak situs yang telah dikunjungi dan disukai dengan membuat bookmark hanya dengan sekali klik. Pada saat Anda mengunjungi halaman yang ingin diingat, cukup klik ikon bintang yang ada di bar lokasi. Firefox akan menambahkannya ke daftar bookmark Anda agar dapat dicari lagi dengan mudah.

  • Situs menyediakan informasi identitas yang dapat diverifikasi dan melindungi data pribadi Anda dicuri dengar saat sedang daring.
  • Situs menyediakan informasi identitas dasar dan melindungi data pribadi Anda dicuri dengar saat sedang daring.
  • Situs tidak menyediakan informasi identitas apapun.

Ketahui dengan Siapa Anda Berinteraksi

Taktik umum penipuan di dunia maya adalah dengan membuat situs palsu yang dikenal dengan situs phishing, yang mirip dengan situs bank, belanja daring, dan semacamnya. Firefox mempermudah cara mengidentifikasi situs palsu semacam ini dengan cara mengklik ikon situs di sebelah kiri bar lokasi.

Memberikan atau memasukkan segala bentuk informasi pribadi pada situs penipu atau situs penyerang semacam ini dapat merugikan Anda sendiri.

Menengah

Bermain-main dengan Tag

Setelah mempunyai banyak koleksi bookmark, Anda akan mulai merasa kebingungan. Agar selalu terkelola dengan baik, Anda dapat menggunakan tag pada halaman yang telah dibuat bookmarknya: Klik ganda pada tanda bintang pada bar lokasi agar Anda dapat memasukkan tag yang diinginkan.

Anda dapat memberikan tag pada sebuah situs sebanyak yang diinginkan (jangan lupa menuliskan tanda koma untuk memisahkan setiap tag). Setelah itu Anda dapat menuliskan tag yang dicari di bar lokasi untuk menemukan situs yang sesuai. Misalnya, saat menuliskan kata "resep" pada bar lokasi, Firefox akan menampilkan daftar situs resep dan masakan yang telah dibuat bookmarknya.

Lebih Jauh bersama Pengaya

Versi standard Firefox memiliki fitur yang cukup lengkap, tetapi kenapa harus berhenti sampai di sini? Ada lebih dari 5000 Pengaya — tambahan kecil agar Firefox berfungsi sesuai selera Anda — yang dapat diunduh untuk memperkaya Firefox dengan hampir semua fungsi yang Anda inginkan. Buka menu Alat → Pengaya untuk membuka jendela pengelola Pengaya dan temukan berbagai cara untuk membuat Firefox sesuai dengan selera Anda.

Mencari Sesuatu di Web

Anda sudah biasa menggunakan bar pencarian yang ada pada Firefox? Tahukah Anda bahwa Anda dapat menyesuaikannya agar dapat bekerja dengan mesin pencari yang diinginkan? Cukup klik pada ikon di sebelah kiri bar pencarian untuk melihat menu pilihan yang tersedia secara default.

Tak perlu berhenti di sini, klik Kelola Mesin Pencari… untuk mengubah urutan daftar mesin pencari, menambah pilihan, dan juga mengatur shortcut papan ketik untuk mesin pencari favorit Anda. Misalnya, Anda dapat mengatur Google dengan shortcut huruf "G" agar pada saat ingin mencari resep masakan, cukup mengetikkan "G resep masakan" pada bar lokasi untuk untuk mendapatkan hasil pencariannya.

Mencari dalam Sekejap

Fitur Pencarian Sekejap pada halaman web akan menghemat waktu. Walaupun Anda dapat menggunakan bar "pencari" untuk menemukan kata pada halaman yang tampil, Anda juga dapat mengklik tempat yang kosong pada halaman tersebut dan langsung mengetikkan kata yang ingin dicari. Kursor pada layar akan langsung meloncat ke kata pertama yang sesuai yang dapat ditemukan.

Anda juga dapat menggunakannya untuk menemukan tautan pada halaman. Misalnya, jika sebelumnya Anda perlu menggerakkan mouse untuk mencari tautan, sekarang Anda dapat langsung mengetikkan kata pada tautan yang ingin dicari dan tekan Enter.

Hapus Semua Jejak

Untuk alasan privasi dan keamanan, tidak baik untuk meninggalkan jejak setelah menjelajah web (terutama jika menggunakan sebuah komputer secara bersama-sama dengan orang lain). Anda dapat menghapus riwayat penjelajahan dan unduhan Anda, menghapus isi tembolok dan kuki hanya dalam sekali klik. Cukup buka menu Alat → Bersihkan Data Pribadi… untuk melakukannya. Atau jika ingin benar-benar berhati-hati, buka menu Alat → Pengaturan → Privasi dan tandai pilihan "Selalu bersihkan data pribadi ketika menutup Firefox".

Canggih

Langsung Menuju Situs Favorit

Anda dapat membuat kata kunci untuk bookmark Anda agar lebih mudah dan cepat diakses. Dari jendela Pustaka, tambahkan kata kunci pendek pada isian kata kunci sehingga Anda dapat membuka halaman yang telah dibuat bookmarknya dengan mengetikkan kata kunci pada bar lokasi. Contohnya, Anda dapat menambahkan kata kunci "taut" pada bookmark halaman account Anda di del.icio.us sehingga saat Anda mengetik "taut" pada bar lokasi, Firefox akan langsung membuka halaman tersebut.

Buat Folder Pintar

Jika Anda adalah tipe "Peselancar web" canggih yang butuh menyimpan dan mengikuti perkembangan berbagai macam situs yang spesifik, hal berikut akan mempercepat Anda: dengan menggunakan jendela Pustaka pada Firefox 3, Anda dapat membuat dan menyimpan pencarian menjadi folder yang secara otomatis diperbarui seiring dengan penambahan bookmark dan riwayat.

Pertama-tama, buka jendela Pustaka dengan memilih menu "Kelola Bookmark" pada menu Bookmark. Setelah itu ketikkan kata yang ingin dicari pada kotak pencari, dan klik tombol Simpan untuk membuat Folder Pintar.

Kelola Unduhan Anda

Jika Anda sering mengunduh banyak berkas, Anda dapat menggunakan jendela pengelola unduhan untuk mengelola semua yang berkaitan dengan unduhan Anda. Anda dapat menunda dan melanjutkan unduhan serta mempercepat pekerjaan Anda dengan membuka berkas langsung dari jendela pengelola unduhan.

Jika Anda ingin mencari unduhan yang lama, buka menu Alat → Unduhan dan gunakan kotak pencarian untuk mencari berkas unduhan Anda. Setelah ditemukan, klik ganda pada berkas tersebut untuk membukanya atau klik kanan dan pilih "Salin Tautan Unduhan".

Temukan Perangkat Pengembangan

Jika Anda adalah Web developer, perangkat pengembang dari Firefox akan mempermudah hidup Anda. Situs Pengaya Mozilla menawarkan banyak perangkat pembantu untuk mempercepat proses pengembangan, termasuk di antaranya adalah Firebug yang dapat digunakan untuk menyunting, men-debug, dan memonitor CSS, HTML, serta JavaScript secara langsung pada halaman Web, Tamper Data untuk melihat dan memodifikasi header HTTP/HTTPS dan parameter POST, serta DOM Inspector untuk memeriksa elemen HTML atau CSS dengan cukup mengklik kanan.

Buat Web Seperti yang Anda Inginkan

Sekarang Anda dapat menggunakan Web-based protocol handler untuk menyediakan akses cepat ke aplikasi web favorit Anda. Misalnya, Anda dapat mengatur Firefox saat mengklik tautan mailto: untuk membuat email baru menggunakan penyedia layanan webmail daripada membuka program email yang terpasang di komputer Anda (catatan: fitur ini hanya tersedia untuk layanan email yang sudah terdaftar di Firefox 3).

Pilih menu Alat → Pengaturan → Aplikasi untuk memilih aplikasi default setiap protokol atau pilih "Tanyakan Selalu" jika Anda ingin untuk memilih aplikasi setiap kali Anda mengklik tautan.

Jika Anda adalah pengembang aplikasi web, temukan bagaimana cara menambahkan dukungan untuk Web-based protocol handler di Mozilla Developer Center.
