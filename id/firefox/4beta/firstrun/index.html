Kabari Kami Saran Anda

Sebagai penguji versi beta, Anda otomatis menjadi anggota tim yang membuat Firefox 4. Kabari kami apa-apa yang Anda suka atau tidak suka dengan mengklik tombol Saran. Anda akan ditanya apakah ingin mengikuti pengumpulan data studi pengguna secara anonim, atau juga mengisi survei tertentu.

Bagaimana cara studi pengguna ini berjalan?

Studi pengguna mengumpulkan data statistik anonim tentang cara pengguna menggunakan Firefox tetapi bukan apa yang pernah Anda buka atau lakukan di Firefox. Anda mempunyai kontrol penuh atas data Anda. Tidak ada data yang dikirimkan ke server Test Pilot Mozilla tanpa izin Anda. Data statistik yang dikumpulkan digunakan untuk mendesain Firefox 4.

Jika saya tidak ingin berpartisipasi, bagaimana?

Anda akan mendapatkan kabar sebelum proses pengumpulan data studi pengguna dimulai. Anda juga akan mendapatkan pemberitahuan sebelum data studi dikirim ke Mozilla. Anda dapat memilih untuk tidak mengikuti studi pengguna, atau mematikannya sama sekali dengan mengklik tombol Saran untuk mengubah pengaturannya.
