3D di dunia Web

WebGL membawa grafis 3D ke Firefox, membuka jalan baru bagi para pengembang untuk menciptakan permainan keren serta visualisasi dan pengalaman yang lebih kaya di dunia Web.

Fontasi Nan Cantik

Fitur fontasi OpenType pada Firefox memberikan kendali lebih lengkap bagi para pengembang dan desainer pada berbagai fitur fontasi, misalnya kerning atau ligatura untuk menciptakan situs web yang cantik.

Pemutakhiran Formulir

Firefox 4 membuat formulir lebih cantik dan sangat mudah untuk diimplementasikan. Dengan fitur seperti atribut daftar (list) baru, fitur otokomplit dan validasi formulir HTML5, serta API Forms baru untuk menyingkirkan segala tetek-bengek pengembangan formulir yang menjengkelkan.
