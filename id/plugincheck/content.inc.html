Pertanyaan umum

Apa itu plugin?

  • Plugin digunakan pada konten video, animasi, dan permainan.
  • Plugin dibuat oleh perusahaan di luar Firefox, misalnya >Adobe Systems dan >Apple.
  • Plugin tidak selalu diperbarui secara otoamtis.

Mengapa plugin harus diperbarui?

  • Plugin usang dapat mengganggu menjelajah web dan membuang waktu Anda.
  • Plugin usang meningkatkan risiko Anda atas serangan perangkat lunak jahat (malware), viruse, dan ancaman keamanan lainnya.
  • Plugin versi baru mengandung perbaikan yang membuat dunia web lebih baik dan lebih aman untuk Anda.

Bagaimana cara Firefox membantu saya?

Pada waktunya nanti, Firefox akan memperbarui pluginnya untuk Anda. Tetapi sebelum saat itu tiba, sebaiknya Anda mengunjungi laman ini secara teratur dan memperbarui plugin Anda agar tetap aman.

Plugin apa saja yang saya punya?

Secara otomatis kami mendeteksi plugin Anda yang tercantum di atas, untuk menampilkan plugin terpasang pada Firefox Anda, ikuti langkah berikut:

  1. Buka menu Alat.
  2. Pilih Pengaya.
  3. Klik tab plugin.

Bagaimana cara menonaktifkan plugin?

Di Firefox:

  1. Buka menu Alat.
  2. Pilih Pengaya.
  3. Klik tab plugin.
  4. Klik salah satu plugin pada daftar.
  5. Klik tombol Matikan.

Perhatian: menonaktifkan plugin berarti Anda tidak akan dapat melakukan beberapa hal. Misalnya, jika Anda mematikan Flash, Anda tidak akan dapat menonton video pada beberapa situs video populer.
