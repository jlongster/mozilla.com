3D på webben

WebGL ger Firefox 3D-grafik vilket gör det möjligt för utvecklare att skapa verklighetstrogna spel och nya slags visualiseringar och uttryck på webben.

Vackra typsnitt

Firefox 4 har typsnittsfunktionen OpenType som ger designer och utvecklare större möjligheter — t.ex. med kerning eller ligaturer — att skapa snygga webbsidor.

Uppgraderade formulär

Med Firefox 4 är formulär både snyggare och enklare att använda. Med funktioner som nya listattribut som aktiverar autokomplettering och formulärvalidering enligt HTML5, ger det nya Form API dig verktyg som gör det enklare än någonsin att utveckla och implementera formulär.
