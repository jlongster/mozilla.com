Berätta för oss vad du tycker

Som betatestare är du en del av teamet som gör Firefox 4. Berätta för oss när du gillar eller ogillar någonting genom att klicka på knappen Feedback. Du kommer också att inbjudas att delta i anonyma användarstudier, eller att besvara eventuella enkäter.

Hur fungerar användarstudier?

Användarstudier samlar in anonym statistik om hur du använder Firefox, men aldrig om vad du ser eller gör på nätet. Du har full kontroll och ingen data skickas till Mozilla Test Pilots servrar utan ditt medgivande. Informationen kommer att användas för att hjälpa till att förbättra Firefox 4.

Men om jag inte vill delta?

Du kommer att meddelas innan någon användarstudie börjar att samla in data, och åter igen innan några uppgifter skickas till Mozilla. Du kan välja att avstå från en studie, eller att stänga av alla användarstudier genom att klicka på knappen Feedback för att hantera dina inställningar.
