Firefox säkerhetsmeddelande

Av säkerhetsskäl rekommenderar vi starkt att du hämtar den senaste versionen av Firefox. Din nuvarande version stöds inte längre och därför kan din dator utsättas för olika hot vid fortsatt användning.

För att få högsta möjliga säkerhet på nätet - och för att dra nytta av de andra förbättringar som finns i Firefox 2 - klicka på ”Hämta den nya versionen” nedan.
