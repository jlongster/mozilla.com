Firefox 4 Beta inkluderar tillägget Feedback som är baserat på Mozilla Test Pilot

När du har installerat Firefox 4 Beta kommer du att upptäcka tillägget Feedback uppe i det högra hörnet. Du kan använda detta för att ge omedelbar feedback när som helst. Du kommer också att inbjudas till användarstudier (som körs genom Mozilla Test Pilot) som samlar in information om hur du använder Firefox, men inte vilka webbplatser du besöker. Du har full kontroll över ditt deltagande och kan välja att inte delta i en studie, eller att se vilken data som har samlats in. Du kommer att tillfrågas innan någon data skickas till Mozilla. Se sekretesspolicyn för tillägget Feedback för mer information.
