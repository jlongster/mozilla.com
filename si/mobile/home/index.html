ඔබේ නිවස

ඔබේ Firefox ඉතිහාසය, පොත්සලකුණු හා විවෘත කල ටැබ ඔබේ iPhone එක මත ලබාගන්න

බාගන්න දැන්ම එය iTunes මත ලබාගන්න – නොමිලේ

  • යන එන අතරමඟ?

    ඔබේ පරිගණකයේ විවෘත කල ටැබ් ලැයිස්තුවට ක්‍ෂණිකව පිවිසෙන්න

  • අඩුවෙන් ලියන්න

    දිගු URL ලිවීම අවශ්‍ය නොවේ, ඔබේ පොත්සලකුණු ටැප් කිරීම මගින් ඔබේ ප්‍රියතම අඩවි කරා ගෙනයනු ඇත

  • සොයන්න

    ලියනය අරඹන්න අගනා තීරුව ඔබව ඔබේ ප්‍රියතම හා වැදගත් අඩවි කරා ගෙනයනු ඇත

ඔබේ iPhone එක වෙත Firefox Home ලබාගන්නේ ඇයි?

නැගිට පිටවන්න

නැගිට පිටවන්න ඔබේ ජාලය ඔබ සඳහා ඔබේ iPhone එක මත රැඳවීමේ පහසුව ලබාදෙයි: ඔබේ ඉතිහාසය, විවෘත කල ටැබ, පොත් සලකුණු, හා අගනා තීරුව — ඔබ කැමති හා ඔබ හැර ගිය අයුරෙන්ම තබයි.

මුල සිට අරඹන්න එපා

දිගු URL නැවත ලිවීමට හෝ ඔබේ ප්‍රියතම අඩවි සෙවීම සඳහා කාලය ගත කිරීමට අවශ්‍ය නොවේ. ඔබ iPhone එක පණගන්වන විටම ඔබේ Firefox ඉතිහාසය ලබාගන්න.

නිවෙස මගේ නිවෙස

නිවස ඉතා වැදගත්, එම නිසා ඔබට ඔබේ පුද්ගලික දත්ත ආරක්‍ෂා කරන එලාම් පද්ධතියක් අවශ්‍යයි. ඔබේ රහස් වදන ඔබේ තොරතුරු වෙත පිවිසෙන්නේ ඔබ පමණක් බව තහවුරු කරයි.

ඇරඹීම

  1. ඔබේ පරිගණකයේ නොමිලේ ලැබෙන Firefox Sync ඇඩෝනය ස්ථාපනය කරන්න.

    Firefox Home විසින් ඔබේ Firefox ගවේශක තොරතුරු ආරක්‍ෂිතව ඔබේ iPhone එක වෙත යැවීම සඳහා Firefox Sync හි සම්මුහුර්ත හැකියාවන් භාවිත කරයි.

  2. Firefox නැවත ඇරඹූ විට, රහස් වදනක් හා මුර පදයක් යන දෙකම සහිතව ගිනුමක් සකසා ගැනීමට පොරොත්තු අනුගමනය කරන්න.

    Firefox Home ඔබේ රහස් වදන ප්‍රාදේශීයව ඔබේ තොරතුරු ගුප්තකේතනය කර ආරක්‍ෂා කිරීමට භාවිත වේ.

  3. Firefox Sync විසින් දැන් ඔබේ දත්ත අපේ සේවාව සමඟ ආරක්‍ෂිතව සම්මුහුර්ත කිරීම අරඹනු ඇත. මෙම ආරම්භක සම්මුහුර්තකරණය මිනිත්තු 20ක් දක්වා ගත හැක.

    Firefox Sync භාවිත කර හැක්කේ ඔබ පුද්ගලික ගවේශක ප්‍රකාරයේ නොසිටින විට පමණි. ඔබේ ආරම්භක සම්මුහුර්තකරණය අවසන් වූ විට, ඔබට Tools > Sync මෙනුවේ අවසන් යාවත් කිරීම දකිනු ඇත.

  4. ස්ථාපනය අවසන් කිරීමට ඔබේ iPhone එක මත ඔබේ ගිනුම, මුරපදය හා රහස් වදන දී ඇති ක්‍ෂේත්‍ර වල ඇතුළත් කර යතුරු පුවරුවේ "අවසන්" යතුර ටැප් කරන්න.

    ඔබට පිවිසීමේ ගැටළුවක් පැණ නැගුනි නම්, උදව් සඳහා මෙහි ක්ලික් කරන්න.

  5. අවසන් වූ විගසම ඔබේ පරිගණකයේ පොත්සලකුණු, ඉතිහාසය හා විවෘත ටැබ වෙත ඔබේ iPhone හි සිට පිවිසීමට හැකියාව ලැබේ.

    ඔබේ දත්ත නොපෙන්වන්නේ නම්, උදව් සඳහා මෙහි ක්ලික් කරන්න.

වඩා විස්තරාත්මක තොරතුරු

* වැදගත්: ඔබ විසින් Firefox Sync ගිනුමක් සැකසූ පසු, ආරම්භක සම්මුහුර්තය සම්පූර්ණ වීමට කාලය ගනු ඇත. ආරම්භක සම්මුහුර්තය අවසන් වූ විට, ඔබට ඔබේ iPhone හි Firefox Home මගින් ඔබේ දත්ත වලට පිවිසිය හැකි වනු ඇත.
